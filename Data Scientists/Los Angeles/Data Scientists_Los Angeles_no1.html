At Systech, you will have the opportunity to:

  • Implement enterprise-level machine learning solutions to solve business challenges using programming languages like Python and cloud platforms like Amazon Web Services

  • Synthesize and leverage complex datasets to drive data strategy/execution and develop Proof of Concepts

  • Design and develop dashboards to present predictions and insights

The ideal candidate possesses one or more of the following:

  • (Working towards) a degree in Business Analytics, Statistics, Data Science, Information Systems, Computer Science, or related field required

  • Practical experience and expertise in applied statistics and machine learning methodologies (E.g. Classification, Regression, Clustering, Time Series Analysis, Hypothesis Testing, Dimensionality Reduction.

  • Hands on experience using statistical programming languages like Python or R (Python Preferred)

  • Experience with data exploration and visualization tools (for ex. Tableau, PowerBI)

  • Strong SQL skills

  • Experience with Database Management/Distributed Framework: AWS, Oracle, MySQL, SQL Server, Spark

  • Knowledge in REST APIs, CI/CD and Docker

  • Experience with Cloud platforms (AWS, Azure, etc.)

  • Demonstrated ability to translate complex data sets into clear and actionable narratives, and the ability to communicate opportunities and challenges to multiple stakeholders

  • Ability to think strategically and deliver optimized solutions under stringent deadlines

  • Excellent communication, documentation, and presentation skills
