ComResource is looking for a Conversational AI Principal Engineer.

This role will be responsible for integrating the following technologies and Services to enable natural language interaction between customers with Voice Virtual Agent, instead of IVR - Google Dialogflow for Conversation Assistant.

Responsibilities:
  • Build utterances and conversations for conversations.
  • Establish the business logic within the use cases and connect it to Genesys Intelligent Automation.
  • Define the architecture to address functional, non-functional requirements and pain points; and implement different components of architecture on Google Cloud Platform (GCP) and Genesys Intelligent Automation (GIA) on premise contact center platform.
  • Design integration touchpoints per GCP speech and Natural language Understanding (NLU) Artificial Intelligence (AI) APIs.
  • Design and configure the Dialog Component in Google Dialogflow. Manage conversational flow by providing unique answer responses to each question intents related to the use cases.
  • Teach, test, and calibrate Conversational AI Technology and iteratively improve the user experience and ability to identify and return appropriate answers to questions pertaining to each use case via structured learning mechanism.
  • Identification of Intents and Entities based on priority use cases, Data collection, Identification, Cleaning, and labeling of the Intents and Entities, and further training of Machine Learning Models in GDF to identify the concerned Intent and Entity utterances.
  • Must have Google Dialogflow, ML algorithms, development using MATLAB / C C++ / Java / Python and or TensorFlow.
