Full Job Description - SR. DATA ANALYST MANAGER

The Sr. Data Analyst Manager at CPT would be responsible for managing the team of Database and Applications Programmers in the following tasks:

  • Overseeing all case related data analysis for the company.
  • Managing all incoming requests from Case Management team.
  • Analyze all requests with an eye towards data analysis best practices.
  • Create an execution plan for each request and assign it to a team member for implementation.
  • QC each completed request for accuracy and completeness.
  • Manage the development of the SQL scripts and applications used to update and migrate data across platforms or environments.
  • Manage the development of SQL scripts and applications to customize data imports or exports and help troubleshoot and/or repair data issues and format data to fit CPT's operations and administration process.
  • Manage the development of data transformations using SQL Server Integration Services (SSIS) or other applications.
  • Assess the potential root causes of technical problems and implement preventative measures through applications, scripts, or jobs.
  • Work closely with the Case Managers to facilitate data analysis on client supplied data, perform calculations and data aggregation for client approval.
  • Manage the development and creation of custom applications to automate some commonly used data analysis methods.
  • Perform testing on all scripts and applications to be used in production.
  • Maintain proper documentation to ensure compliance standards are met according to company SOPs.
  • Manage the development of interactive case websites as needed to facilitate the submission of claims online.

The Sr. Data Analyst Manager is also required to perform the above requirements as an individual contributor to the team for large complex cases that require advanced expertise.

Day-to-Day Responsibilities include:

  • Managing the team of Database and Applications Programmers.
  • Scheduling and assignment of work.
  • Supporting the team when there are questions.
  • Prioritizing the day-to-day tasks of the data and applications team based on schedules and deadlines.

Advancing the work in data analysis, website construction and application development of the department through the continued enhancement of tools and development environments used in the day-to-day operations.
