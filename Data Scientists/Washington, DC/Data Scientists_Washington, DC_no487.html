Title: Senior Oracle DBA / Data Scientist , Washington, DC

Clearance Required: SECRET


The senior oracle DBA defines, designs, and/or develops relational and/or multi-dimensional databases for warehousing of data. As the senior data architect for the organization, he/she will review current data structures and recommend optimizations and reconfigurations as warranted.


Responsibilities:

  • Daily maintenance including monitoring and database maintenance tasks
  • Install and Build database structures
  • Develop and maintain shell scripts
  • Query and performance tuning
  • Develop and maintain documentation
  • Program in PL/SQL
  • Performance tuning of databases
  • Reviews database design and make recommendations regarding enhancements and/or improvements
  • Maintains security and integrity controls
  • Develop stored procedures and/or triggers
  • Install database server software
  • Design and build databases, schemas, stored procedures, and other database objects
  • Set up routine database backup procedures
  • Perform database recoveries
  • Tune databases for performance. Monitor database performance trends
  • Assist developers and end-users with database usage, query development, and tuning
  • Develop operating system scripts to automate backups, data transfers, monitoring, etc.
  • Install and support the database components of packaged database applications
  • Support database connectivity from client applications
  • Export and import of database objects using Data pump utilities EXPDP and IMPDP
  • Advanced Data Modeling
  • Data Warehouse expertise

Required Skills:

  • Proficient in PL/SQL
  • Database developers who have knowledge of administration.
  • Hands on experience with Oracle Database and MS SQL
  • Experience in Data migration and recovery
  • Prior experience or ability work effectively with virtual teams
  • Min 10 years working as an Oracle DBA
  • Min 2 years experience with MS SQL Server
  • 3-5 years with Unix/Solaris/Windows
  • 5-7 years' experience in DBA tasks such as configuring backups, restoring DB from backups, troubleshooting DB issues, applying patches, data load operations
  • 5-7 years' experience with SQL programming
  • 5-7 years’ experience with Data Warehouses
  • 5-7 years’ experience with Business Intelligence
  • Experienced in migrating Databases from Oracle to SQL Server

Desired Skills:

  • Experience in using Migration tools such as SSMA
  • Knowledge in Azure Cloud and Azure SQL
  • Knowledge of Agile environment
  • Certifications of Oracle or Microsoft

Required Education:

  • Education Requirement with specific field of study: Masters of Information Systems Management, Associates in Computer Science, B.S. in science, health or related engineering field
