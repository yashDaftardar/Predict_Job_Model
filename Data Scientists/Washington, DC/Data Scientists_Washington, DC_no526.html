Cleared Data Scientist (Full Performance)

Kepora LLC
McLean, VA
Description:

The candidate will focus on the customer's business analytics, metrics collection, and analysis efforts. The customer’s management team requires a data scientist to characterize the work force demographics, workforce productivity and output, and the value of collection to the analytic workforce.

The candidate shall design and implement advanced code and table configurations for complex data sets, compile data analysis, and conduct needs assessments using deployed tools. Additionally, the candidate will stay current with the customer's enterprise metadata collection tools, produce ad hoc metrics reports to answer immediate needs, and leverage the customer's existing coordination process.

TS SCI w/ Poly Required

  • 2+ years' of relevant technical experience required
  • Bachelor's degree in Computer Science, Mathematics, Data Science, Statistics, or a related field
  • Professional experience using the Python
  • Professional experience with analytic methods and tools in: applied mathematics, data modeling or probability and statistics
  • Experience with machine learning
  • Experience presenting complex, technical findings
  • Experience with relational and non-relational database technologies (i.e. Oracle, PostgreSQL, MySQL, RDS, etc.)
  • Experience using advanced Microsoft Excel methods.
  • Experience with code repositories and build/deployment pipelines (i.e. Jenkins, Git)
  • Experience with data science techniques or formal data science training

Desired Skills:

  • Experience with data visualization technologies and packages (i.e. Splunk, Kibana, Bokeh, Leaflet/Folium, Tableau)
  • Experience with creative problem solving skills to address procedural and administrative issues within the assigned problem set
  • Experience with web analytics processing and procedures
  • Familiarity with Amazon Web Services infrastructure
  • Familiarity with Amazon Web Services’ native data science capabilities
