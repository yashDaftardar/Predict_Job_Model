Fantastic opportunity to practice data science and oversee an open-source data collection activity for a government customer in Washington D.C. (Joint Base Anacostia-Bolling). The position leads a team of data engineers and collectors that focus on open-source data extraction, transformation, storage, and analysis to gain valuable mission-specific insights from the data. The scientist will work closely with data engineers to improve data pipelines, semantic models, database stores, and query methods; and communicate insights gained and recommend ways-forward to both the team and customer.

(KEY SKILLS: “Data Science”, “Statistics”, “Open Source Data Analysis”, “Big Data”, “Cloud Technology”, “Leadership and Communication Skills”)

The data scientist will be able to perform the following tasks:

Extract features from large data stores containing open source publicly/commercially available information (PAI/CAI) in a wide range of possible formats, including JSON, XML, raw text logs, industry-specific encodings, and graph link data. Such data often contains bad records, partial records, errors, or other forms of “noising.”

Use software tools and write interface and query routines to support analysis of data drawn from natural language processing and computer vision to identify objects in text, image, video, and audio files.

Execute data science methods using common programming/scripting languages: Python, Java, R (statistics).

Develop operational analytics that result in workflow efficiency and/or extract new insights from collected data holdings. Leveraging available technology, work closely with data engineers and collectors to bridge the activity’s technical gaps, optimize data extraction; and lead/help the team to create, validate, and implement data models to solve problems and answer questions.

Apply descriptive and inferential statistics to describe data and make predictions about the data, including statistical tests to determine confidence for a hypothesis, common summary statistics (e.g. mean, variance, and counts), fit distributions to datasets and use those distributions to predict event likelihoods. Also, enhance open source collection requirements and support data evaluation objectives through statistical analysis and providing subject matter expertise during evaluation exercises.

The data scientist will be required to convey his/her findings through scientific visualization, written products, oral briefings, and interactive data visualization to both technical and non-technical customers.
