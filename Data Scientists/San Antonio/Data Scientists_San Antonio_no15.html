Space Data Systems is a CMMI-DEV V2.0 ML2/CL2 appraised high-performance software engineering team that empowers exploration by creating capable, sustainable software using proven technologies and effective software engineering processes in a people-focused work environment.

Objectives of this Role:

  • Work with small teams to develop and deploy Java web applications to support a variety of NASA/ESA space missions
  • Develop web applications to support space missions by providing tools for planning mission activities, instrument telemetry data management, spaceflight operations tools, and space manufacturing support

Daily and Monthly Responsibilities:

  • Develop new software functions in Java
  • Create and improve existing web applications with Spring, JavaScript, jQuery, AJAX, and CSS
  • Write SQL to work with MariaDB and other databases
  • Collaborate with scientists and engineers to plan new features and review products in development

Requirements:

  • Requires a Bachelors or a Masters with a 3.00 GPA in Computer Science or closely related field
  • Must have strong communication skills, be able to succeed in a team environment, and interface with clients
  • 0 years: Academic or industry experience with Java or either C++, C#, Groovy, or Scala and willingness to learn Java is required
  • 0-5 years: Academic or industry experience with Spring, JavaScript, jQuery, AJAX, JSP, Servlet, or SQL (Oracle or MySQL)
  • A valid/clear driver's license is required

Special Requirements:

Must be a U.S. citizen or Permanent Resident due to ITAR work in section.
