Full Job Description

The Co-op will run from July to December 2021. This position will require independent creativity as well as the ability to follow verbal and written instructions from multiple scientists and engineers. Reporting to the Senior Engineer in the FPL Vision Systems department, this person will be responsible for:

  • Building tools to aggregate measurement data into a SQL database
  • Working with APIs
  • Programming analysis software required to automate the inspection of features in an image
  • Managing the testing, analysis and calibration of test equipment

Requirements:


  • Physical science or engineering major with programming experience, preferably with image analysis or data science
  • 1+ years of experience programming with Python
  • Experience using one or more of the following Python packages: OpenCV, Pandas, NumPy, SciPy, or Django
  • Minimum GPA of 3.0.
  • Interest in computer vision, machine learning, and data science
  • Good communication skills
  • Good organizational skills and the capacity to manage own schedule
  • Must follow safe laboratory practices and be capable of quickly learning new techniques
  • Local address
