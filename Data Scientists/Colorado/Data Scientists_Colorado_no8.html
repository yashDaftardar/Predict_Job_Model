Title: Machine Learning Intern

Location: Westminster CO - Remote CO

Department: Central AI

Trimble Central AI

The Trimble Central AI team was formed to provide AI (artificial intelligence, machine learning and deep learning) expertise to all Trimble franchises and businesses. By using innovative thinking and technology, we help create AI solutions that enhance new and existing Trimble solutions. Join some of the most brilliant minds in a variety of industries and make a difference for the communities we serve.

Position Summary

The team has an exciting opportunity for a Machine Learning Intern to develop their skill set while contributing to projects that aim to solve industry problems with machine and deep learning. The Intern will work with the team’s Machine Learning Engineers and Data Scientists, running tests and experiments, developing simple applications and creating educational content to internally promote self-service data science tools managed by the team.

Responsibilities

  • Run tests and experiments to assist Machine Learning Engineers and Data Scientists

  • Develop simple applications to improve the team’s efficiency, e.g. automate repetitive tasks or enhance ability to communicate ML results

  • Produce Data Science Workbench tutorials that demonstrate the use of Central AI deep learning and analytic modeling frameworks

Required Skills:

  • Pursuing a degree in Computer Science or related discipline at an accredited university

  • Knowledge of math, statistics and algorithms such as K-means and SVM

  • Strong communication and collaboration skills within a team

  • Self-driven and directed mindset

  • Ability to communicate results clearly

Preferred Skills:

  • Basic knowledge of machine and deep learning algorithms

  • Python and TensorFlow

  • NVIDIA Jetson hardware
