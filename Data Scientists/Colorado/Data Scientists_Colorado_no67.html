Full Job Description

IMIDEX is a computer vision company that developing diagnostic algorithms to help radiologists detect lung cancer in chest X-rays. IMIDEX is comprised of a select team of data scientists, clinicians, and engineers on a mission to save lives. We are looking for an outstanding Software as a Medical Device (SaMD) Python Engineer to help deliver medical device products for use in the radiological field to market.

We are looking for a person with strong full stack machine learning experience in Python and the ability to take initiatives on the project. The successful candidate must be able to visualize the entire project and successfully build, analyze, test / debug and deploy software components as the project moves through the development process. Additional consideration given to candidates with web development and R experience.

Responsibilities:

  • Implement a complete software development process from conception to completion
  • Contribute technical expertise to the team and assists in the evaluation, requirements gathering, time estimation, and planning of software project
  • Analyze, write, build, and deploy a machine learning software product
  • Understand the reasoning behind key product and design decisions and contribute good solutions to implementation and architecture discussion
  • Own software components all the way through to production
  • Build cloud and web-based machine learning API
  • Make architectural design and technology decision
  • Implement tests to validate and verify your implementation
  • Follows standards and industry best practices with regard to the design, implementation, and maintenance of software application
  • Support fellow engineers through peer code reviews and constructive discussions that concern architecture, data model, and feature implementation decisio
  • Provide and receive meaningful feedback form the group to ensure the extensibility and reusability of the cod
  • Implement the automated testing and providing feedback to management during the development proces
  • Ship well-tested, secure, reliable, and maintainable code within committed timelines that delights product user
  • Maintain and upgrade the software following deployment

Qualifications

  • Experience developing production machine learning (ML) applications
  • Bachelor's degree in Computer Science or equivalent
  • 5+ years of software development experience; 2+ years as full stack developer
  • Proficiency with Python and machine learning frameworks like TensorFlow
  • Proficiency with web APIs development (e.g., REST)
  • Proficiency with database technologies (e.g., SQL server, MongoDB)
  • Experience with analytics and data visualization libraries
  • Experience with Docker / VM / Automation
  • Experience with Google Cloud Platform and machine learning environments
  • Experience with implementing security controls in web applications

Additional Qualifications:

  • Full stack development experience (frontend, backend, and database)
  • Proficiency with R
  • Experience with data standardization, preprocessing, and data harmonization
  • Experience serving deep learning models on the cloud using technologies such as TFX
  • Experience in containerization technology (ex. Docker) and web services architectures and technologies
  • Experience with analytics libraries such as numpy and visualization libraries such as matplotlib and plotly

Location:

IMIDEX is headquartered in Denver, CO, in the River North (RiNo) neighborhood and the Catalyst Health Technology Innovation Center. The position is at our headquarters in Denver; however, we will consider remote candidates that have demonstrated the ability to work in a team environment remotely and would be able to occasionally travel to Denver (after COVID conditions have improved) or consider moving in the next year.

Qualified candidates, please submit your CV to https://imidex.com/careers/
