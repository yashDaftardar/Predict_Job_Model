Title : Data Scientist


Location : Wabtec (formerly GE Transportation) – Grove City, PA


Job Description :

The Data Scientist will be a vital element in providing suggested direction to our customer’s management team. As a key member of our Analytics team, you will be responsible for pulling various data sets across multiple platforms, so that it can be analyzed in efforts to discover productivity opportunities. This position conveys discoveries in a clear and concise manner to an assorted audience thru standard and custom visualization tools.


Essential Responsibilities :

  • Provide key insights to the business, leveraging many of the data points available to be analyzed.
  • Leverage data visualization techniques for delivering results in both a graphical and textual format.
  • Designs and codes from specifications, analyzes, evaluates, tests, debugs, documents, and implements complex software apps.
  • Uses coding methods in specific programming languages to initiate or enhance program execution and functionality.
  • Participate in the evaluation, recommendation, and selection of software solutions.
  • Develop, maintain, and report intranet metrics.
  • Interface with different departments within the organization regarding data availability and fidelity.
  • Research/analyze data processing functions, methods and procedures.


Desired Skills :

  • A solid understanding of statistics
  • Familiarity or knowledge of SQL database/coding
  • Extensive experience in test automation using Selenium, QTP and Jasmine Framework or other automation tools with working knowledge of writing Java/Java script in a SAFe/Agile environment
  • Working knowledge of CI/CD to support continuous automation and deployment process with basic knowledge of Linux/Unix scripting is required
  • Experience with issue-tracking and agile project management tools such as Rally, Confluence, HP ALM etc.
  • Experience working with virtual machines (VMs) as part of a test/integration infrastructure
  • Knowledge of cloud technologies in AWS


Qualifications/Requirements :

Experience in data analysis with related programming skills.
