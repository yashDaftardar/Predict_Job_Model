The ideal candidate for this position will be able to reason about how to best implement services running on embedded systems which resource constraints while still developing solutions using modern tools. Having a focus on taking data and running inference/prediction algorithms with minimal latency. Work in a team with machine learning expertise, filling in gaps from the embedded engineering side.

JOB RESPONSIBILITIES:

  • Design and implement services to do ML inference (audio/video)
  • Messaging from ML output to appropriate services
  • Receive messages to change service and/or ML output according to user preference
  • Analyze inference speed and help identify areas to improve speed

Requirements

SKILLS:

  • C, C++
  • Strong understanding of linear algebra and numerical optimization
  • Experience with programming numerical algorithms (in machine learning or other fields) is desired
  • Experience developing large scale deployable software products
  • Experience working on large, complex codebases
  • Familiarity with version control tools (GIT)
  • Familiarity with CI/CD pipelines (ideally for cross-compiled projects)

HELPFUL SKILLS:

  • TensorFlow/TensorFlow Lite, Caffe
  • ARM Compute Library, ARMNN
  • Older ARM architectures (ARMv5)
  • Rust
  • DSP – Audio/Video
  • Gstreamer plugins
