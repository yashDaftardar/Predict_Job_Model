Full Job Description

Schweitzer Engineering Laboratories (SEL) seeks a talented individual for a Data Scientist position. This position will employ data science and artificial intelligence methods, high performance computing, and integration of these methods into software pipelines. The applicant will participate in state-of-the-art research and application of modern data and software techniques to a wide variety of power systems data. Applicants are expected to have a solid basis in a related field and an enthusiasm for innovation and creativity.

Applicants must be US citizens for this position.

Essential Duties and Responsibilities

1. Support SEL's data analytics research efforts.
2. Learn and apply machine learning and artificial intelligence methods to power systems data.
3. Develop experiments using the scientific method.
4. Provide support to engineering staff as needed.
5. Continuously identify, measure, and improve processes.
6. Follow and apply SEL Values, Principles of Operations, and World Class Manufacturing Principles.
7. Understand. Create. Simplify.
8. Other duties as assigned.

Required Qualifications

Bachelor of Science (BS) degree in a relevant field such as data science, computer science, or electrical engineering, mathematics, or statistics

Two or more years work experience in relevant engineering or scientific field

Strong understanding of fundamental machine learning topics such as feature engineering or dimensionality reduction

Solid object-oriented programming skills using Python, C#, C++, Matlab or Java

Strong writing, documentation, and speaking skills
Ability to learn new skills and assume new responsibilities
Ability to work cooperatively in a team environment
Background check results satisfactory to SEL
Negative drug test result(s)

Preferred Qualifications

Ability to use revision control and source code management systems such as Git

Experience in electric power system specific fields such as controls power system modeling

Experience in applying object detection, computer vision, or time series data analysis

Experience using cloud computing resources such as Amazon Web Services or Azure

Physical Requirements

None
