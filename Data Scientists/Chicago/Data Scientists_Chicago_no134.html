Position Summary:

Primarily focused on building and deploying tools that will fuel our data science solutions. The role will work with Data Scientists and Engineers, and build enterprise software that powers innovation at a major insurance broker. Our data platform is fully supported by IT resources and is geared to produce Machine Learning solutions that meets the highest standards in the industry. The Machine Learning Programmer will be tasked with creating and maintain those standards, building new tools for advanced analysis and deploying models across the organization.

Key responsibilities further entail:

Writing modular and maintainable code leveraging version control along with implementing a robust testing strategy

Regularly contributing to the existing data science code base to improve maintainability, fix bugs/issues, and build additional features

Collaborating with data scientists, data engineers and analysts to understand major pain points and building enhancements to existing tools and infrastructure or developing completely new packages to alleviate these pain points

Working with IT to maintain our data science infrastructure and improve model scalability.

Develops technical documentation outlining deployment processes as well as collaborates with other team members on defining coding best practices and ensuring these are enforced.


Required :

  • Bachelor’s degree in a technical field, such as computer science, data science, mathematics, or engineering.
  • Strong analytical, problem solving, and critical thinking skills.
  • Strong proficiency in a programming language such as R or Python.
  • Strong proficiency in writing modular and reusable code.
  • Experience with SQL and relational databases along with cloud and distributed computing tools.

Preferred :

  • Advanced degree in a technical field, such as computer science, data science, mathematics, or engineering.
  • Experience with more advanced programming languages such as Scala or Java.
  • Experience preparing code for production such as using version control (e.g. Git), writing tests (i.e. unit and integration), and leveraging CI/CD tools (i.e. Jenkins, CircleCI, Travis CI, etc.)
  • Knowledge of supervised (i.e. regression and tree-based) and unsupervised (i.e. clustering and dimensionality reduction) models.

Work Traits:

  • Passion for continued learning and development of new skills.
  • Applies quantitative thinking to solve business problems.
  • Learns technical skills quickly.
  • Passion for collaboration across data science and IT disciplines.
