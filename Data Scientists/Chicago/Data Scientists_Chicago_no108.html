Job Description


We are looking for a talented Machine Learning Engineer, responsible for developing cloud and edge based applications in support of priorities within the McDonald’s Global Data Hub. McDonald’s restaurant and digital technology powers 71 million orders in 120 countries across the globe daily – serving 1% of the total global population.

The ideal candidate will have previous experience deploying machine learning solutions at the edge through a combination of on-site installation and remote connection.

This position will collaborate cross functionally to understand and translate business needs into machine learning solutions. You will design and develop scalable solutions leveraging Machine Learning and Deep Learning models, working closely with data scientists and data engineers.

Responsibilities:

  • Apply statistical and machine learning methods to analyze large, complex data sets
  • Communicate highly technical results and methods clearly
  • Translates machine learning algorithms into production-level code
  • Collaborates with development teams to test and deploy machine learning models
  • Creates metrics to continuously evaluate the performance of machine learning solutions
  • Maintains and improves the performance of existing machine learning solutions
  • Experience developing machine learning planforms deployed on the edge across a distributed network.
  • Develop and IoT connections and dataflows in McDonald’s restaurants for consumption and integration into cloud environments.

Qualifications
  • Bachelor’s degree or higher in Computer Science, Mathematics, Economics, Statistics, or a related quantitative discipline
  • Experience developing, training, and evaluating classical machine/deep learning models, such as, SVMs, Random Forests, Gradient Boosting, NN, CNN, ResNet, GAN, etc.
  • Experience developing, training, and evaluating deep-learning models using public deep learning frameworks (e.g. PyTorch, TensorFlow/Keras)
  • Expert knowledge of data modeling and understanding of different data structures
  • Experience with object oriented programming in python
  • Experience leveraging cloud technology to scale modeling solutions (e.g. docker, kubernetes)
  • Experience with AWS (S3, sagemaker, greengrass, monitoring with new relic), GCP (ComputeEngine, CloudGPUs, cloud storage, pub/sub), or similar
  • Experience in developing state-of-the-art data analysis techniques for large-scale machine learning products
  • Excellent analytical skills with an engineering, algorithmic mind to architect tools, pipelines and systems to handle large-scale, real-time ML analysis, whether in batch or streaming.
  • Demonstrated ability to facilitate and coordinate complex data analysis and development activities across teams in agile sprints with minimal direction
