Full Job Description

We are looking for both experienced and junior level data scientists to collaborate both with clients and internal departments to perform research projects on various data pipelines, data sets, and data products.

Responsibilities:

  • Use data science skillset to improve the Code Willing platform. This can include anything from budget forecasting to ETL pipeline optimization.
  • Perform research on various data science technologies and platforms. The purpose of this research is to find ways to improve existing technologies used by Code Willing and their clients.
  • Complete research projects for clients. These projects can range from exploring new data set vendors to developing machine learning models.
  • Use statistical and machine learning techniques to perform data quality and anomaly detection on the data sets produced by Code Willing.
  • Write documentation and white papers for completed research projects.
  • Present to clients upon completion of projects or release of new data sets.

Minimum Qualifications:

  • Bachelors degree or equivalent experience in a quantitative field such as Computer Science, Mathematics, Statistics, Economics, or Finance
  • Proficient in Python and/or R
  • Proficient in SQL and working with large-scale databases
  • Basic knowledge of statistics (hypothesis testing, ARIMA, time series analysis, and probability)
  • Basic Knowledge of a variety of machine learning techniques (clustering, classification, regression, decision trees and etc.)
  • Strong communication skills (written and oral)

Preferred Qualifications:

  • Masters degree or higher or equivalent experience in a quantitative field such as Computer Science, Data Science, Analytics, Mathematics, Statistics, Economics, or Finance
  • 2+ years of experience with data science packages like Pandas, Numpy, and Matplotlib
  • 2+ years of experience with Machine learning packages like Scikit Learn, XGBoost, CatBoost, and LightGBM
  • 2+ years of experience with Deep Learning libraries like PyTorch or Tensorflow
  • 2+ years of experience scripting in a Linux environment (Python, Bash, Fish, Zsh, etc.)

Nice To Have:

  • Knowledge of other programming languages like C/ C++, JavaScript, Scala
  • Experience with Docker
  • Experience with Git
  • Experience working with financial data
  • Experience working with big data technologies like Spark
  • Experience working with cloud technologies like AWS
  • Experience building and optimizing ETL pipelines
  • Knowledge of Quantitative Analysis

Location: Baton Rouge or New Orleans

Salary: $50,000-$75,000 based on experience
