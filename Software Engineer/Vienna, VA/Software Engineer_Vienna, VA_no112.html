Will work in teams to develop embedded software in support of US Navy's sonar and anti-submarine warfare mission. Will develop models and algorithms in MATLAB and C/C++ to enhance and augment the sonar and other auxiliary Submarine Systems. Will use C/C++ programing language to implement acoustic processing solutions based on algorithms developed by Systems Engineering and Signal Processing engineers. This developed code must be optimized to operate within the boat's allocation of processors and memory footprint. Candidate will conduct integration and provide test support and problem resolution across system variants.
Basic Qualifications:
C/C++ Programming Language
Python or MATLAB Programming Language
Signal Processing
Proficiency in Linear Algebra and Differential Equations
Debug and Problem Resolution
Desired Skills:
Undersea Acoustic Processing and Algorithms
Working knowledge of Sonar Signal Conditioning, Beamforming and Post-Processing Techniques
Software Development Lifecycles
AGILE Processes
DevOps Processes
