2. Job Description
Software Engineers are the brains behind the design, installation, testing and maintenance, and client support of software systems. You do more than play around with code; what you write will likely help the business run more smoothly, or provide better service. Knowledge of programming skills is a prerequisite. However, the particular language you will focus on depends on the requirements of the specific project. The role of the Software Engineer is to build high-quality, innovative and fully performing software that complies with coding standards and technical design.
