Back End Engineer

MK Decision
San Diego, CA

We are looking for a Back End Engineer to join our Development Team.

MK’s Back End Engineers are responsible for building consistent, reliable software and features for internal and external users that upholds MK’s brand and reputation. As a Back End Engineer, you will work closely with our Senior Engineers to ensure system consistency and improve user experience. You will develop and maintain functional and stable “server-side” web applications to meet MK’s needs.

Your responsibilities include:

  • Participate in the entire application lifecycle, focusing on coding and debugging
  • Write clean code to develop functional web applications
  • Troubleshoot and debug applications
  • Perform UI tests to optimize performance
  • Manage cutting-edge technologies to improve legacy applications
  • Collaborate with Front-end developers to integrate user-facing elements with server-side logic
  • Gather and address technical and design requirements
  • Set up and maintain environment configurations and monitor system health
  • Provide training and support to internal teams
  • Build reusable code and libraries for future use
  • Liaise with developers, designers and system administrators to identify new features
  • Follow emerging technologies

Qualifications and Skills:

  • Proven work experience as a back-end developer
  • In-depth understanding of the entire web development process (design, development and deployment)
  • Hands-on experience with programming languages like Java, Groovy, Kotlin, and Python
  • Excellent analytical and time management skills
  • Teamwork skills with a problem-solving attitude
  • BS degree in Computer Science or relevant field equivalent
  • Experience building and deploying applications on Amazon Web Services
  • Highly organized with strong attention to detail
  • Must be self-directed and able to work independently, as well as work well with others
  • Strong written and verbal communication skills
  • Analytical thinker, effective at analyzing data and working to grow their analysis skills
  • Proficient with G-Suite (Gmail, Drive, Docs, Sheets)

Preferred Qualifications and Skills:

  • Knowledge of CMS framework
  • Familiarity with front-end languages (e.g. HTML, JavaScript, TypeScript and CSS)
  • Financial technology industry experience is a plus!

At MK, we're looking for people with drive, integrity, and humility. You are encouraged to apply even if your experience doesn't precisely match the job description. We welcome critical thinkers who are comfortable challenging the status quo. Join MK.
