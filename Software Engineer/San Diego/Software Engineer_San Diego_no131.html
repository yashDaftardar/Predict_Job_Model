Job Overview

Be a part of the engineering team responsible for designing, developing, and integrating embedded RF Software radio front-end, transceiver, and baseband control drivers. Innovate, design, and implement C/C++ embedded drivers for cutting edge RF and modem hardware technology for 5G. Participate in the product development lifecycle from inception and design to bring-up, optimization, verification and commercialization.

You will be part of a Team working on 5G mmW and Sub6 RF Control Software controlling state of the Art Antenna arrays for Analog Beam forming.

We have several openings that include:
-RF Software Driver Development
-RF Control Software for Front End component
-RFSW Beam Forming Control
-RF Factory Calibration.
-RF Self Calibration.
-RF Software Driver Architecture and Design with a focus on code portability using C and C++.

You will be working as part of a teams and also interfacing with other Teams to support the integration of RF Software components.

-Design and implement the software control for power optimization of RF Hardware.

-You will also be working with state of the art test equipment used to calibrate antenna arrays including both conducted and radiated antenna testing.

Interface with cross-functional teams within the overall modem organization: ASIC design, modem/RF systems, systems test, hardware, Firmware, Platform. Design and implement test mode interfaces to enable hardware characterization and tuning procedures to optimize RF performance.

Optimize device control transactions to meet strict timing requirements of 5G NR.

All Qualcomm employees are expected to actively support diversity on their teams, and in the Company.

Minimum Qualifications

+ Bachelor's degree in Engineering, Information Systems, Computer Science, or related field.

+ 1+ years experience with Programming Language such as C, C++

Preferred Qualifications

+ College graduate with a BS/MS or PhD in CS, CE or EE

+ Experience with RF test equipment measurements for power or spectrum or waveform quality. Experience with debugging embedded software using an in-circuit emulator (Lauterbach Trace32 JTAG) for an ARM processor

+ Experience with development of Layer 1 (Physical), firmware or RF Driver software using the C programming language, for one of the following technologies: LTE, 5G. Experience with embedded build systems including compiling, linking, loading, and executing firmware images

+ Experience with power and spectrum signal analysis using IQ capture data
