Key Qualifications
  • 5+ years of experience in embedded software development.
  • Excellent software developing and debugging skills on embedded real time operating systems
  • Excellent knowledge on embedded systems debugging methodologies and tools.
  • Validated understanding of modem cellular technologies and baseband system architecture (core, cache, memory, bus, Its, sleep/wake, clock, power, etc)
