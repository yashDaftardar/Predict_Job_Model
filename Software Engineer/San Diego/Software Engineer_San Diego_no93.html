Background Information:

Innovative Defense Technologies (IDT), provider of automated software testing, data analysis, and cybersecurity solutions for complex, mission-critical systems in the US Department of Defense (DOD), is seeking a Software Engineer to be based in our San Diego, CA office.

Overview:

We are looking for a Software Engineer to design, develop and integrate software solutions to support DoD customers. The main role of the Software Engineer is to build high-quality, innovative, and performant software that complies with coding standards and technical design.

Software Engineer responsibilities include gathering user requirements, defining system functionality and writing code in various languages, like Java and C++. The ideal candidate is familiar with the software development life cycle (SDLC) from preliminary system analysis to tests and deployment.

All applicants must be able to obtain/maintain an active U.S. Security Clearance.

Responsibilities Include:

  • Code and test automated system analysis algorithms in Java, JavaScript, Python, C++
  • Develop, integrate, and test Java applications operating in lab environment
  • Support lab-based test and integration activities at IDT in San Diego, CA and on-site in customer labs in Southern California
  • Learn and understand how complex customer systems function in order to integrate our software with them in a reliable fashion
  • Participate and provide feedback in design reviews & code reviews
