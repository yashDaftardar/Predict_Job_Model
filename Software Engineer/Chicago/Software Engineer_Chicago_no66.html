Overview

The IT Developer is responsible for the creation of advanced or custom O365 solutions that may expand or customize the standard offerings from IT vendors to maximize effective communications and business processes. The IT Developer, when creating these custom and unique solutions, will utilize development skills and tools related to the programming and development of O365 technologies. In addition, this position requires the individual to have a senior level understanding of O365 development tools, processes, and technologies and to assume responsibility for the successful completion of complex software project tasks.

The IT Developer performs tasks in association with IM application and technology teams. Duties may be performed directly or coordinated with external technology and/or service partners. The IT Developer is expected to be self-directed and perform responsibilities with minimal supervision.

Responsibilities

1) Develops or customizes O365 solutions and related services.
a) Work with end user sponsors and others to develop program specifications and assess appropriate technology
b) Develop solutions using programming and scripting tools for Microsoft – Javascript frameworks; PnP, SPO Framework
c) Develop familiarity and expertise utilizing vendor base API framework, e.g. Epic App Orchard and FIHR
d) Deliver and support custom application solutions for Microsoft Azure cloud environments
e) Develop and support applications using IoT, mobile (Apple & Android), and other evolving technologies as solution needs indicate

2) Provide integrated development and workflow solutions using standard O365 development methods and tools, including Agile methods
3) Develops and documents procedures for assigned solutions and services.
a) Monitors and report on performance for all assigned applications and services.
4) In-depth understanding of the SharePoint development lifecycle in support of multiple user groups.
5) Stays informed of Office 365 features and capabilities and works with necessary teams to plan, test, implement and communicate any changes to user community.
6) Work with cross functional business areas to develop process improvements using SharePoint and Power Platform
7) Support and maintain software integration between SharePoint and other in-house business applications
8) Assist and troubleshoot issues with Power Apps and Power Automate
9) Serves as liaison to vendors for software maintenance.
10) Participates in other department or organizational project tasks as required.
11) Completes assigned Help Desk support tickets within IM Department standards
12) Other job functions as assigned.
13) Manages and maintains development code in a centralized development code repository.

Qualifications

1) Education: Bachelor’s Degree in Information Systems/Technology/Computer Science (or equivalent work experience).
2) Ability to work as either a team lead or team member in a multi-disciplinary team environment
3) Demonstrated software project management skills and experience in both team member and leader roles.
4) Five (5) plus years of knowledge and experience working with standard development tools, software development skills, and techniques including but not limited:
a) experience with relevant programming languages and tools – Microsoft, .Net, Microsoft Visual Studio, C#, Windows PowerShell Scripting, Client Side Scripting, etc.
b) use of software tools and IDEs, including Microsoft Visual Studio and GitHub
c) high availability software solution techniques
d) secure application design
e) client/server and cloud application software design
f) software performance tuning and debugging in a complex enterprise environment.
5) Five (5) plus years of experience working with Microsoft SharePoint technology in both on-premise and/or Office 365 cloud environments.
6) Possesses Best Practice Change Management and Documentation skills.
7) Analytical thinking for effective problem determination and correction is essential.
8) Excellent time and project management skills, customer service and interpersonal skills, system diagnostic and communication (written and oral) are critical for success in this position.
9) Industry certification in the above skillsets highly preferred.
