Qualifications:

Your role at Huron: Currently we are identifying candidates with Junior Level .Net Developer experience for an upcoming project. We are looking for Consultants to begin as soon as possible. This will be a full time salaried position and will be 100% remote. Applicant must have the following experience:

Applicant must have the following experience:

  • .Net Developer experience is required

  • Programing Languages: C#, TSQL, Javascript

  • Web Development: ASP.net Core, ASP.net Blazor, Web API, REST, HTML/CSS

  • Database and SQL Server experience

  • Must have some experience developing .Net applications in C#

  • Must want to learn and grow and develop new skills

  • Must want to be in the Healthcare space, or is already currently in the Healthcare space

  • Must have an understanding of data and databases – and must have some experience creating tables, view, procedures

  • Huron requires a Bachelor’s Degree in a field related to this position or equivalent work experience

  • 1-3 years of experience in software product and test development with complex systems and/or application software

  • Interest in applying technology systems to business processes

  • Strong problem-solving skills and attention to detail

  • Highly proficient in Microsoft Excel and other Microsoft Office products

  • Applicant must have a very strong consulting background

  • Strong leadership skills

  • Provide expert guidance for Integration Development

  • Review customer requirements, perform analysis, design and documentation of current state and future state workflows

  • Maintain regular communication with client, vendor, team members and team leaders regarding project issues, risks, and needs

  • Classify problems as system, training, or process and recommend appropriate corrections
