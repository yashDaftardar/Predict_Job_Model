As a Software Engineer on the Engineering Operations team, you will improve and maintain the Product Development infrastructure, tools, and scripts responsible for housing and building our core products. The Software Engineer works under direct mentorship of Sr. Software Engineers.

Come join the team responsible for interconnectivity of a global network tasked with organizing and transmitting data in a fast growing environment. Build with us as we work in Azure and Cloud technologies such as Palo Alto, Kubernetes, Kong, and tech stack includes Powershell, Terraform, and Go.

Responsibilities

    • Develop the scripts and tools required to provision, configure and monitor azure infrastructure and its applications
    • Troubleshoot environments as problems arise, test fixes, and perform follow-ups to ensure problems have been adequately resolved
    • Identify, document, automate, and engineer processes for repeated tasks
    • Document your code additions, test procedures, and project results
    • Troubleshooting issues by fixing defects

Preferred Qualifications

    • Experience with infrastructure and distributed systems in a cloud environment (Azure)
    • Experience with configuration management and CI platforms (Chef and Jenkins)
    • Code using PowerShell, Go or Python
    • Experience with source control management tools (Git)

Minimum Qualifications

    • Experience with software engineering best practices: unit testing, code reviews, design documentation, debugging, troubleshooting
    • 1-3 years of software development of commercial-grade systems and application
