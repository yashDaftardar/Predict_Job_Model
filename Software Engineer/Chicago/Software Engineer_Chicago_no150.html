Summary

Varex Systems Engineering Team is seeking a Software Engineer to join our dynamic software team to develop and maintain our cutting-edge software in industrial NDT and security cargo screening X-ray radiography applications.

The candidate shall have good knowledge of latest Windows 10 application development using Microsoft .NET environment; A motivated self-starter who desires to achieve technical excellence; Strong analytical skill to solve technical problems; Familiar with Agile software development methodologies; Proficient in .NET application development using C#, WPF, WinForms, C++, software debugging, performance tuning, etc. Prior experience in SW/HW interfacing and integration is preferred.

Job Description

Responsibilities include:

  • Follow the internally established product design control process
  • Analyze product requirements
  • Work with cross functional team members to conduct reviews
  • Scoping and estimation of project tasks
  • Develop technical design document
  • Implement software functional modules
  • Perform unit and integrated tests
  • Debug and support the existing software products
  • Work within the team to conduct periodical technical review for improvement

Other duties as assigned.

Job Requirements:

Proficient in using C#, Java, C++, latest .NET framework, Microsoft Visual Studio 2015 or later; 2-3 years commercial software development experience preferably in the areas of industrial, security or medical imaging, high-performance graphical, interactive software applications; Knowledge in the interfacing and integration of HW including detectors and motion systems; Good knowledge and experience in GUI development using WPF/XAML, WinForms, MVVM; Experience in database application, distributed computing; Strong software design, implementation, coding, debugging skills; Work quickly with a high level of accuracy and high quality deliverables on time; Experienced in software development process and workflow, as well as use of software configuration management tools; Strong analytical, oral and written communication skills. Able to work independently or with a team; Demonstrable work history / experience as self-motivated, high performance software developer is desired.
