The Job

As the Software Associate I you will design, modify, develop, write and implement software programming applications and components; supports and/or maintains software applications and components; works from written specifications and pre-established guidelines to perform the functions of the job and maintains documentation of application processes.

What You’ll Do:

  • Utilize established development tools, guidelines and conventions including but not limited to SAP; Visual Studio, TFS, C#.NET, JavaScript, HTML5, CSS, AJAX, XML, Xamarin, SOLR; SQL; Python
  • Read and understand business and functional requirements
  • Design, code and test new business applications across verticals (Merchandising, Retail Support, Finance, etc.)
  • Enhance existing systems by analyzing business objectives, preparing an action plan and identifying areas for modification and improvement
  • Maintain existing software systems by identifying and correcting software defects
  • Troubleshoot production incidents and recover production systems.
  • Create technical specifications and test plans
  • Create new and extend existing operational documentation

What You Need:

Candidate should have 1-2 years of experience in programming and a Bachelor’s degree in Computer Science, Information Systems or related field. Excellent written and verbal communications skills, knowledge of systems development life cycle methodologies, and knowledge of relational database concepts and data access techniques including SQL are a must. Candidate must be able to work independently as a self-starter, as well as within a team environment, and have strong problem-solving skills and attention to detail.

Technical requirements include:

  • SQL
  • SAP/ABAP
  • Visual Studio
  • TFS
  • ASP.NET
  • C#
  • MVC
  • JavaScript / jQuery
  • HTML5
  • CSS
  • AJAX

  • Xamarin
  • SOLR
  • SQL
  • PYTHON
