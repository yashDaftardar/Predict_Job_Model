JOB DESCRIPTION

Our Client, a technology-driven e-commerce company is seeking Front-End Engineers who will help them in developing and delivering game-changing digital solutions.

RESPONSIBILITIES

  • Build and support robust tools, applications, and frameworks
  • Produce thoughtfully designed code that is reliable and maintainable
  • Develop good, comprehensive, automated functional tests
  • Review code and provide feedback to developers on ways to improve code quality, clarity, and performance
  • Review tests and provide feedback on ways to improve general code coverage, positive and negative test cases, and data management

QUALIFICATIONS

  • 2-4+ years of professional experience as a software engineer
  • Strong knowledge of software engineering fundamentals
  • Thorough knowledge of JavaScript ES5 and ES6, HTML, CSS
  • Familiarity with continuous integration and build tools (Webpack, Docker, etc.)
  • Mastery of one programming language and working knowledge of at least one other
  • Familiarity with multiple frameworks from languages and domains
  • Understanding of multiple programming paradigms
  • Good grasp of design patterns and anti-patterns
  • Be geniality interested and enthusiastic about learning new technologies

NICE TO HAVE:

  • Knowledge of / experience with Vue.js
  • Experienced with testing tools such as Jasmine, Cypress, Mocha, and others
