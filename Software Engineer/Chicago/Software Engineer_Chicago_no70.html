About the Client:

A leader in technology-enabled solutions for the Supply Chain Industry. They excel at helping their customers have real-time access and visibility to their data with comprehensive reporting for better decision making.

About the C# Software Engineer Position:

Looking for a Software Engineer that will contribute to the engineering of large-scale web-based applications to enable the business while supporting architectural vision of quality, scalability, performance and function.

About the C# Software Engineer Responsibilities:

  • Work hand-in-hand with Product Owners to break down user stories into small functional pieces
  • Serves as a resource to peers and provide mentoring whenever possible
  • Resolve defects in a timely and effective manner
  • Identify tactical risks and raises/resolves issues effectively
  • Contribute to merge up/down through development process
  • Ensure that operational teams are effective during deployment (dry-run or production)
  • Carry out root cause analysis on defects to ensure fixes are happening in the right way
  • Active member of a product team that solves complex challenges and builds working software
  • Produce high quality code

About the C# Software Engineer Requirements:

  • 5+ years of experience full-stack Software Engineering with HTML, CSS, JavaScript, Angular and .Net programming languages
  • Must have 2+ years’ experience developing/scripting with either AWS, Azure, Google API’s and Integration Web API’s.
  • 2+ years of experience developing a commercial-grade business application leveraging data structures, DBMS, SOA Architecture or writing code in OO programming language ideal
  • Strong time and project management skills in a fast-paced environment
  • Writing design documentation and technical requirements
  • Development with MS SQL Server
