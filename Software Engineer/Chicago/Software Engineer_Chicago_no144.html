Software Development Summer Intern – Chicago, IL
ADM Investor Services

Duties and Responsibilities:
  • Design business software using object oriented principal, tools and languages
  • Implement high quality software systems and business solutions in a fast-paced environment
  • Maintain existing software systems by identifying and correcting defects

Skills and Abilities:
  • Familiar with the following programming languages: C#, VB.Net, JavaScript, HTML
  • Web development and coding knowledge required
  • Familiar or experience with User Interface Design (UI), Security Principles, Object-Oriented Design, Web Services (REST/SOAP)
  • Familiarity with SQL and dashboard or report design tools
  • Capable of proactively exercising independent judgement and problem solving skills
  • Able to effectively and efficiently move projects from conception to completion
  • Flexible to meet current day or task demands
  • Works well as individual contributor or as part of a team

Experience/Other Requirements:
  • College student pursuing a 4-year Computer Science degree
  • Overall GPA of 3.0 or better
  • Promotes the ADM Way Core and Values
