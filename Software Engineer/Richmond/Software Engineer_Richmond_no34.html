Full Job Description

SpectraQuest, located in Richmond, Virginia, is a manufacturer of products for machinery fault diagnosis and vibration analysis. We are growing both domestically and internationally and a motivated person will have a tremendous growth opportunities. Spectra Quest, Inc is currently seeking interns for the following positions. Successful candidates will be considered for the full-time employment after 3 month of demonstrated exemplary performance and the company will apply for visas if needed.

1) Software Engineer Software development engineer for developing commercial packages for vibration analysis and diagnostics using LabVIEW.The primary duties include programming data acquisition packages and performing dynamic signal analysis using advanced digital signal processing techniques. The candidate must possess an in-depth knowledge of modern signal processing.

Requirements:
  • Proficient in LabVIEW programming
  • Data acquisition experiences using DAQmx
  • Digital signal processing
  • Sound and vibration fundamentals
  • Test and measurement techniques
2) Vibration Engineer Working on rotor dynamics, structural dynamics, and vibration testing. The primary duties include (1) performing diagnostics tests on machinery fault simulators, (2) analyzing and modeling results, (3) writing technical reports, and 4) publishing on the SpectraQuest website. The intern will work closely with an experienced mentor and with members of cross-functional development teams, and will have the responsibility for attending team meetings and for the completion and documentation of one or more assigned projects.

Requirements:
  • Vibration and rotor dynamics fundamentals
  • Data acquisition experiences
  • Digital signal processing
  • Test and measurement techniques
  • Strong mechanical engineering skills
  • Strong analytical and problem solving skills
  • Writing skills and Internet techniques
  • Excellent organizational, communication and interpersonal skills
  • Senior or graduate students
