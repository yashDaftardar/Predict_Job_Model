Full Job Description

The DevOps Engineer is responsible for implementing, maintaining and constantly improving the build system. This position ensures that VRS has a performant CI/CD pipeline that gives our development and infrastructure teams quick feedback and assurance that code is passing all tests and gates in the pipeline hosted in the cloud and/or on premise. This position drives increased efficiency across the teams, eliminating duplication, leveraging product and technology reuse at a release level.

Essential Duties and Responsibilities include the following but are not limited to:

  • Articulates the Release / Deployment Management process, policies, procedures and deployment
  • Plans the design / deployment of software release packages, in alignment with Release Calendar, for new / modified application systems
  • Manages the triage process of complex build/deployment problems, release and environment issues and bugs
  • Determines the readiness of each release based on release criteria; quality of release, production environment readiness, training and support plans, roll-out and back-out plans, and risk management plan
  • Creates, reviews, publishes and manages release plans, formal release notification, deployment instructions and launch plans as needed
  • Reviews Releases to ensure all security and compliance considerations are met.
  • Facilitates Releases / Deployment planning meetings with identified release participants to create Release and Deployment plans;
  • Continually seeks opportunities for improvements and provide feedback to build successful quality Releases.
  • Administers the on-premise source code repository currently managed in TFS.
  • Plans the migration from on premise TFS environment to Microsoft Dev Ops.
  • Functions as a subject matter expert to projects and implementations regarding DevOps & Cloud Architecture with Azure.
  • Strategizes, plans and manages all processes related to continuous integration, continuous delivery, and process automation.
  • Hands on development of automated deployment and monitoring of cloud infrastructure, web applications, and APIs.
  • Collaborates with Architects, OS Engineers, IT Team Managers, Business units and Engineers across the organization to deliver a comprehensive solution.
  • Provides technical support for the production environment.
