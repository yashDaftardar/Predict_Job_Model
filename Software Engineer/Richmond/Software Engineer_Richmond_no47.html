Title: Web UI Developer

Location: Tampa, FL

  • Responsible for front end development of internal and external website applications and understanding core presentation layer requirements to recommend and implement solutions that follow best practices.
  • Translates comps and wireframes into semantically correct web templates.
  • Integrating HTML templates and client-side code with server-side applications and content management systems to create functional and aesthetically pleasing user experiences that integrate cleanly into our existing systems and adhere to our design standards.
  • Responsible for SIT and UAT.Test, debug and troubleshoot web coding to ensure that code is deployable and functional. Reduce bugs in our production environment.
  • Develop responsive experiences for a variety of browsers, operating systems, and screen sizes to create functional and aesthetically pleasing user experience for a variety of platforms that our users interact.
  • Ensure that applications comply with our UX and Enterprise brand standards.
  • Utilize best practices and knowledge of internal or external business issues to improve products or services.
  • Effectively collaborates with IT counterparts in order to make joint decisions in the best interest of the product.
  • Ensure solutions are sound while adhere to budget, timelines and quality standards.

Qualifications

  • Requires a BS/BA in Computer Science, Mathematics, Engineering, or other technical area of study
  • 2-3 years experience in information architecture, usability, mobile/web user interface design/development
  • 2 or more years experience in client-side development in the consumer space; or any combination of education and experience, which would provide an equivalent background. Prefer experience working in an enterprise environment.
  • Build and update websites using HTML 5, Drupal (CMS), PHP, Bootstrap, CSS 3, JavaScript, jQuery, JSON
  • Optimization of the websites to meet the latest SEO requirement
  • Utilize Google analytics to manage and report website traffic trends
  • Ensure all website are Section 508 compliant and meet all requirements of accessibility for people with disabilities
  • Work with graphics editing tools like Adobe Illustrator/ Photoshop
  • Knowledge of fixing security flaws for the flaws found in security scanning tools like Veracode, Qualys etc.
  • Healthcare Medicare experience is a plus
