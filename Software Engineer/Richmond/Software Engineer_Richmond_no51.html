Position Description

The Development Operations Engineers (DevOps Engineers) provide daily support of the UNOS operating environment. This position handles a variety of responsibilities surrounding computing infrastructure maintenance, monitoring, source code repository management, release management, automation scripting, configuration management, site reliability engineering, server optimization, developer support and security. The candidate must possess excellent communication skills, should be comfortable working autonomously and within a team environment, and be willing to work on-call and after hours.

Key Responsibilities:

  • Proactively conduct assessments and continuously optimize computing solutions in terms of security, performance, scalability, and resiliency.
  • Work closely with Agile software engineering teams optimize the technology stack for performance, maintainability and fault tolerance.
  • Collaborate with infrastructure engineers to automate provisioning and management of computing resources through scripting and configuration management.
  • Conduct code promotion and change management through CI/CD pipelines. Automate testing and apply zero downtime production deployments.
  • Design and build logging and monitoring systems to identify bottlenecks and assist with the analysis and troubleshooting of hardware and software issues across server infrastructure.
  • Maintain high availability infrastructure and provide tier 3 operational support for mission critical systems.
  • Document, drive, and execute against industry best-practice processes and organizational procedures.
  • Perform system performance and load testing procedures.
  • Perform security audits and system hardening measures.
  • Share routine operations task and on-call support.

Minimum Requirements

2 years directly related DevOps experience. 3-5 years preferred.

Critical Skills

  • Foundational cloud computing experience
  • Knowledge of hyper converged and hybrid computing architectures
  • Principles of application security in cloud environments
  • Strong programming/scripting experience
  • Knowledge in automation scripting, configuration management, infrastructure-as-code
  • Experienced with continuous integration and continuous deployment (CI/CD)
  • Knowledge of virtualization, OS management, and networking fundamentals
  • Systems monitoring, logging and alerting (e.g. SCOM, Azure Monitor, Splunk, AppDynamics)
  • Knowledge of database scripting (Microsoft SQL)
  • Understanding of Agile/Scrum
  • Strong problem solving skills
  • Excellent cross team collaboration
  • Excellent written and verbal communications skills

Additional Skills & Qualifications

  • Azure DevOps
  • Microsoft Azure
  • Windows Server 2012+
  • Configuration management and Infrastructure-as-Code (e.g. Ansible, Terraform, Puppet, SCCM, etc.)
  • Containerization and Orchestration (Docker, Kubernetes)
  • API Management (e.g. Apigee, Azure API Management)
  • Load Balancing (Kemp, Azure Load Balancer, Azure Application Gateway)
  • Programming languages/Development Frameworks: C#, Powershell, ASP.NET, .NET Core, Bash, T-SQL, RESTful web services preferred
  • Experienced in source control management processes (Git, TFVC)
  • Experience with Content Management System Administration (Wordpress, Umbraco)

Education

About Us

The United Network for Organ Sharing (UNOS) coordinates the nation’s organ transplant system, providing vital services to meet the needs of men, women and children awaiting lifesaving organ transplants. Based in Richmond, Va., UNOS is a private, nonprofit membership organization.
UNOS members encompass every transplant hospital, tissue matching laboratory and organ procurement organization in the United States, as well as voluntary health and professional societies, ethicists, transplant patients and organ donor advocates.
