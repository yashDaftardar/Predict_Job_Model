Full Job Description

West Cary Group is a full-service, digital marketing and technology agency based in Richmond, Virginia, which services numerous mid-sized local and Fortune 500 clients across a vast array of industries. Hopefully you enjoy the frenetic, dynamic, and exciting environment of a fast-growing firm, because we’re looking for a dynamic Front End Developer.

Keep reading if:

  • You want to build software, not attend meetings
  • You love your work, but love balancing that with life
  • You need to constantly learn new things and tackle new problems
  • You want to be more than a small cog in a large machine
  • You want to have fun with the people you work with

Position Responsibilities:

  • Work on an agile team and as an individual to build world class websites and applications
  • Code in HTML, CSS/SCSS, JS and other tools to build dynamic websites
  • Must be able to write CSS based on comps delivered by the design team
  • Work on projects using React or similar JavaScript frameworks
  • Work with CMS tools as needed
  • Be able to work with Creative team to provide guidance on the technical implementation of designs
  • Work with backend engineers to coordinate the build of full stack applications
  • Other duties as assigned

Essential Skills and Experience:

  • High Diploma or equivalent required. Associates or Bachelor’s degree in Engineering, Computer Science or related field preferred.
  • Detail-oriented, ambitious, and a solid tactical thinker who knows how to turn a creative brief or creative direction into user-friendly, brilliant Web sites and online applications
  • A minimum of 2 years of proficiency in writing HTML 5, JavaScript, CSS
  • A strong understanding of cross-browser compatibility issues, specifically from Windows Internet Explorer 6 to current
  • Understands how to code for different devices
  • A minimum of 2 years of experience using Windows and Mac Web browsers
  • A minimum of 2 years of experience converting design mockups/comps to code
  • Ability to work with CMS tools like WordPress
  • Ability to apply critical thinking to a wide variety of problems
  • Possession of excellent verbal and written communication skills
