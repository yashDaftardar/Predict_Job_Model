Job Description

Title:

Software Engineer

Location:

Boca Raton, FL

Role/Function:

Our team is actively seeking a motivated and talented engineer who is passionate about building digital experiences. You will be part of an inter-disciplinary team of hardware, software, system, and test engineers to develop and integrate cutting-edge technologies for a leading financial services company, building and maintaining the platforms that help power and manage over 1000 employees and associated persons.

Description of the Primary Responsibilities

  • Troubleshooting issues in our existing applications/jobs
  • Building new applications and jobs using C#, Python, Java
  • Adding features to existing applications
  • Determine operational feasibility by evaluating analysis, problem definition, requirements, solution development, and proposed solutions
  • Develop software solutions by studying information needs, conferring with users, and studying systems flow, data usage, and work processes

Minimum Qualifications:

  • BS degree in Computer Science, Engineering or relevant field
  • Knowledge of professional software engineering and best practices for the full software development life cycle, including coding standards, code reviews, source control management, build processes, testing, and operations
  • Proven work experience as a Software Engineer/Developer
  • Proven work experience with SQL Server and or equivalent.
  • Ability to understand, troubleshoot and compose complex queries
  • Experience with test-driven development
  • Proficiency in software engineering tools
  • Ability to document requirements, specifications and training material

Other Qualifications (Preferred but not required):

  • General familiarity with financial services data model
  • Work experience in the financial services industry
  • Experience with ETL tools such as Alteryx, SSIS, etc.
  • Master's degree in Computer Science, Engineering or relevant field
