Job Description
Jacobs is seeking a full time Software Developer 1. Under governmental direction, provide software and systems engineering tasks for the requirements, design, implementation, integration, and support of all software and associated documentation for new, upgraded, and existing PEX capabilities. Using an Agile software engineering approach, specific functional requirements are fluid throughout the life cycle of the project. The PEX program is conducted in accordance with Agile principles, one of which states: “welcome changing requirements, even late in the development.” Through the iterative process of requirements and product definition embodied in the Agile approach, capabilities may be reduced and new capabilities added to the PEX system over the course of the period of performance.

PEX has an immediate opening for a Front-End Developer (Angular/HTML5) who will both design and implement cohesive UI elements within a robust web application based on requirements. This position will work within a small team of front end and middle layer developers in an Agile development environment.

Typical Duties

Work scope may include any/all of the following duties:

  • Develops software in a CMMI Maturity Level 3 environment using Scrum/SAFE and Agile methodologies.
  • Maintain and update existing front-end systems (ASP.NET, C#, Javascript, Silverlight, Angular, TypeScript, Web services, Breeze, Web API...).
  • Develop new front-end and/or middle layer features
  • Collaborate with team daily, remotely, to integrate new features
  • Breakdown Requirements
  • Create User Stories
  • Participate in Release Planning
  • Work in two week iterations
  • This list is not intended to be all-inclusive and may be expanded to include other duties and responsibilities.
