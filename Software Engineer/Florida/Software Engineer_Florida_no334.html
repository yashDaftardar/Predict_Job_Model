Description:

Responsible for the daily administration and management of the Rubrik data backup environment and ensures the availability, functionality and performance of the backup infrastructure. Responsible for daily monitoring of backup reports, trends, and metrics, and takes necessary action for remediation of failed events, including collaboration with other teams. Proactively engages with supporting vendors to plan and apply required software updates. Responsible to work with server architects/engineers for the engineering, build, installation / configuration, operation, and maintenance of the data center infrastructure, systems hardware and software and the related infrastructure running in the Window Server environment. Responsible for hardware administration including provisioning, monitoring, tuning, capacity planning, security, troubleshooting, support and maintenance.
