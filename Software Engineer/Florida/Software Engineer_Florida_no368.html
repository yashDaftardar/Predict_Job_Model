What does an exceptional Software Developer, Advisory do?

The Software Developer, Advisory is responsible for leading a large VisionPLUS development projects. He/She must have VisionPLUS experience with expertise one of one or more V+ modules – CMS, FAS, or TRAMS.

In this role, you can look forward to:

  • Data Migration effort includes extract, transform and load the data from the current source system into Fiserv LATAM platform.
  • Data Migration is a critical work stream in the whole migration process
  • Data Migration work stream interfaces with all other work streams and provides critical current system information to the other work streams as needed to ensure successful migration
  • Coordinates and manages Conversion Discoveries, Data Mapping and Migration Development
  • Manages a Team of Developers to ensure timely deliveries with minimal issues.
  • Develop file by file Migration routines based on the data mapping specifications
  • Develop Driver programs to perform the Migration process for all the files taking in source system files and producing Vision files.
  • Mock test will be done to perform the actual Migration process, and the validation
  • Executes Dress Rehearsal test to perform the actual Migration process and this test will run as close as possible to the live implementation.
  • Works closely with Regional Teams to ensure successful Mocks, DR and Live Implementations
  • Manages all aspects of Data Migration for Live Client Migrations

Basic Qualifications/Skills:

  • 7 - 10 Years experience with VisionPLUS Analysis & Development
  • Have strong background and experience in VisionPLUS Application Design, Development and Support.
  • Provide Functional and Technical Guidance to the team members to achieve the team's objectives towards Application Design, Development and Support.
  • Should have strong techno-functional knowhow and should be able to independently provide solutions/designs for complex functionalities.
  • Deal and liaise with internal and external stakeholders.
  • Ensures adherence to established standards and may consult with senior managers on technology solutions.
  • Supports existing business systems applications and may be responsible for leading technical aspects of a project.
  • Good knowledge in emerging technologies.
  • Technical Skills: Mainframe, Micro focus. Tools: Endeavor, Change-man, SDK, File-Aid, File Master, VP scheduler, Expeditor, IBM Debug tool. Languages: COBOL, CICS, JCL, Putty. Database: DB2, VSAM
  • Bachelor's Degree in Engineering Technology or equivalent.
