SUMMARY

Researches, designs, and develops computer software systems, in conjunction with hardware product development, by performing the following duties.

ESSENTIAL DUTIES AND RESPONSIBILITIES include the following. Other duties may be assigned.

  • Analyzes software requirements to determine feasibility of design within time and cost constraints.
  • Consults with hardware engineers and other engineering staff to evaluate interface between hardware and software, and operational and performance requirements of overall system.
  • Formulates and designs software system, on IBM PC, using scientific analysis and mathematical models to predict and measure outcome and consequences of design.
  • Power System Calculations
  • Develops product specifications
  • Codes hi-level algorithms.
  • Codes assembly language algorithms.
  • Codes Window applications using C# and C++ Visual Studio or any Object Oriented Language.
  • Codes communications protocols.
  • Develop and troubleshoot testing procedures. Work with Engineering Test Technicians.
  • Write and debug software using real-time programming
  • Develops and directs software system testing procedures, programming, and documentation.
  • Revise complex manuals, existing products as needed.
  • Work with other team members to accomplish goals.
  • Comply with all work rules, including but not limited to, the Company’s safety and attendance policies.

EQUIPMENT/TOOLS: IBM PC, Lab equipment as required, telephone, FAX machine, copier, calculator.

EDUCATION and/or EXPERIENCE: BSEE/BSCE or Master’s degree with a minimum of three years related experience. One to three years in software coding using “C#”, “C”, “C++”, and/or any Object Oriented language.

SKILLS & KNOWLEDGE

  • Familiar with high speed serial communication techniques.
  • Knowledge of Power System Theory.
  • Familiar with developing algorithms on microprocessors.
