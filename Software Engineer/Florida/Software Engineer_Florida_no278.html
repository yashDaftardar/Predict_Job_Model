POSITION SUMMARY:

The position reports directly to the IT Project Manager and is responsible for the design, development, and maintenance of custom developed software applications for the various business divisions under Porpoise Pool and Patio.

Required Skills

POSITION QUALIFICATIONS/REQUIREMENTS:

This position requires a thorough knowledge of the software development life cycle including requirements gathering, development and testing methodologies. An Associate’s or Bachelor’s degree in Computer Science or related developmental technology required. A minimum of 5 years work experience in the software development life cycle required. Specific technologies required include: C#, Restful Web Services, T-SQL, ASP.Net, HTML, Javascript and JSON. Knowledge of basic business and accounting practices necessary. Excellent interpersonal skills required. Position requires approximately 10% travel. Web Service development experience is necessary, knowledge of RESTful web service is required.

ESSENTIAL FUNCTIONS OF POSITION:

  • To perform this job successfully, the associate must be able to perform each essential duty satisfactorily.
  • Provide clear technical documentation and release notes
  • Make use of SQL databases
  • Produce custom applications using .NET
  • Test and debug applications
