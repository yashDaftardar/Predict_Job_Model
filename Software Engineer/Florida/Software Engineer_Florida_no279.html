In this role, you have the opportunity to

Work as part of a collaborative product team to design, develop, implement, and test secure processing software in embedded and enterprise secure processing solutions.

This role is responsible for

  • Modifying software to fix errors, adapt it to new hardware, improve its performance, or upgrade interfaces
  • Designing and testing C/C++ & Python programs in a Linux environment
  • Developing and maintaining software test suite to verify system capabilities and requirements
  • Collaborating with engineering team to design and develop software architectures and APIs
  • Capturing system specifications and performance requirements
  • Applying secure coding practices to new and legacy software
  • Supporting design team in bring-up and test of hardware products
  • Maintaining and operating CI/CD tools and platforms
  • Maintaining and adding new FPGA development boards to the lab network for CI/CD systems
You are part of

Mercury's system security engineering team that is focused on embedded and enterprise secure processing solutions. You will be engaged during all phases of the product lifecycle and will have the ability to influence software compatibility and system design decisions as well as the responsibility to design, develop, simulate, test and document embedded software systems and products

To succeed in this role, you should have the following skills and experience:

  • B.S. in Computer Engineering, Computer Science, Electrical Engineering or prior work experience in embedded software development
  • Proficient in C, C++, and Python
  • Ability to write and review technical documentation
  • Troubleshooting and logic skills
  • Ability to work in a team environment
  • Excellent communication skills, both oral and written
  • Must have or be eligible for Secret level Security Clearance
Additional Preferred Qualifications:
  • Proficient in command line scripting -- bash, tcsh, etc.
  • Coursework related to Object Orientated Programming
  • Experience with Linux/UNIX operating environments
  • Experience with automation/configuration management tools -- Jenkins, cron, etc.
  • Experience configuring and maintaining storage systems and protocols -- SMB, NFS, iSCSI
  • Experience with VMWare vSphere Cluster
  • Experience with networking systems and protocols -- Firewalls, DNS, SSL, TCP, etc.
  • Familiar with Vivado, Libero, and/or Quartus design workflow
