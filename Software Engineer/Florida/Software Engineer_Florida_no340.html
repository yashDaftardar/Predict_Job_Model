Full Job Description

The AWS DevOps Engineer must be able to meet the key criteria below:

  • Location: Must be onsite in either Orlando or Rockledge, FL (50% telework); or 100% remote if located in the Washington D.C. area
  • Years' Experience: 5+ years
  • Education: Bachelors in IT related field
  • Clearance: Active Secret Clearance preferred
  • Work Authorization: Must be a United States Citizen
  • Key Skills:
  • AWS and Infrastructure as Code experience
  • Must have extensive hands-on experience with automation and scripting (python, bash, etc)
  • Experience using Jenkins, Ansible, or other CI/CD technologies
  • Must be willing to obtain Security+ certification

Overview

Do you want to help IBR build a portfolio of next-generation data collection systems? As an AWS DevOps Engineer at IBR, you will support the Agile based engineering of a robust, secure, and scalable enterprise web portal solution supporting 1M users hosted in AWS. You will work with multiple application teams to migrate systems into AWS environment in support of the MyNavyPortal program.

Responsibilities

  • Takes direction from the DevOps Engineering Lead to migrate several applications into DevOps environments, either cloud or data-center based.
  • Provides technical guidance to application teams to take advantage of cloud technologies, and implement cloud infrastructure, as needed.
  • Works with application teams to document application internal/external interface requirements for Development, Testing, Staging and Production environments
  • Works with application teams to ensure compliance with High Availability and Disaster Recovery related concept of operations.
  • Implements middleware application specific requirements as needed
  • Implements migration efforts with application teams, including data migration.

Qualifications

  • Experience designing and building web application environments on AWS including services such as EC2, S3, Lambda, ELB etc.
  • Previous experience migrating application teams from on-prem to cloud infrastructure preferred.
  • Experience with CI/CD frameworks (ie. Jenkins, Docker, Ansible, Chef, Puppet)
  • Experience in at least one automation and scripting tool experience (ie. Bash, Python, Shell, Perl)
  • Experience maintaining and administering configuration management systems such as Enterprise GitHub.
  • Experience maintaining and administering software build systems such as Jenkins.
  • Experience maintaining and administering artifact repository systems such as Artifactory.
  • Ability to automate workflows through scripting or other technologies such as Ansible or Puppet.
  • Experience determining and implementing middleware requirements.
  • Demonstrated understanding of IT management, administration, applications and system development lifecycles within a substantial IT environment.
