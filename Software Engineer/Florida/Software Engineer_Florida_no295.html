Full Job Description

The Candidate selected will be a member of the Software Integration and Test Team and would have the following primary job duties and responsibilities :
  • Provide software engineering services for the analysis, design, development and maintenance of complete web-based application systems supporting the Naval Reserve Officers Training Corps (NROTC) and Navy Junior Reserve Officers Training Corps (NJROTC) programs.
  • Develop applications in ASP.NET Web forms and ASP.NET MVC using the Microsoft Visual Studio 2015 (or later) product suite and C# programming language. Develop the Graphical User Interface (GUI) utilizing HTML5 standard for page structure, CSS and Bootstrap for styling, certain client-side technologies for efficiency such as JavaScript and JQuery. Utilize AJAX for targeted server data requests vice full-page rendering.
  • Design, develop and maintain standard reports using modern reporting tools like Crystal Reports Developer (v11 R2 or later), and non-standard reports or data retrieval using various tools based in Structured Query Language (SQL).
  • Perform unit-level testing of application products to validate readiness of the product to enter into formal testing phase.
  • Adhere to established software Configuration Management standards by utilizing Microsoft Team Foundation Server (TFS) source code repository for code check-in/out.
  • Provide input to assist in formulating test scripts and plans for formal testing of software products. Provide information, guidance, procedures and/or other applicable documentation to assist independent testers in the QA testing process.
  • Respond to Test Defect (TD) reports by analyzing the use-case situation reported, then taking appropriate action to resolve the defect.
  • Ensure products comply with Cyber Security directives and guidance from Department of Defense (DoD) and Department of Navy (DoN) authorities, and are specifically in-compliance with Security Technical Implementation Guides (STIG) instructions for the application (APP STIG).
  • Design, develop and maintain back-end database objects and stored procedures using Structured Query Language (SQL), Data Dictionary Language (DDL) for Oracle 12c (or later) database management system, that are required by the NETFOCUS component applications.
  • Coordinate with Database Administration (DBA) group on issues that require higher user-privilege to address or implement, such as table space maintenance, schema creation, restarting of application pool, and/or for assistance in trouble-shooting issues of a systemic nature.
