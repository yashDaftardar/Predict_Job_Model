ARA Emerald Coast Division is seeking a senior-level software developer to support the development of the Joint Anti-Air Combat Effectiveness (J-ACE) operational tool used by tri-Service aviators, mission planners, and analyst. J-ACE includes models of aircraft and missiles and is used “under the hood” in every fighter squadron in America for training, developing and validating operational tactics and for mission debriefing. J-ACE evaluates overall weapon system effectiveness from weapon launch through target intercept and displays results in a graphical 3D environment.

The candidate must have the ability to work independently, as well as with a team of developers and analysts/engineers trained in operational tactics, mathematics, and other skills. The ideal candidate will be proficient using an Integrated Development Environment (IDE) such as Microsoft Visual Studio to efficiently troubleshoot and debug software. The J-ACE team utilized Agile practices.

Job Duties include:

  • Write code
  • Perform requirements analysis and develop software functional derived requirements
  • Develop software designs that meet software requirements
  • Utilize version control software such as GIT
  • Utilize application lifecycle management tool (e.g. - Microsoft Team Foundation Server (TFS))
  • Evaluate proposed software changes or additions
  • Prepare software documents

Required Skills:

  • Must have a strong knowledge of the C++ programming language
  • Familiarity with thread-safe, event-driven, and object-oriented programming
  • Must be able to work well within an Agile team environment and be able to adapt quickly to change
  • Good writing and verbal presentation skills
  • Active SECRET clearance ( or eligible to receive clearance )

Preferred Skills:

  • Experience developing large C++ applications
  • Experience using a build automation tool such as CMake and using a unit testing library such as Google Test
  • Proficient in an Integrated Development Environment (IDE) such as Microsoft Visual Studio to efficiently troubleshoot and debug software
