Full Job Description

PRIMARY FUNCTIONS:


  • Develop production ready code from specifications

  • Gather and document business requirements in FHCP approved template

  • Write unit tests for code changes based on business requirements

  • Develop code that fulfills business requirements and adheres to standards and best practices

  • Communicate status daily for assigned tasks through the project management system

  • Peer code review of software modifications prior to unit testing

  • Accurately estimate the time to complete assigned tasks and document within project management system

  • Communicate timely with project managers and supervisor on work efforts

  • Align work on assigned tasks to Change and Release Management and SDLC policies and procedures

  • Work effectively with inter-disciplinary user groups to define project requirements

  • Maintain and resolve issues with current applications

REQUIRED QUALIFICATIONS, SKILLS, AND EXPERIENCE:


  • Bachelor’s degree in Management Information Systems or related field, or over 3 years working as an FHCP Information Systems employee, is required

  • 3 years minimum experience with the following technologies is required:

o Visual Studio 2017 and latest release is required

o Microsoft SQL Server 2016 and latest release is required

o .NET framework versions 2.0 through Core and latest release is required

o C# programming language

  • Knowledge of REST and RPC APIs is required

  • Knowledge of WPF, WCF, WebAPI, WSDL is required

  • Knowledge of HTML5, JSON, XML

  • Knowledge of application security best practices is required

  • Demonstrated proficiency in one or more of the following programming languages:

o JavaScript

o SQL

o VB.NET

  • Experience with Microsoft Team Foundation Server (TFS) 2017 or latest release of Azure Devops is required

  • Experience working from written requirements is required

  • Experience with work estimation is required

  • Strong oral and written communication skills are required
