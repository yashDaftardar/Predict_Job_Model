Software Design Engineer 3

Cole Engineering Services, Inc.
Orlando, FL
Remote
Job Summary: Integrate OneSAF with an external system. Develop/modify OneSAF components, as needed, and diagnose and fix software/interface issues. Support customer development and integration team to identify OneSAF modifications, perform OneSAF updates, support integration events and provide input to documentation.

Responsibilities include: Java development and testing, support for integration and test and Trouble Report fixes.

Essential Job Functions:
· Implement components as directed by more senior engineers
· Participate in Agile software development ceremonies (scrum, retrospective, etc.)
· Support integration into larger system developed remotely

Required Skills:
· Object Oriented development background
· Demonstrated Java development skills
· Ability to use Atlassian tools of Jira, Confluence, Bitbucket for tracking and maintaining development artifacts
· Ability to generate automated tests to support capability regression tests
· Develop and execute test procedures
· Ability to test and debug
· Ability to decompose requirements into Epics, User Stories, and Tasks
· Recent experience in design/development within OneSAF.
· Ability to interface with a customer development team and environment.
· Ability to work independently to identify updates/modification to OneSAF, as needed.

Knowledge in any of the following areas would be considered a plus:
· Integrated development environments such as IntelliJ or Eclipse
· Distributed Development
· Cyber security

Required Education/Experience:
· Bachelor’s degree in a technical discipline such as electrical engineering, computer science, information technology, systems or software engineering from an accredited college or university. May consider experience in lieu of degree.
· 5+ years of related job experience in working on government DoD programs

Other Requirements:

Please note that pursuant to a government contract, this specific position requires U. S. Citizenship status with ability to obtain a SECRET to TOP SECRET security clearance. Security Clearance requirements will be specified in the Government's Task Order.
