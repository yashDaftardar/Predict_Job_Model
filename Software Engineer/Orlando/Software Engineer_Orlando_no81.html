Work you'll do/Responsibilities

  • Develop functional, system integration, and accessibility test cases that define all the test scenarios that need to be tested
  • Create or coordinate the creation of test data to allow the execution of the test cases
  • Develop manual tests for Android, iOS, and according to the test cases to verify that the code meets the requirements and acceptance criteria as specified in user stories, visual designs, and definition of done
  • Log all defects that are identified during Scrum and System Integration Testing and work with development team to assist in their remediation
  • Monitor automated tests with the Client's automated test suite so that the tests run nightly, and email reports are sent out for failures .
