Job Description

CACI is looking for motivated candidates to join a testing team that supports multiple customers in the Intelligence Community (IC). Candidates should combine a willingness to learn, ability to be self-sufficient and a technical background.

What You’ll Get to Do:

The testing team supports both functional and technical changes in a cloud environment. You’ll be involved with creating, enhancing, and executing tests for web applications and web services. You’ll be involved with automating test cases for reliable, repeatable test execution.


More About the Role:

  • Work with a team of other testers to create, document, and execute test cases
  • Work with JIRA, Confluence, & TestRail to perform tests and record test execution
  • Learn CGI Momentum Financials and custom web-based applications
  • Execute and automate web application testing using python scripting and selenium
  • Execute and automate SOAP and REST web services using SOAPUI and python scripting


You’ll Bring These Qualifications:

  • Undergraduate Degree Required and 3 or more years of experience
  • US Citizen
  • Works independently with general supervision
  • Attention to detail
  • Able to solve difficult and complex problems and issues effectively with guidance
  • Ability to communicate effectively and collaboratively in a team environment
  • Familiarity with test and bug tracking tools
  • Familiarity with testing web applications through User Interface Testing and SQL queries
  • Ability to execute automated and manual test scripts, report findings, and log applicable defects
  • Ability to identify, capture, and create or modify test cases
  • Must have an active Top Secret SCI, with the ability to obtain a Poly

These Qualifications Would be Nice to Have:

  • Experience with relational databases
  • Experience with python or other programming language
  • Experience with git or other version-control system
  • Familiarity with creating test automation scripts
  • Familiarity with object-oriented concepts
