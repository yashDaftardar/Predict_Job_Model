The Challenge:
Are you looking for an opportunity to make a difference and help build cloud-based solutions to our countries greatest intelligence systems? What if you could find a position and team where Cloud and DevOps are more than just buzz words? Where you can make a difference as a Software Engineer in understanding, designing and architecting cloud and hybrid-cloud based systems that leverage the latest tools and methods to increase the speed to mission of new capabilities and maximize efficient systems using modern software techniques.

As a Software Engineer at Booz Allen, you’ll incorporate open source tools and native Cloud resources to develop automation pipelines based on virtual infrastructure to streamline the software development life cycle from development to production. You’ll implement continuous integration and deployment in development, testing, and production environments. This is an opportunity to broaden your skillset into areas like DevSecOps, chaos engineering, Cloud-based development and design, containerization, and serverless while developing the foundation for systems that will improve national security.
