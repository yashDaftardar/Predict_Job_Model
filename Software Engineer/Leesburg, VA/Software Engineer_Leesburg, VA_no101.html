Python Software Engineer


The Python Software Engineer will be focused on release management for Finch Computing you are responsible for helping develop our CLI-based release management tooling and integrating into our AWS CI/CD process, using services like AWS Code Commit, Build, and Deploy, as well as Lambdas that tie into DynamoDB. You will work with a Release Manager to design and implement an efficient release management process for deploying multi-application microservices-based cloud architectures in coordination with Infrastructure, Security, Data, and Application development teams. You will help drive efficiency across the entire build and release process, ensuring timely and accurate releases, eliminating duplication, and leveraging technology to automate releases.


Responsibilities:
  • Design, build, and manage Python software systems to manage and deploy different versions of software across various infrastructure environments
  • Help support and improve the development infrastructure including continuous build environments, continuous integration, continuous deployment, testing platforms, and test automation frameworks
  • Support integration of new technologies into the CI/CD delivery pipeline
  • Respond rapidly to remediate build errors and release issues in coordination with application developers and infrastructure automation engineers
  • Work with project teams to create build release schedules and own the packaging and build process
  • Perform as technical liaison between application development, infrastructure operations, and security teams on every aspect of software builds, environment releases, and release management issues in an agile, microservices-based, multi-repository cloud environment
  • Assist teams with understanding GitOps, versioning, tagging, multi-repo and multi-component releases, branching code, and enabling this all in Jira
  • Provide expertise to team members in applying SDLC principles and methodologies like lean/agile, CI/CD, security, scalability, documentation, as well as refactoring and testing techniques
  • Ensure the appropriate documentation and communication of processes, practices, and troubleshooting techniques
  • Mentor and coach client resource to promote advocacy and support for efficient build and release processes
  • Maintain and manage close interpersonal and professional work relationships with the client, other vendor partners, and internal team members to foster an effective collaboration space
Knowledge and Skills:
  • Python-heavy software development background.
  • strong analytical abilities to analyze the effectiveness of software releases and the configuration of tools used to enable and automate the release process and make improvements based on feedback from the teams.
  • have effective communication and leadership skills, and high EQ to collaborate with colleagues across the software development life cycle to increase the velocity and consistency of deployments and promote quality software.
  • have practical experience with CI/CD best practices in a large multi-team large-scale application environment.
Education and Certifications
  • BS or BA degree in computer science or related technical field or equivalent experience.
  • AWS Certified DevOps Engineer Pro
Clearance Requirement : Public Trust
Location : remote
