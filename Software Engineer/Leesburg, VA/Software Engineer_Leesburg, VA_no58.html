Job Title: Software Engineer Intern

Job Location: Frederick, MD


Job Description :

We are seeking a Software Engineering Intern at our research facility in Frederick, MD. Interns will gain hands-on experience working directly on mission-critical applications. Apply your course work experience in an Agile software development lifecycle and expand your knowledge with the support of a cross-functional team.

Job Responsibilities :

  • Active participant on Software development team utilizing industry best practices
  • Present design to peers and project leadership
  • Implement approved design including component testing
  • Support peer review process as both reviewer and reviewee
  • Create supporting documentation for features
  • Work with Quality Assurance team to ensure task are verified
  • Bring new perspective to the team
  • Participate in Agile team activities to include Sprint Standups and Retrospectives

Job Requirements :

  • Minimum of one year of undergraduate coursework in a technical degree; Computer Science, Computer Engineering, Electrical Engineering, or similar degree
  • Familiarity with Object-Oriented Programming concepts
  • Familiar with Software Development languages
    • C, C++
    • Python
    • JavaScript, Flutter/Dart
  • Familiarity with Service based architecture utilizing asynchronous communication methods
  • Must work well with others and be a team player
  • Must be able to work in an Agile environment
  • Must be a U.S. Citizen
  • Must be able to successfully pass a pre-employment drug screen and a comprehensive background check
Preferred Skills:
  • Experience developing web-based applications using HTML, CSS, JavaScript, or similar languages
  • Experience developing mobile applications using the Flutter framework or Python Kivy library
  • Previous Internship experience
