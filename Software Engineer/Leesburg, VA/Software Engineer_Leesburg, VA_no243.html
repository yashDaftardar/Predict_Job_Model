Full Job Description

Are you interested in enabling end-users and data scientists access to the data they need when they need it? We’re hiring engineers to morph an existing system built on top of ElasticSearch into a Data Hub. This system currently ingests data from multiple sources, normalizes that data into a common data model, provides full-text search across the entire collection, and allows access for downstream dissemination services. Through this re-architecture, we will expand the capabilities to provide a secure, highly available, and scalable environment for both humans and machines to derive the insights and analytics required to make mission-critical decisions. With your help, we will enrich the data with external AI/ML tools via event based processing, provide additional analytics and change detection services, and provide data scientists access to the data holdings to enhance our existing processes and tools.

We’re looking for Software Engineers with knowledge of various data access patterns and underlying technology implementations. You should be knowledgeable on how to properly secure the data from unauthorized access at multiple data access points (e.g. search discovery rules, repository read rules, S3 bucket policies, etc). You should also know how to keep distributed data stores in-sync and how to detect state drift. Finally, you should have excellent knowledge of the AWS ecosystem to support both the existing infrastructure (built on EC2 with CloudFormation) and to help us build the next-generation data hub.

Interested in learning more? Please apply today!

About BTI360

BTI360 is a software development company based in the Washington DC area that provides big data software solutions to clients in the federal government.
