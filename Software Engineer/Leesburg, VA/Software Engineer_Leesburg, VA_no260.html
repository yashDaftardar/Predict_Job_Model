Job Title: Software Engineer

Location: Reston, Montgomery or Remote

Clearance: Secret Security Clearance


IndraSoft, Inc. is seeking a highly qualified Software Engineer to work at Reston, VA or Montgomery, AL. The selected candidate will be part of the Integrated Budget Documentation and Execution System multi-disciplinary team, which is directly involved with the security of budgeting information for various government entities. The Software Engineer will be responsible for contract activities; programming, testing and support of the web-based application developed to support the Air Force budget documentation. He/She will assist in the design and development of IDECS components, units and modules that meet product specifications and development schedules using CMMI Level 3 structured development process, helps analyze software requirements and operational feasibility by evaluating analysis, problem definition, requirements, solution development and proposed solutions determine feasibility of design within time and cost constraints, tests and debugs assigned components and units, assists in writing and updating technical documentation such as interface design descriptions, user manuals, system documentation, and training materials.

Qualifications:

To perform this job successfully, an individual must be able to perform each essential duty satisfactorily. The requirements listed below are representative of the knowledge, skill, and/or ability required. Reasonable accommodations may be made to enable individuals with disabilities to perform the essential functions.

Qualifications Required:

  • Bachelor’s Degree in computer science, software engineering, physics, mathematics or electronics or equivalent background and experience.
  • 5+ years of experience with the following software development application/tools: Microsoft Visual Studio, Programming Languages: Oracle 11G, Java, C#.NET, Microsoft ASP.NET 4.6.1, C#, C++, VB.NET, Visual Basic 6.0, HTML, XML, JavaScript, VB Script, Restful.
  • Experience working with MS SQL Server databases and have ability to design, build and modify SQL queries, stored procedures, triggers, views and tables. Ability to modify or create packages in SQL Server Integration Services.
  • Experience with the System Development Lifecycle (SDLC) for software development.
  • Proven skills for writing code in the above languages, testing software designs, integrating code in current architecture, monitoring system performance, and recommending system re-designs based on results.
  • Experience in applying DoD and AFI procedures and guidance that control DoD networks, system architecture, software development, and network security.
  • Must possess one of the following DoD 8570 IASAE Level II Certification (CASP+ CE, or CISSP, or CSSLP).
  • Must be U.S. Citizen with ACTIVE Secret Security Clearance.
  • Effective Customer Service and management skills/capabilities; including a demonstrated ability to work cross-functionally (internally) and/or externally.
  • Strong English language skills (both written and verbal).


Qualifications Desired:

  • Certified Software Development Professional and or Certified Information Systems Professional.

Essential Functions and Responsibilities:

  • Assist with technical design and development of Microsoft SQL server.
  • Ensure the IDECS applications are functional, on-line, and available for use by the Government to the maximum extent possible.
  • Participate in multi-functional area workgroups that develop recommendations for system architecture, software upgrades and system development.
  • Troubleshoot infrastructure functionality and performance issues including interfaces to databases and web applications.
  • Provide systems operational support, technical skills to monitor, operate and maintain technical infrastructure and test new methods, software, data flow and interfaces with existing systems.
  • Assist with maintaining CI/CD pipeline with DI2E using Jenkins for application publication and deployment to test and production environments.
  • Implement web frameworks used to develop enhanced user experience.
  • Complete all tasks as defined and prioritized in the software development plan ensuring all code is written, tested, and integrated with the rest of the code base and version control system. Design software modules and software interfaces according to specified software. Participates in peer reviews of software design, code and documentation and follows up on the results.
  • Perform security vulnerability findings from scans of source code using Fortify.
  • Ensures a smooth transition for the development team by developing instructions on installation & rollback with recommendations about environment and approaches.
  • Reviews change requests from a software design and implementation perspective, considering the impact on software and respective interfaces and estimating the time and resources needed to implement a solution to the proposal. Constraints such as the system and software architecture, system design and security procedures must be considered.
  • Develop and maintain associated software related artifacts, such as: Software Design Descriptions, Interface Design Descriptions, Software User Manuals, Software Build & Installation Plans, and other technical documents. All tasks and artifacts will be maintained in accordance with applicable DoD and AF directives and policies.
  • Provides advice at the software expert level internally to other developers and users.
  • Provide customer support during semiannual budget cycles.
  • Contributes to the overall software development lifecycle including requirements gathering/analysis, design, development technologies, release/version control, testing, deployment, and support.

Provide post release support in all instances, including training where necessary to internal customers, and occasionally with external customers.
