Position Description


The Senior Developer (C#) will be responsible for the design, development, testing and maintenance of desktop applications, web applications and APIS to integrate manufacturing instruments with the Production Information Management System (PIMS). The primary responsibilities include:

  • Work with process development team and PIMS team to analyze new product manufacturing process and SOP.
  • Architect, design and implement applications and APIs to integrate instruments in product manufacturing process.
  • Maintain and troubleshoot existing code to support daily operations.

Position Requirements

  • BS or MS in Computer Science.
  • 5+ years of professional software development experience.
  • 3+ years of desktop and web application development experience using C#.
  • Expert knowledge of MVC web application framework .Net.
  • Expert knowledge of Object Oriented Programming concepts and design patterns.
  • Strong knowledge of MySQL, SQL Server, SQL and ORM.
  • Experience working with REST APIs, code packages, deployment tools.
  • Experience in Test-driven Development (TDD).
  • Experience with development using SDKs of automated liquid handlers (Perkin Elmer, Tecan, Beckman Coulter, Hamilton, and XL20) is a plus.
  • Knowledge of biotech products manufacturing process and laboratory information management system (LIMS) is a plus.

Personal Requirements

  • Strong analytical and problem-solving skills.
  • Self-motivated and ability to learn new technology and domain knowledges.
  • Strong verbal and written communication skills with the ability to work effectively across various technical and non-technical stakeholders in the organization.
