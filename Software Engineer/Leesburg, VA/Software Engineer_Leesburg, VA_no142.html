The Challenge:

Are you looking for an opportunity to build a powerhouse system that will automate cybersecurity processes and critical missions? You know a good system is more than just a nice-looking interface + data and you need the right combination of clean code, APIs, analytics, and infrastructure to help your team build a user focused solution. As a software engineer, you’ll have the chance to build your skills and solve a problem with a complete end-to-end solution in a fast-paced, agile environment. We’re looking for someone like you to build enterprise web applications from vision to production-ready system.

As a software engineer on our team, you’ll work with analysts and developers to overcome your customer’s most difficult challenges using the latest architectural approaches like microservices, alongside open source frameworks and tools like jQuery, React, and d3. You’ll use your passion to learn new tools and techniques, while helping to identify and implement system improvements. Using coding language, you’ll build automated cybersecurity framework. You’ll analyze the needs and the environment to make sure the solution you’re is developing considers the current architecture and operating environment, as well as future functionality and enhancements.

Work with us as we build systems to change our clients for the better.

Join us. The world can’t wait.


You Have:

  • 3 years of experience with software development teams
  • 3 years of experience with object-oriented programming and design patterns in C# or Java
  • 3 years of experience with version control software, including Git or Subversion
  • 2 years of experience with HTML, CSS, and JavaScript
  • 2 years of experience with relational databases and SQL, including SQL Server, MySQL, or PostgreSQL
  • Experience with C#, .NET, .NET Core, Web API, MVC, or LINQ
  • Knowledge of programming fundamentals, concepts, and language mechanics
  • Ability to comprehend web technologies, web applications, software systems and networks, and security
  • Secret clearance
  • BA or BS degree in Science, Technology, Engineering, or Mathematics

Nice If You Have:

  • Experience with Scripting languages, including Python
  • Experience with DevOps, Cloud hosting, and Systems Administration, including AWS or Azure
  • Experience with Web Service and API development
  • Experience with JavaScript libraries, including jQuery, React, and d3
  • Top Secret or TS/SCI clearance
  • MA or MS degree in CS
  • DoD 8570 IAT II Certification
