Bachelors degree in software engineering or a related technical field is required (or equivalent experience), plus a minimum of 6 years of relevant experience. Agile experience preferred.

Job Description

We are seeking motivated candidates to join our insider threat detection, systems integration team. Our mission-oriented team is responsible for the design, testing, deployment, maintenance, operation, and evolution of the systems directly supporting the insider threat detection program of a large government customer in the United States Intelligence Community (USIC). We have an immediate opening on the team for a motivated Software Developer with a self-starter mindset who is up to date with the latest tools and techniques. This is a challenging yet rewarding position that provides an opportunity to leverage cutting edge technologies in pursuit of a vital mission that protects people, sensitive information/technologies, and the national security posture of the USIC.

Most of the work will be performed in Chantilly, Virginia, which is located approximately 25 miles west of Washington D.C., near the Dulles International Airport. The selected Network Engineer will support a 6+ year contract that General Dynamics recently secured.

Description:

  • Design, develop, and deploy client facing software
  • Interact with other software developers, customers, and stakeholders
  • Solve complex problems and improve processes
  • Collaborate with team members to ensure code is completed efficiently and effectively
  • Perform root cause analysis on production systems

Required skills:

  • Hands-on experience with one or more of the following software languages: C#/.NET, Java
  • Strong knowledge of Microsoft development tools (Visual Studio, SQL Management Studio, etc.)
  • Solid understanding of object-oriented architecture and design
  • Experience building customer facing software
  • Experience designing and developing client-side, server-side, and database components for new application features using C#/.NET, Web API, SQL Server, jQuery, Knockout.js (or similar framework), JavaScript, CSS and HTML5.
  • Hands on experience with SQL Server
  • Solid understanding of Git
  • Practical experience with Agile or Scrum software development methodologies

Preferred/bonus skills:

  • Python application development and/or scripting in a Linux environment
  • Experience with NoSQL databases (MongoDB preferred)
  • Experience with graph databases (Arango DB preferred)
  • Demonstrated experience with Web Server Configurations (IIS preferred)
