Full Job Description

The Software Engineer I is responsible for the development and implementation of software for Arcimoto applications and systems. The Software Engineer I improves and manages software applications from inception to use. This may include allocating resources to complete programming projects and designing detailed plans to ensure the software is functioning properly.

RESPONSIBILITIES AND DUTIES

  • Cloud infrastructure data collection development.
  • ERP system development.
  • Systems integration and interoperability development.
  • Participate in Engineering through multiple development stages: implementation, prototyping, testing, and deployment.
  • Creation and maintenance of development documentation.
  • Completion of assigned projects on time and on budget.
  • All other duties assigned.

REQUIRED QUALIFICATIONS AND SKILLS

  • Experience with common stack development languages and tools
  • Ability to write functional, clear and maintainable code.
  • Experience with version management tools and processes.
  • Experience with developer tools and deployment processes.
  • Must be well organized.
  • Able to concurrently handle multiple projects.
  • Able to work as part of a team as well as independently.
  • Proficient with Microsoft Office or Google Applications software.
  • Bachelor's Degree in computer science or equivalent experience
  • 2+ years related experience

PREFERRED ABILITIES AND KNOWLEDGE

  • Experience with AWS services including EC2, IoT, Lambda and API Gateway.
  • Experience with Python.
  • DevOps and developer support.
  • Linux virtual server deployment and administration.
  • HTML/CSS/JavaScript front end development.
  • Ability to work remotely while completing tasks on time
