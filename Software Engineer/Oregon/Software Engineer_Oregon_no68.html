Job Description


The Data Center Platform Architecture Integration and Validation (PAIV) team is looking for a Software Solutions Engineer to join our dynamic and growing team. As a Software Solutions Engineer you will bring your broad understanding and experience in software to be used to aid in the validation of different server technologies.

Your responsibilities will include, but are not limited to:

  • Work with Architecture Design and pre-silicon validation teams in improving post-silicon test content and providing feedback for future on die features

  • Enablement of new technologies and features for server platforms as well as development of methodologies and validation plans for different server platform products.

  • Create test content targeting of new and existing server platform features for different memory subsystem technologies.

  • Leverage existing solutions for at scale execution of test content and debug failures resulted from test content.

  • Use and enhance test automation suites at scale to orchestrate and execute solutions.

  • Leverage debug tools and techniques to ensure functionality and performance meets platform specs.


Qualifications


Education Requirements

  • Bachelor’s degree in Computer Science, Computer Engineering, or closely related field.

Minimum Qualifications

  • 3+ years of experience in software development with cloud environment solutions or UNIX/Linux platforms including things like development, profiling, debugging, kernel traces and/or debugging.

Preferred Qualifications

Software experience in one or more of the following:

  • MySQL Hadoop HDFS Amazon Web Services.

  • Cloud environments and cloud workloads.

  • Test automation frameworks, test design and execution, CI/CD environments and DevOps tools.

  • Virtualization technology use cases and workloads.

  • Cloud workloads, infrastructure scaling and orchestration.

  • SOA containerization technologies and distributed systems.

  • Embedded software development and silicon validation.

  • Systems/server architecture.

  • Memory hierarchy pools and software use cases for different memory tiers.

Inside this Business Group


The Data Center Group (DCG) is at the heart of Intel’s transformation from a PC company to a company that runs the cloud and billions of smart, connected computing devices. The data center is the underpinning for every data-driven service, from artificial intelligence to 5G to high-performance computing, and DCG delivers the products and technologies—spanning software, processors, storage, I/O, and networking solutions—that fuel cloud, communications, enterprise, and government data centers around the world.
