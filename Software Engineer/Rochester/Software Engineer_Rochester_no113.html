Full Job Description

Description:

The Software Developer is responsible for the design and development of key software components used in the Clerio Vision R&D instrument systems and manufacturing equipment. The individual will work directly with the Engineering and Software Development staff to develop integrated software solutions that allow scientific and manufacturing instruments to be operated efficiently and with direct integration to our common data and workflow management framework.

Responsibilities:

· Design and develop software applications to assist R&D and Manufacturing personnel.

· Participate in cross functional development teams designing next generation of products and manufacturing systems.

· Act as a stakeholder in the component selection process, evaluating for SDK availability and integration effort.

· Investigate errors that occur in the system to perform root-cause analysis and mitigate as necessary through the bug tracking process.

· Help to develop, and ensure adherence to, regulatory and quality standards and processes

Required Qualifications:

· BS Degree in Engineering, Computer Science, Applied Mathematics, Physics or relevant degree.

· 3+ years of R&D /Manufacturing software development experience

· Experience in development and integration of .NET based, object-oriented applications and services using the Microsoft Visual Studio tool chain

· Experience with integration of 3rd party hardware components into .NET software applications

· Experience in leveraging RESTful APIs for client to service integration

· Experience with formal software development processes in a small team environment

· Must have good problem-solving skills, be self-motivated and a team player

  • Experience with peripherals and hardware interfacing via serial, USB, and other interfaces

Desired Qualifications:

· Experience with software development practices for Medical Devices or another highly regulated industry – design control, risk assessment, defect tracking, configuration management, etc

  • Familiar with GIT Version control tools
  • Experience maintaining a code and defect database
  • Familiar with Matlab
