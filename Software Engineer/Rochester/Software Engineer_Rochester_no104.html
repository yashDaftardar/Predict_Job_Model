Web Software Engineer

Poseidon Systems has an immediate need for a Software Engineer to support software development efforts in our Rochester, NY location. This is a great opportunity to become a key contributor within a company experiencing rapid growth.

Job Summary:

We are currently seeking a Software developer to join the Poseidon Systems engineering team. This engineer will be responsible for the development of a cross-platform application for the acquisition, analysis, display, and reporting of data collected from Poseidon’s machinery health monitoring systems.

Responsibilities can include but are not limited to the following:

· Develop and test a cross-platform web application

· Develop and test database utilities.

· Define and maintain software build and release schedules factoring in customer needs and company objectives.

· Generate design documentation (requirements, test procedures)

· Develop additional ancillary mobile applications for interfacing with products in the field.

Requirements:

· Proven background in full stack mobile and web application development

· Proficient in JavaScript, HTML, and CSS

· Proficient in Node.js and Express.js

· Proficient in Angular (experience with Ionic a plus)

· Experience with MongoDB, InfluxDB, GraphQL, Google Cloud Platform

· Strong User Experience and User Interface skills

· Excellent documentation skills

· Creative, reliable, highly-motivated, hard-working and resourceful
