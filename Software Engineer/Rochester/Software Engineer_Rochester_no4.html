Job Description

Mastodon is looking to hire a Software Test Engineer to support our expanding product portfolio. As a Software Test Engineer, you will be responsible for qualifying and introducing new product platforms to operations, developing and sustaining test software platforms for enterprise system distribution, and developing software interfaces to products. You will also develop Graphic User Interfaces (GUI’s) for complex engineering systems. You will be operating in a dynamic manufacturing and engineering environment with continuous interaction with key Mastodon personnel.

Primary Responsibilities

A candidate in this position will perform many duties, including the following:

  • Read and interpret interface control documents (ICD’s), and produce optimal product test solutions at board and module level.
  • Performs application and system testing functions including test data creation, functional, regression, and complete workflow testing.
  • Develops and maintains test cases, test procedures/scripts, and/or test data for all testing functions.
  • Completes analysis against requirements and technical specifications to build appropriate test plans and data.
  • Develop and produce functional software drivers for 3rd party test equipment
  • Qualify new product software builds and deployments for manufacturing products.
  • Synthesize test documentation detailing product test plans and procedures for all Mastodon products
  • Recommends and implements opportunities for productivity and process improvements.
  • Strong understanding of UI, cross-browser compatibility, REST API’s, general web functions and standards
  • Familiar with modern web frameworks (Vue JS, React, Angular, etc.)

Minimum Qualifications

The candidate qualifying for this position will meet the following experiential requirements and will be able to demonstrate the depth of that experience during an interview:

  • Bachelor’s degree or higher in computer engineering, computer engineering technology, or computer science with 0-2 years of industry experience in a software test development environment.
  • Minimum 3.0 GPA
  • Understands basic electrical concepts, terminology, and theory.
  • Ability to demonstrate advanced operating level knowledge of the following tools: Python, MATLAB, HTML, JavaScript, C#, Tortoise GIT, GIT Bash, JIRA, Microsoft Office Suite.
  • Ability to construct, modify, and interpret automated test programs for electronic subassemblies through final assembly written in Python and MATLAB
  • Ability to achieve and maintain a DOD security clearance

Desired Skills

  • FPGA programming, Microprocessor Programming, JTAG in Systems Programmers, IEEE 1149.1 Boundary Scan
  • Experience with PC and machine control interfaces (USB, GPIB, ETHERNET, HDMI)
  • Embedded communications experience (SPI, I2C, RS-232, LVDS)
  • Linux OS Experience
  • Experience with relational databases such as PostGreSQL.
