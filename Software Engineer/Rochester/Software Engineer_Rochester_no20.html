Specialist, IT Development Sr. - Job Description

  • Analyze and estimate needed modifications to existing custom and Infinium software that resides on an AS/400 (iSeries).
  • Responsible for development work involving the following Applications:
>>>>> Accounting (GL, AP, FA, Projects)
>>>>> Materials Management (Supply Chain/Purchasing)
>>>>> HCM (Human Capital Management – HR/Payroll & Time Entry)
  • Support all Infinium applications on an as needed basis.
  • Work with customers, and team support members, to identify and fine-tune project requirements.
  • Prepare required project design documentation.
  • Design, develop, and programmer acceptance test all modifications and new development.
  • Facilitate/address all user acceptance testing issues for projects.
  • Troubleshoot and support ERP applications which, includes on-call rotation.
  • Follow SLC procedures for all projects.
  • Comply with internal and IS policies and procedures including IS Governance, Change Management, etc.
  • Manage concurrent projects.
  • Work in a team-oriented environment supported by IS management, system analysts, testing analysts, and other programmers.
  • Some travel may be required.


Qualifications:

  • Strong AS/400 (iSeries) and RPGLE programming background required (4 + years experience).
  • ERP System/Application experience required (2 + years).
  • Experience with CL programming and OS/400 required.
  • File transfer (ftp, Excel to AS400, AS400 to Excel) required.
  • Excellent analytical and system design skills required.
  • Strong analytical and problem-solving skills required.
  • Experience with Microsoft Excel and Word applications required (2 + years).
  • Must be detailed oriented.
  • Experience with CVS source control, DBU, STRDBG, and SQL a plus.
  • Experience with Turnover application a plus.
  • Understanding of Agile Methodologies and JIRA experience a strong plus.
  • Experience with a testing application, i.e. HP Quality Center, a plus.
  • Web apps/network experience a plus.
  • Excellent verbal and written communication skills.
  • Self-motivated with strong interpersonal skills.
  • Familiar with Remedy, Change Management, and IS Request systems a plus.
  • Participate in ‘team’ environment to accomplish required goals as a team.
