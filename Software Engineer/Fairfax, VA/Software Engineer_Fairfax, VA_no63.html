Job Description:

We are seeking a Junior Software Engineer, located in the Springfield, VA area, to work as a collaborative member of a software engineering team working to enhance Novetta's flagship C4ISR product, Ageon ISR. In this position, you will interact with advanced sensor systems out of Fort Belvoir, VA and create modern software solutions for ISR operations.

Ageon ISR is a command and control (C2) web application and visualization platform that synthesizes data from a vast array of sensors and systems, allowing an operator to see a real-time, geospatial common operating picture (COP) of their mission environment, as well as take manual and automated command of controllable mission assets.

Responsibilities include:

  • Implement new software capabilities in an agile software development life cycle (SDLC)
  • Work with advanced sensor equipment on programs supporting operations of national defense
  • Implement high-quality software capabilities that comply with feature design criteria and team standards
  • Troubleshoot and resolve software application issues in customer environments
  • Travel to customer locations to perform sensor integrations, functionality tuning, and/or troubleshooting
  • Collaborate with a hardworking team in functional disciplines such as: quality assurance (QA), information assurance (IA), system engineering (SE), product management (PdM), and project management (PM)

Basic Qualifications:

  • Experience with Python (2+ years).
  • Experience with machine learning or statistics (2+ years).
  • Ability to work both independently and collaboratively.
  • High levels of curiosity, creativity, and problem-solving capabilities.
  • Strong written and verbal communication skills.
  • Comfortable navigating the command line.

Desired Skills:

  • Ability to implement and troubleshoot application software using C#/.NET
  • Proficiency with Visual Studio (IDE) and Git (VCS)
  • Experience creating web applications using HTML, CSS, and JavaScript
  • Ability to learn quickly and adapt to using new processes and technologies
  • Knowledge of good software design patterns and best practices
  • Strong written and verbal communication skills
  • Strong team collaboration skills

Security Clearance:

Must be eligible to obtain a U.S government security clearance.
