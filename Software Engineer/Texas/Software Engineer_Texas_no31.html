Job Brief:

We’re looking for mid level Software Engineer to create and deliver outstanding experiences for users and help implement improvements to our API architecture and codebase using Go and PHP. This is a great opportunity for anyone wanting to learn or work with Go and build processes for moving data around.

What You’ll Need:

  • Mid level proficiency in back end software development (preference for backgrounds in Go, PHP, Python, or C#)
  • Experience in any modern back end frameworks
  • Adept at Git
  • Willing and excited to learn and work with new technologies/languages
  • Knowledge of SQL and experience developing APIs is a plus
  • A passion for engineering, learning, and teamwork is a must!

What You’ll Do:

  • Utilize cutting edge technologies to develop and support back end solutions and APIs which deliver outstanding user experiences across multiple form factors with attention to functionality and reliability
  • Learn technologies and languages on the job (including Go and PHP, if you haven’t used)
  • Troubleshoot and optimize back-end application performance
  • Work closely with data and have opportunities to influence data retrieval, transformation, and enrichment
  • Collaborate with peers and other teams
