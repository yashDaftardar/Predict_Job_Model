Lutron Electronics Co., Inc. is seeking an Embedded Software Engineer (multiple openings) for its office in Austin, TX to write and review codes for new software features and consult on technical problems. To be considered for the position, interested individuals must apply online and submit academic transcripts at careers.lutron.com using Description Keywords “Embedded Software Engineer.” #LI-DNI
