What you'll do
Johnson Controls has an exciting opportunity for a Software Engineer with Android developing experience to develop and maintain features on an Android system app. Your primary focus will be the development of an Android application that communicates with a router using MQTT and REST API calls. You will be working along-side other engineers working on different layers of the infrastructure. Therefore, commitment to collaborative problem solving, sophisticated design, and creating quality products is essential.

How you'll do it

  • Translate designs and wireframes into high quality code
  • Design, build, and maintain high performance, reusable, and reliable Java code
  • Ensure the best possible performance, quality, and responsiveness of the application
  • Identify root causes of bugs reported by customers and beta testers
  • Help maintain code quality and organization
