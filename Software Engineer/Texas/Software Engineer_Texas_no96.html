Full Job Description

We are looking for a skilled and motivated engineer to work on HP TechPulse, an enterprise product. TheTechPulse development team is part of HP Inc, Personal Systems Software organization. TechPulse product provides analytics and telemetry services to enterprise customers. You would be required to solve complex engineering problems and work with diverse and global teams. You should be a quick learner and think independently.

Responsibilities:

  • Develop, debug and create solutions using VC++, .C#, .NET framework
  • Create high level designs and when required, work with other designers/ architects.
  • You may also need to understand, debug and create proof of concepts around Windows Technologies.
  • Co-work on features with global teams.
  • Deliver using agile process using CI/CD pipelines.
  • Participate in code reviews.

Education and Experience Required:

  • Bachelor's or Master's degree in Computer Science, Information Systems, or equivalent.
  • Typically 0-2 years experience.

Knowledge and Skills:

  • Experience or understanding of software applications design tools and languages.
  • Good analytical and problem solving skills.
  • Understanding of design for software applications running on multiple platform types.
  • Understanding of basic testing, coding, and debugging procedures.
  • Strong Object Oriented Programming Skills, Data Structures and Design Patterns.
  • Good written and verbal communication skills; mastery in English and local language.
