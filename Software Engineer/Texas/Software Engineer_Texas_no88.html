Responsibilities:
Reporting to the Director of Engineering, the Build Engineer will support the Engineering and Production teams, by orchestrating the automated CI/CD pipeline, ensuring games flow efficiently through build, packaging, distribution, and release. This role will master the details of build, packaging, and release across multiple desktop, mobile, and console platforms and distribution channels, ultimately playing a key role in breathing life into a range of AAA games.


Essential Functions:

  • Design and maintain build scripts for numerous game titles, across multiple build hosts (Windows, Mac, and Linux) and multiple target platforms (Windows, Mac, iOS, PS4, Switch, etc)
  • Work with various teams to orchestrate build pipelines on TeamCity or Jenkins build farm
  • Collaborate with IT to ensure scalability and efficiency of the build farm, including cloud resources, flexible build node configuration, artifact distribution, and implementation of build pipeline metrics
  • Create and maintain documentation of build pipelines both at the code and end user level
  • Master the details of build, packaging, and release across multiple desktop, mobile, and console platforms and distribution channels
  • Manage game configuration and upload for online stores such as Apple App Stores and Steam
  • Provide support for integration of third-party or in-house tools, such as telemetry or launchers, into game projects

Preferred Qualifications:

  • 5+ years software development experience
  • 3+ years game industry experience in a build / release related role
  • Experience with configuring and maintaining TeamCity or Jenkins
  • Proficiency in one or more scripting languages, such as Lua, Groovy, Python, or Bash
  • Experience with Unreal Engine build pipeline
  • Experience with software development tools, such as Visual Studio, Xcode, and CMake
  • Experience building software for a mobile or console platform
  • Strong knowledge of a version control system, preferably Perforce
  • Strong knowledge of one or more host environments: Windows, Mac, or Linux
  • Working knowledge of networking and network file systems
  • Experience with cloud storage, replication, and compute instances
  • Basic knowledge of RESTful APIs
  • Collaborative team player and problem solver
  • Experience with bug-tracking software
  • Degree in Computer Science, Math, or related discipline(s)
