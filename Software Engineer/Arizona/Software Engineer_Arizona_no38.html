Job Description


This Engineer will work with a team of Software Engineers in a Technology Development environment to develop critical software for the High Density Modular Tester.

The High Density Modular Tester is the system tester used at Intel production to test Intel Silicon and assembled products. The High Density Modular Tester Operating System team support the entire fleet of Intel Products. Our Testers are used in the factories to ensure Intel Product Reliability.

HDMTOS owns the full stack of the tester software from BIOS to FPGA to higher levels in the stack all the way to Graphical User Interface.

This Software Engineer will participate in the development of SW and contribute to Software Releases. Development is done in C++, CSharp and Python scripting. This Engineer will also have an opportunity to participate in an Agile Environment and lean about tools such as GIT for code repository ADO MS Azure for SW Management among others.

Responsibilities may be quite diverse of a software technical nature and includes but is not limited to developing software for HDMT Tester Operating System . Developing software based tools to automate engineering data analysis tool error checking tool debug software tool test integration and more. Developing software test content and improving test and Dev Ops infrastructure. Assisting engineers in performing software test debug troubleshooting.

This is a 1 year contract with the posibility to extend it.
