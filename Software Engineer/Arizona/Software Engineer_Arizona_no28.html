Overall Purpose

This position develops, tests and maintains software product applications.

Essential Functions

  • Assist with designing, developing, documenting, testing, and debugging new and existing software systems and/or applications.
  • Participate in full development life cycle including requirements analysis and design.
  • Understand technical specifications based on conceptual design and stated business requirements.
  • With technical guidance, can support, and document software functionality.
  • Revise programs with technical guidance as needed.
  • Participate in software design meetings.
  • Able to assist in refining, testing, and debugging programs to meet needs.
  • This position is closely supervised and has limited independent judgment.
  • Complies with all security policies and procedures, to ensure that the highest level of system and data confidentiality, integrity and availability is maintained.

Desired Skills

Automation

  • Ability to use Unix/Linux command line programs and create/edit scripts.

  • Able to analyze results of automated deployment scripts and makes modifications.

Design

  • Attends design review and project meetings as required.

  • General understanding of software programming languages.

Coding

  • Familiarity with Java or C# or C++ or Objective-C language programming skills.

  • Familiarity with SQL or PL/SQL and relational database concepts.

  • Experience with software engineering principles.

  • Ability to meet deadlines and perform well in a team environment.

Testing

  • Able to maintain test scripts as needed.

  • Produce and report project test metrics.

  • Documents test cases and defects as needed.

  • Performs unit, functional, load, user interface, performance, regression and system integration testing on new code or modified code prior to production releases.

Support

  • Ability to support/maintain existing products and add new features as requested by customers.

Collaboration

  • Agile/Lean methodologies experience is desirable

  • Able to estimate some project activities and tasks.

Experience

Education and experience typically obtained through completion of a Bachelor’s Degree in Computer Science and/or 1 or more years of related industry experience.

Preferred skills:

  • 1-2 years of Java programming experience
  • AWS
  • Python

Physical Requirements

Working conditions consist of a normal office environment. Work is primarily sedentary and requires extensive use of a computer and involves sitting for periods of approximately four hours. Work may require occasional standing, walking, kneeling and reaching. Must be able to lift 10 pounds occasionally and/or negligible amount of force frequently. Requires visual acuity and dexterity to view, prepare, and manipulate documents and office equipment including personal computers. Requires the ability to communicate with internal and/or external customers.

Employee must be able to perform essential functions and physical requirements of position with or without reasonable accommodation.

The above job description is not intended to be an all-inclusive list of duties and standards of the position. Incumbents will follow instructions and perform other related duties as assigned by their supervisor.
