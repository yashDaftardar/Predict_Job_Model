Position Overview:

The Edgesource Corporation (Edgesource) is seeking a Web Developer to support their US Army Intelligence Center of Excellence (USAICoE) customer in Ft. Huachuca, AZ. The person in this role will provide technical support to the USAICoE?s Intelligence Knowledge Network (IKN), a knowledge management portal providing tools and resources to the Military Intelligence (MI) community. The IKN is the flagship portal for the MI community with reach-back support and tools that enable MI professionals to easily organize, share, and exchange knowledge. Due to COVID protocols, this position currently allows for 2 days of remote work with 3 days onsite.


Position Responsibilities:

  • Design, develop, troubleshoot, debug, and implement software code for the IKM portal and its components.
  • Work directly with graphic designers and other members of project teams to develop the site concept, interface design, and website architecture.
  • Assist with interface implementation efforts.

Required Skills/Experience/Education:

  • Experience developing web applications using HTML, .Net Framework (C#), MVC, Object-Oriented Programming, SQL, and JavaScript.
  • Knowledgeable of web application IIS setup.
  • Experience with Bootstrap and jQuery preferred.
