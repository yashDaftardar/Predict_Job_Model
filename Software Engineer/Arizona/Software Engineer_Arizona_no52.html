Rancher Labs, a fast-growing and well-funded software startup, is seeking a UI engineer in to join our development team located in Tempe, Arizona. Rancher Labs builds an open source Kubernetes platform called Rancher and develops a number of widely-used open source projects including k3s, Rio, Longhorn, etc. Rancher products are popular with the DevOps community and are known for their highly-polished and intuitive UX. Apply for this role if you are interested in the following:

  • Continuing to grow your JavaScript framework and UI development skills
  • Learning the latest cloud and Kubernetes technologies
  • Being part of a fast-growing startup environment
  • Writing open source software for millions of users world-wide

Requirements:

  • 3+ years of UI development and JavaScript experience
  • Strong background with single page apps using either Ember, Vue, Angular or React
  • Ability to work with multiple JavaScript frameworks or learn new JavaScript frameworks when required
  • Strong HTML5, CSS3 and LESS or SASS background
  • These skills would be nice to have though not required: Docker, Kubernetes, Linux, AWS, REST API design, GitHub
