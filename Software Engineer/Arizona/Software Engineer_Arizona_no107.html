Fender Musical Instruments Corporation ("FMIC") is located in Scottsdale, AZ . Our brand mission is to accompany each player at every stage with products and brand experiences that fuel the pursuit of musical expression. Within Fender Global Research & Development , we are focused on building products that resonate with our customers.

An American icon, Fender was born in Southern California and has built a worldwide influence extending beyond the studio and the stage. A Fender is more than an instrument, it's a cultural symbol that resonates globally.

We are looking for a Senior Design Engineer, Embedded Software to work on guitar and bass amplifiers. You will have the opportunity to learn from existing products, but also to chart the course for future technology-based amplifiers. You are a self-starter, with a proven ability to continuously learn on your own.

About this job:

  • Collaborate with coworkers as a member of a product-focused SCRUM agile team
  • With guidance implement application features in C++ and platform features in C
  • Compose unit tests
  • Write development tools and test scripts in python
  • Help maintain automated builds and software releases
  • Propose, document, and implement better ways of doing things


Who you are:

  • Someone excited to use embedded software to create tools for musicians
  • Thinks logically, and acts based on the facts
  • Breaks big problems into smaller ones, estimates work, and stays organized
  • Has experience with microcontrollers or Linux
  • Has C++ experience and exposure to other high level languages (python, C, java or others)
  • Applies software design patterns and object oriented design where appropriate

What you need:

  • Bachelor's degree or higher. CE/CSE, CS, EE preferred.
  • 3+ years professional software development experience
  • 2+ years C++ experience
  • 2+ years combined experience with other high level languages (python, C, java, C#)
  • Familiar with Git or other version control
  • Everyday experience using Linux for development, including the command line
  • If you have more or less experience, that's fine too. Apply to start the conversation.

Bonus Points:

  • Music related hobby projects
  • Demonstrated contributions to open-source software
  • Experience using CI tools like TeamCity, Jenkins, Github Actions, or Circle CI
  • Yocto / OpenEmbedded recipe development
  • Experience debugging embedded systems
  • Graphical User Interface development for embedded devices using frameworks like QT
  • Real-time audio processing experience, multicore programming experience
  • Linux kernel development, or real-time patch experience
  • BLE, WIFI, Bluetooth Streaming on Linux or microcontrollers
  • Familiarity with embedded peripherals like GPIO, UART, I2C, SPI, ADC, USB
  • Understanding of DSP fundamentals, familiarity with amp modeling and effects algorithms
  • Familiarity with the sound design process for music products with speakers

About Fender Musical Instruments:

Fender Musical Instruments Corporation (FMIC) is one of the world's leading musical instrument manufacturers, marketers and distributors, whose portfolio of brands includes Fender®, Squier®, Gretsch®, Jackson®, EVH®, Charvel® and Groove Tubes®, among others. For more information, visit www.fender.com.
