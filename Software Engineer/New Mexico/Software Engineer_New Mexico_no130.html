Overview:
Encantado Technical Solutions is currently seeking an Application Programmer II to support an IT Managed Services program for the Department of Energy at Sandia National Laboratories (SNL). This position will be in Albuquerque, NM.
Responsibilities:
Position Responsibilities:
  • Developer will work as a .NET software developer with experience solving hard problems in building real-time, state of the art, web applications.
  • Individual must be well rounded, having experience with real-time databases, distributed data protocols, and modern web development methods and tools.
  • Assist with analysis of information requirements.
  • Aid in the evaluation of problems with workflow, organization, and planning.
  • and help in the development of appropriate corrective action.
  • Design, develop, implement, and maintain existing and new technology for several corporate applications.
  • Come up with solutions that may not always be obvious, while still delivering in short time frames.
  • Manage software application code using version control and problem tracking software expediently and accurately.
  • Generate software documentation and reports.
  • Work effectively in an Agile software development (or adapted Agile) environment.
  • Participate in the design of software tools and subsystems to support reuse and domain analysis.
  • Assist Applications Engineer and Applications Programmer to interpret software requirements and design specifications to code and integrate and test software components.
