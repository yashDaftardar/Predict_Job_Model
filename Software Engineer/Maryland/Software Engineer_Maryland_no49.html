Full Job Description

BAE Systems is seeking a Software Engineer to join the software development team who will develop and sustain integrated, feature-rich capabilities that promote modernization of visualization and presentation tools. The software development team will provide application development and support to tools that allow mission analytic users to view existing data or contribute new information in a collaborative environment.

The Software Engineer for this position may be professionally focused on UI or may be a full stack SWE with an interest in performing UI development. The Software Engineer should have some level of experience with UI technologies employed on the project and those with experience will be given preference. UI technologies include modern JavaScript Frameworks (React or Vue preferred), node/npm, webpack, Git and Jest testing.

Additional information:
  • Approximately 15 team members made up of software, system, and test engineers
  • Work performed in contractor facility in Annapolis Junction.

Required Education, Experience, & Skills
  • Bachelor’s degree in Computer Science or related discipline from an accredited college or university is required. Four (4) years of SWE experience on projects with similar software processes may be substituted for a bachelor’s degree.
  • No demonstrated experience is required.
  • Analyze user requirements to derive software design and performance requirements
  • Debug existing software and correct defects
  • Provide recommendations for improving documentation and software development process standards
  • Design and code new software or modify existing software to add new features
  • Integrate existing software into new or modified systems or operating environments
  • Develop simple data queries for existing or proposed databases or data repositories
  • TS SCI level clearance with the appropriate poly required
