Full Job Description

OneMain is growing, we are looking for a SQL Software Engineer to join our team in Baltimore. You will be performing the following tasks:

  • Performs development and support involving Microsoft SQL Databases, Stored Procedures, Views, Triggers, and SSIS/SSRS packages.

  • Analyzes requirements and develops recommendations for new functionality.

  • Analyzes existing processes/procedures and develops recommendations for improvements.

  • Provides support for Test and Production Environments.

  • Participates in planning and implementation of project code deployments and infrastructure updates.

  • Collaborates with Test Planners, Development Leads, and Business Analysts to understand projects and test requirements.

  • Participates in work groups formed by several disparate teams.

  • Provides near zero defect quality.

Basic Qualifications:

  • Minimum B.S. Computer Science or related discipline from accredited college or university or 4 years equivalent experience

  • At least 2 years of experience designing/developing with Microsoft SQL Server

    • Database Design, Integration Services (SSIS), and Reporting Services (SSRS)

  • In-depth knowledge in creating various data base objects: SQL Queries, Stored Procedures, UDF, Views, Triggers

Preferred Qualifications:

  • At least 2 years of experience with:

    • Software Development Life Cycle (SDLC) Phases

      • Requirement Gathering, Analysis, Designing, Coding, Testing, Implementation, and Preparing Documentation

    • Designing/Developing Web Applications, Reporting Services (SSRS), and Integration Services (SSIS).

      • VB.Net, HTML, CSS, JavaScript And JavaScript Frameworks, AJAX, C#, ASP.Net, WebAPI, Web Services, PowerShell

    • System Analysis, System Architecture, And Data Modeling with Relational Databases

    • Developing in an Enterprise Setting

    • Utilizing CI/CD Development Practices (Gitlab)

    • Utilizing Red Gate Tools

  • Excellent Problem Solving and Analytical Skills.

  • Ability to Work Multiple Projects Simultaneously

  • Excellent Communication and Interpersonal Skills

  • Self-Motivated and Quick Learner

  • Proven Experience Working with Clients

  • Experience with Financial/Lending Services

Note: Employment-based non-immigrant visa sponsorship and/or assistance is not offered for this specific job opportunity.
