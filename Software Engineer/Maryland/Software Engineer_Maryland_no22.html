Description

Job Description:

Leidos is seeking an entry level Software Engineer Intern who will be part of a test team with opportunities to test different products, such as web applications, websites, documents and databases. The testing will include both manual and automated testing.

The candidate will provide software development support for all software applications tiers including front-end functional testing, middle-tier testing of web services and other APIs and testing back end ETL and other data processes. In addition, the candidate is expected to learn the EPLC process, software development process, Safe Agile implementation techniques as well as tools used on the program. The candidate will be working with a team of other application engineers. The candidate should be able to collaborate with more senior team members.

Primary Responsibilities

Specific roles & responsibilities for the Software Engineer intern (paired with more senior engineers) position include but not limited to the following: Collaborate with business analysts, other developers, and test engineers on the team to produce high quality software that delivers business value and follows established design patterns and processes – can independently develop, code or modify, test, and debug application programs of varying degree of complexity - Integrate existing software into new or modified systems or operating environments - Troubleshoot production problems within assigned applications software - Assist less experienced staff in user specifications, coding, testing, and debugging. - Implement recommendations for improving documentation and software development process standards - Actively engage in process improvement initiatives

Basic Qualifications

  • Bachelor’s degree in progress – preference for a junior level candidate
  • At least academic experience performing software tests
  • Experience with drafting development processes and test strategies or related academic documentation.
  • Awareness of the software development lifecycle and Agile-Scrum development methodology
  • Strong verbal and written communication skills
  • Able to obtain a Public Trust clearance to get a Health and Human Services badge

Preferred Qualifications

  • Desired Skills include:
  • Knowledge of and academic experience with software development techniques and at least one software language (Angular, Java)
  • Experience incorporating software into agile frameworks
  • Experience working in public health and federal contracting
  • Knowledge and/or experience in using JIRA, Git/Bitbucket
