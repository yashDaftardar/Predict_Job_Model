This role is responsible for working at a system level understanding the designs and interactions of the system hardware and software components. Conducts tests, research, and integration efforts.

Primary Responsibilities:

Responsible for performing the following software development and integration efforts in an avionics environment:

  • Web application development
  • Aircraft messaging
  • Communications based Software Development
  • Flight planning and charting
  • Using open-source software
  • Object-oriented software design and development
  • RF engineering
  • Avionics Integration
  • Communicating technical issues and status in both written and oral form to internal and external customers.

Basic Qualifications:

  • Bachelor’s degree and 3 years of relevant experience OR
  • In absence of a degree, 7 years of relevant experience is required
  • Must be a U.S. Citizen
  • Experience in software engineering with C++, Python, and object-oriented software design and development.

Preferred Qualifications:

  • Ability to adapt to change, determined to accomplish tasks based on program schedule, collaborate with teammates in order to make the best decisions, enjoy learning new technologies, be self-motivated and contribute to a positive work environment.
  • Experience with Angular development
  • Experience in the complete secure SDLC
  • Experience with requirements gathering and technical writing
  • Experience with Cybersecurity
  • Experience in web application development
  • Experience in mobile device software development
  • Linux System Administration skills

Experience with one or more of the following: (listed in order of preference)

  • C++
  • Python
  • Strong database design and advanced SQL (Oracle, PostgreSQL) skills.
  • Javascript
  • Java
  • Unix/Linux]
  • Rapid development
  • Frameworks such as; REST, Web Services
  • C#, HTML, PHP
