What you will get to do:

Senior software engineer designing, developing and debugging full-stack platform software working in a team environment. Candidate must be proficient with .NET / C# and will be working with advanced technologies, integrating and interfacing hardware to the software platform.

  • Develop enterprise platform software using Microsoft .NET / C# environment, building RESTful API’s for web services, using technologies such as Entity Frameworks, LINQ, Angular and DevExpress. Interface & Integrate the latest Industrial Internet of Things (IIoT) hardware to the software platform.
  • Contribute to the software development lifecycle, focusing on designing, implementation and debugging software solutions.
  • Collaborate with a team of software engineers and testers using a scrum based agile software development methodology
  • Work closely with product managers to understand requirements and design software solutions to satisfy customer requirements.
  • Work with QA to debug software issues and ensure robust software quality
  • Building RESTful API’s for web services
  • SQL Server database schema & query design
  • Develop robust, multi-threaded product software, preferably on Windows
  • Work in a highly collaborative team environment, developing commercialized production software
  • Use unit testing tools, profilers, continuous integration and GIT
  • Work with sales & field engineers to develop integrated software / hardware solutions for customer requirements, and to support escalations from customers for field issues.


What You’ll Bring:

Scope:

This position requires an experienced software engineer proficient in Microsoft technologies.

Education & Experience:

  • BS degree or higher in Computer Science
  • Minimum of 10 years’ of experience developing product-based end-customer software
  • Minimum 8 years’ of that software development experience working in a team environment
  • Experience with device layer software and integrating with firmware a plus.
  • Experience working with customers to support production installs, customer implementation support and escalation support is a plus.
  • Experience with workflow engine design (i.e. event bus, message queues, rules engines), and other technologies typically found in N-Tier business applications
  • This position is located in Rochester, NY

Competencies:

  • A winning attitude, self-motivated, personally accountable and determined team player, able to accept direction
  • Awareness of risk management and mitigation techniques

Communications:

  • Key communications contacts (internal/external) and level of persuasion required
  • Strong, frequent written and verbal communication to peer and management audiences
  • Detail oriented, demanding completeness and adherence to highest standards with excellent time and project management skill
  • Experienced with distributed team project development across time zones and cultures
