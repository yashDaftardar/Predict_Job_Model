THE EARNEST RESEARCH COMPANY

Earnest Research is a VC-backed data innovation startup driven to change the way professionals understand consumer and business behavior. Working with world-class data partners, we transform raw data into a source for business and investment professionals to ask better questions so they can make better decisions. We believe, in the right hands, data has the power to change the way we work.


SOFTWARE ENGINEER, INTERNAL SYSTEMS

Earnest is seeking a Software Engineer to join the Internal Systems Team. We are a small team of engineers building business-critical tools and systems for Earnest's client-facing teams. Our software helps organize business knowledge, automate information gathering, and unify our data. We do most of this work in Haskell and SQL and run it on Google Cloud Platform.

On this remote-first team, you will partner with other software engineers, data analysts, and product managers to understand how we organize and work with our data and knowledge systems and help craft solutions to improve quality of work and efficiency across the company. You will have the opportunity to work on modeling tricky real-world ideas and relationships in a database. You will write event-driven software to ingest information from various sources. Your code will parse useful data out of messy, unstructured, documents. You will collaborate with your peers to build web services and custom tools to help analysts streamline their workflow and find the information they need more easily.


TECHNOLOGIES

All of our code runs in Google Cloud Platform with some legacy projects on AWS. Most of our application code is written in Haskell and runs in Docker. We also use a lot of SQL and work primarily with Postgresql and BigQuery.


RESPONSIBILITIES

  • Become a part of a distributed multi-discipline team in an Agile environment to build software critical to our growing business
  • Architect and design data-driven APIs for managing internal knowledge
  • Write functional code for the cloud to automate processes
  • Help design simple data models in a relational database
  • Write code to ingest and parse useful information
  • Support your colleagues with code reviews, knowledge sharing, paired programming, well-timed jokes, and other forms of collaboration

QUALIFICATIONS

Required:

  • At least 2 years of experience working on a software development team building software for the cloud
  • Some experience with relational databases and SQL and a desire to learn more
  • Some experience with (and passion for) functional programming and desire to learn Haskell
  • Curious and eager to "dive deep" into complex domain problems
  • Willingness to collaborate and learn from a team

Preferred:

  • Experience with relational databases, data modeling, and SQL
  • Experience with non-relational databases such as Neo4j, Datomic, or ElasticSearch
  • Experience with a functional programming language: Haskell, Docker, or Nix
  • Google Cloud Platform, especially Cloud Run, Cloud Build, BigQuery, and Cloud SQL
  • Experience with common collaboration tools like Git and Slack
  • Experience writing parsers
  • Experience with automated testing, unit testing, property testing, and CI/CD
