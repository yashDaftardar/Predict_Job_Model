Engineering is at the core of Vestorly and we are looking to add the best, brightest, and most passionate software engineers to our teams. If you are excited about working with a cutting edge stack to solve hard problems, we want to talk to you. Our software engineers have come from a variety of backgrounds and have the talent and dedication to architecting, implementing, and shipping scalable, efficient, and reliable high quality software.

We expect our engineers to have experience in areas such as: TDD, Scrum, web-browser architecture, multi-threaded coding, data-science algorithms, protocol stack, core web languages such as JS/Ruby/Python, and Software design principles like MVC and the Gang-of-Four patterns.

Requirements

  • B.S. in Computer Science or similar technical field
  • Highly talented in software engineering, architecture, and algorithms
  • Passion for best practices, automation, testing, and reviewing code
  • Exceptional coding skills
  • Experience with web development and analytical methods

Server & Back-End Engineering

Build the REST API, models and analytics behind our platform. Design high-performance REST-APIs and back-end content processing algorithms. Experience with high-speed search/eventing systems, MongoDB, Lucene, ELK stack, Ruby on Rails, Python, NLP, and general machine learning techniques.

Client & Front-End Engineering

Work closely with the product team to design and build the most customizable front-end in the industry. Experience with Ember.js, Coffee, SASS, Emblem, OAUTH1/2, REST API.

DevOps & Integrations

Run a set of high-performance servers and write the code to optimize data and monitor infrastructure. Ship sample code to developers and partners and develop security, load and performance testing toolsets. Experience with Ruby/Python, AWS, Varnish, NGINX, OWASP Security, Server Scalability and Performance Tuning.

Content Engine & Data Science

Design high-performance REST-APIs and back-end content processing algorithms. Experience with Python, NLP, and general machine learning techniques.

Product & UI / UX

Build great UI and design great products. Experience with UX, Design, photoshop, wireframing, testing, best-practices, Scrum vs Waterfall methods, and project management.
