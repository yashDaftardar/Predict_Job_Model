AWS DevSecOps Engineer
IT-Digitial Omni Channel
Full-Time


Position Summary

Cyber Security and Risk Management department is seeking a skilled DevSecOps Engineer with a strong cyber security background to join their Engineering and Operations team. Engineer will be responsible for automating cyber security tasks and leveraging new technologies.

Key Responsibilities

  • Maintain GitLab source repository that includes CloudFormation templates to manage security objects via CI/CD pipeline, for 18 AWS accounts
  • Design and implement enhanced CloudFormation template checks, Lambda functions, AWS Config checks, CloudWatch alarms etc. to ensure that all security configurations deployed via CloudFormation and other means conform to application security requirements
  • Review AWS Config and Security Hub reports and work with stakeholders to remediate non-compliance
  • Create and maintain documentation for security configuration, operations and automation aspects of AWS environment
  • Create and maintain automation scripts for non-AWS Cybersecurity applications and business requirements such as security reporting and Identity and Access Management
  • Ensure that all new AWS account and application deployments comply with security standards and policy

Qualifications

  • Experience maintaining AWS Security Products, e.g. SecurityHub, GuardDuty, AWS Config, etc.
  • Hands-on experience with Python programming language and AWS Lambda functions
  • Experience with CI/CD pipelines, e.g. AWS CodePipeline, Gitlab, etc. and using Infrastructure as Code using CloudFormation
  • Building and maintaining cyber security tools and solutions
  • Experience with AWS EKS and ECS services
  • Strong cyber security background
  • AWS Landing Zone an advantage
  • Identity and Access Management an advantage

Minimum Requirements

  • Bachelor’s Degree
