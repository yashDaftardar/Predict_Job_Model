Description

Jenzabar is looking for a talented, experienced and passionate full-stack Software Developer to join our team. Developer work involves designing and developing a web-based administrative application for higher education. Candidate should be proficient in the technologies mentioned below, and preferably have experience with responsive web design techniques to build interfaces which will adapt to many devices. If you enjoy working in a team environment to develop an application that helps colleges achieve their goals, then please consider joining the Jenzabar team.

Responsibilities

  • Develop world class software applications, test software and system interfaces to meet project needs using a variety of programming languages, software packages, utilities, and platforms that work well across multiple devices.
  • Work closely and effectively with your team, management and designers, translate design specifications.
  • Contribute to design, architecture of our platform and interaction with QA to meet business requirements, maintain standards for software development, documentation, and user interface design.
  • To work in full Agile sprint including discovery, application architecture, design input, user interface/database programming and code reviews.

Education

Candidate must possess a minimum of BA/BS degree in computer science or related technical field or have relevant practical experience.

Requirements/Strengths

  • Experience using Java technologies including Jakarta EE, SQL Queries, Stored Procedures, Relational Databases, HTML/HTML5, CSS, JavaScript.
  • Experience in developing cross-browser, multi-tier applications, Enterprise websites, Higher Education websites.
  • Experience with Subversion and/or Git version control and working experience using Eclipse IDE.
  • Experience integrating third party solutions through Web Services and messaging frameworks.
  • Familiar with GWT/SmartGWT and Twitter Bootstrap CSS Framework.
  • Knowledge of software development methodologies and practices, especially Agile/iterative development processes.
  • Proven ability and desire to both estimate tasks and meet deadlines, working effectively with minimal supervision in a team environment.
  • Motivated, hardworking and flexible, good analytical and problem-solving skills.
  • Strong communication skills.
  • Quick learner who enjoys a challenge and can see the bigger picture.
