Job Summary
The Software Engineer Co-op will assist in the development of user interfaces, regression tests, APIs, and/or firmware for high-performance embedded systems.

Detailed Description of Project:

Assist the VTI software team with multiple projects, including:
  • Implement regression tests for iDDS
  • Assist with adding iDDS support on additional existing products
  • Port Java SFPs to Javascript/HTML5
  • Migrate embedded application from Python 2 to 3
  • Assist with updating existing firmware, driver, and test code to support new product variants
  • Create documents & example programs to demonstrate key features for new and existing products

The exact project priorities will be determined after the intern is hired, depending on their skills & interests as well as the software team’s current priorities.
Position Requirements:
Minimum Qualifications (Experience and Skills)
  • 2+ years of Computer Engineering, Computer Science, Software Engineering, or similar college program.
  • Must be a US Citizen or Green card holder for ITAR compliance.
  • Fast learner.
  • Ability to follow directions.
  • Good interpersonal skills, written and oral communication skills, and ability to work cooperatively in a team environment.
  • Good organizational skills
  • Ability to multitask effectively and meet deadlines.

Preferred Qualifications
  • C, C++, Python, Javascript, HTML5, Linux
