Job Title: Software Engineer (Additive and CNC Table Products)
Position Summary and Purpose:
The experienced Software Engineer designs, develops, maintains, and tests Control and Support software for all product lines.
Duties and Responsibilities:
ESSENTIAL:
  • Design, implement, debug and test software products to support new machine designs
  • Maintain and update existing software products
  • Create clear and accurate documentation of software revisions and procedures
  • Serve as technical support for manufacturing and service
  • Provide technical information for the creation of user manuals
  • Provide assistance to other Engineers
Education and Experience:
REQUIRED:
  • Bachelor's degree in Electrical, Computer, or Software Engineering or Computer Science
  • Two or more years software experience
  • Successful completion of large-scale and/or long-duration software project
  • Strong customer orientation
  • Commitment to company values of continuous improvement
  • Adopt and promote lean practices in all activities, considering the elimination of wasteful practices for any goal other than the creation of value for the end customer
Knowledge, Skills, Abilities, Characteristics, etc.
REQUIRED:
  • Must be able to solve challenging software problems
  • Must be fluent in C, C++, and/or C#
  • Prior experience with Microsoft Visual Studio
  • Prior experience with Windows-based industrial control systems
  • Must be creative, energetic and enthusiastic
  • Must be able to successfully design and implement complex software applications as well as modify/maintain existing software
VALUED BUT NOT REQUIRED:
  • Experience with 3D Printing and Slicers
  • Experience with motion control systems
  • Experience with PLCs
  • Experience with CNC machines
Reporting to this Position:
None
