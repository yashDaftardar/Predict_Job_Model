Job Description

VividFront is looking for a Front End Web Developer who is motivated to combine design and development knowledge to build websites and applications. Responsibilities will include translation of the UI/UX designs to actual code. You will work with other developers and designers to bridge the gap between design and development implementation, taking an active role on both sides by defining how the application works, looks and performs.

Responsibilities

  • Develop new user-facing features
  • Build reusable code and libraries for future use
  • Ensure the technical feasibility of UI/UX designs
  • Provide input on designs and best practices
  • Optimize code for maximum speed and scalability
  • Collaborate with other team members

Required Skills & Experience

  • Proficient understanding of web markup, including HTML5 and CSS3
  • Basic understanding of CSS pre-processing, such as LESS and SASS
  • Proficient understanding of client-side scripting and JavaScript frameworks
  • Basic familiarity with tools such as Gimp or Photoshop
  • Good understanding of cross-browser compatibility issues and ways to work around them
  • Ability to understand CSS changes and their ramifications to ensure consistent style across platforms and browsers
  • Proficient understanding of code versioning tools, such as Git
  • Basic understanding of SEO principles and ensuring code adheres to them
  • Ability to set up sites and applications on a local, development environment
  • 3+ years of front end development

Good-to-Have Skills & Experience

  • Basic understanding of Vue,js, KnockoutJS, and ReactJS
  • Experience in the LAMP stack with platforms such as WordPress, Magento, and Laravel
  • Basic understanding of Twig
