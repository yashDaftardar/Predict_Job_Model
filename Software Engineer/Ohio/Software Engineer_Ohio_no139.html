Full Job Description

A successful senior software/cloud developer provides input and support for, and performs full systems life cycle management activities (e.g., analyses, technical requirements, design, coding, testing, implementation of systems and applications software, etc.). He/She supports teams and emerging technologies to ensure effective communication and achievement of objectives. This position provides knowledge and support for applications development, integration, and maintenance. Support the development and operations of cloud based solutions in a fast-paced environment. Utilizing a wide array of AWS Government services and a modern codebase, these solutions are built on leading edge technologies.


Requirements

  • Deep understanding of Nodejs and it’s surrounding ecosystem
  • Experience with Single Page web applications, DOM manipulation and generic HTML, CSS, etc.
  • Experience with different data formats such as JSON, XML, CSV

Desired

  • Serverless archetecture
  • Experience developing solutions for cloud environments (AWS)
  • Working understanding of the JavaScript build process, such as webpack
  • Experience with the Git version control process.
  • A working understanding of webservices, what are the different transports, their uses and restrictions.
  • Experience with general SQL and the different dialects used by vendors.
  • NoSQL experience such as MongoDB, DynamoDB, CouchDB, etc.
  • Experience with a web presentation frameworks such as JQuery, Bootstrap
  • Experiance with Elasticache and Kibana
  • Ability to consistently put forth the effort to finish tasks; meeting client/team deadlines in a full time 40 hour+/week role
  • Working with modern collaboration tools such as Jira, Slack, GitHub, Google Docs, etc.
  • Preferred DevOps interest or competency

Education Requirments

  • This position requires a bachelor’s degree in Computer Science and/or related degree.
