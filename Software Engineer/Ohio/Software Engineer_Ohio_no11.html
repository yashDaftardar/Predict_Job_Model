There is currently available to qualified candidates a position as a Software Design Engineer, new graduate. We are making you aware of this opportunity to generate qualified external candidates.

The selected candidate will be responsible for designing, implementing, and enhancing a variety of software and firmware applications supporting Keithley’s product portfolio. The job involves development and maintenance of software and embedded firmware code that interfaces with test-and-measurement hardware such as SMUs, DMMs, switch matrices, CV meters, pulsers, AFGs, and power supplies. This candidate will design, prototype, code, and unit test product features targeting Keithley equipment. He/she will report to the Software Manager and contribute to a multi-discipline product team, working directly with customers, applications engineers, manual writers, audit engineers, and commercial marketing.

Job Requirements:
Working with Released Product Engineering team, Application Engineers, and Technical Leads to perform assigned development tasks on legacy products.

Design for maintainability, manufacturability, and overall quality

Implement tasks based upon stated goals, independently and collaboratively

Solve difficult problems of diverse scope in creative and practical ways

Create and support design documentation that is maintainable

Support organizational goals for Software development process

Time balancing multiple projects of varying complexity

REQUIRED

A four-year technical degree in Electrical Engineering, Computer Engineering, Computer Science, or equivalent.

Ability to quickly learn new concepts and handle complex coding assignments.

HIGHLY DESIRED

C/C++ programming language experience with excellent debugging skills.

Experience programming in scripting languages such a Python, Perl or Lua.

Working knowledge of software version control, LAB-View, Windows Drivers, embedded Linux, and/or embedded RTOS environments.

Experience interfacing to hardware peripherals via UART, SPI, I2C, Ethernet, USB, GPIB, and/or PCIe.

Familiarity with Object-Oriented design principles, POSIX API, and Agile development processes.

Comfortable collaborating with hardware engineers to program routines to control hardware.

PLUSES

Knowledge of product development for test and measurement equipment for electronics.
