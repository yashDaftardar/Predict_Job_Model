Full Job Description

General Data Company, Inc. ( www.general-data.com ) is a leading provider of products and technologies for identification, tracking and data management applications. General Data delivers innovative bar code labeling, identification, and wireless products and solutions that improve business process accuracy, productivity and performance.

General Data Company is looking for a Software Developer to develop, create and modify computer applications software or specialized utility programs. The ideal candidate will implement innovative customer solutions as well as develop core product enhancements. This position requires an individual who is comfortable working on a variety of projects simultaneously. Application development will include desktop, mobile, web and Windows service application software.

This position will report to our Cincinnati Ohio office. This not a remote position.

Duties and Responsibilities:

  • Develop and create desktop, mobile, web, Android and Windows service application software for customers.
  • Ensure a quality end product by conducting thorough system testing.
  • Estimate and prioritize tasks for software releases and implementation.
  • Assist in creating user manuals and installation documentation.
  • Configure/assemble hardware components.
  • Provide estimates on how long it will take to develop solutions.

Qualifications:

  • Bachelor's Degree in Computer Science, Software Engineering or related field.
  • Working knowledge of:
  • C# or Java.
  • JavaScript.
  • HTML
  • Scripting and/or programming tools.
  • Microsoft Windows (version 10 and higher).
  • Microsoft Office.
  • Local area networks.
  • General knowledge of:
  • Software development and implementation.
  • System analysis and functional design.
  • Structured programming techniques.
  • The Software Development Lifecycle (SDLC).
  • Data modeling and database design.
  • User interface principles and design standards.
  • System design documentation.
  • Documentation and training.
  • Application architectures.
  • Fluent programming/developing in web authoring languages such as AngularJS, React and NodeJS.
  • Working knowledge of label formatting software is a plus.
  • Ability to quickly and independently learn about new technologies, business and technical terms and concepts is essential.
  • Strong analytical and product management skills required, including a thorough understanding of how to interpret customer business needs and translate them into application and operational requirements.
  • Passion for creating a user experience that is intuitive and simple for the end-user.
  • Excellent verbal and written communication skills and the ability to interact professionally with a diverse group, executives, managers, and subject matter experts.
  • Enthusiastic approach to problem solving and teamwork. Ability to work in a self-directed work environment.
  • Highly detail oriented.
  • Demonstrated organizational skills with the ability to manage multiple tasks and meet tight deadlines in an environment with competing priorities.
  • Proficiency in the use of Microsoft Office software, including Excel, Word, and Outlook.
  • Ability to meet or exceed goals set by company.
  • Domestic and International travel may be required (up to 5%).
