Hobart Weigh/Wrap is part of the ITW Food Equipment Group. We are seeking a qualified Software Engineer. This position will be based in Troy, Ohio centrally located to Columbus, Dayton, and Cincinnati.

Job Description

The selected candidate will design, modify, develop, implement, and test high quality Embedded Linux network application software for scale and wrapper products. Individual will be responsible for weigh wrap product graphical user interface, wireless and wired networking, and business logic development.

Business Unit Qualifications

Plan, develop, and implement projects as an individual contributor or as a member of a cross-functional team working on weigh wrap products. Design and/or evaluation of software components and systems of limited scope requiring knowledge of accepted engineering theory and principles. Troubleshoot, test, and implement wireless network solutions and security. Direct interface with customers to define and clarify requirements.

Qualifications

  • BS in Computer Science, Computer Engineering, or Electrical Engineering (4-year degree)
  • Minimum 2 years software development with an object-oriented programming language (C++) or Python scripting
  • Basic understanding of web markup, including HTML5 and CSS3
  • Solid understanding of wireless and wired networks
  • Customer interactions/presentation of technical information
  • 0-15% travel depending upon customer needs

Bonus Qualifications

  • Linux command line, networking, and system administration
  • Linux programming to initiate and maintain wireless and wired network connections, MS Visual Studio experience
  • Qt5 graphical user interface development
  • Java application development
  • Familiarity with JavaScript and advanced JavaScript libraries/frameworks (i.e. ReactJS)
  • Familiar with use of the SVN version control system
  • Experience with development projects employing USB, i2c, SPI, RS422/485/232 protocols
  • SQL Database development concentrated on query language, db creation, and maintenance
