The Software Development Engineer in Test (SDET1) is a novice developer who works in partnership with other SDETs and the application developers to pursue efficiency and quality in the final product. This individual has a strong quality assurance mentality and will create test automation to meet that end.

ESSENTIAL JOB FUNCTIONS

  • Writes test step definitions, creates test data, enhances and normalizes Gherkin scripts, assist in triage of test results, set up tests for execution.
  • Focused on 2-4 Test Automation Types and Tools.

REQUIRED QUALIFICATIONS

Education/Certifications:
  • Associates Degree and/or Technical Bootcamp or equivalent work experience required.

Experience:
  • Candidates can range from entry level to ten years of experience.
  • Proficient in one of the major tech stacks including: JavaScript, Java or C#.
  • Experience in actual work, classroom/bootcamp, or personal projects.
  • Passion for technology and learning new tools and frameworks based upon project work.
  • Experience or mastery of one of the test types is plus: E2E and UI Tests, API and Integration Tests, Performance Tests, Unit Tests (Java and JavaScript), Software Quality Analysis and Linting Tools, Security Scans and Tests, Service Virtualization, Build, Deployment and DevOps Tools.

Competencies and Skills:
  • A self-starter who delivers high quality work and can adapt to new challenges, either on their own or as part of a team.
  • Excellent planning and organization skills, with a commitment to delivering on aggressive deadlines.
  • A strong desire to learn and develop problem solving skills.
  • A basic knowledge of programming and web technologies.
  • A basic understanding of quality assurance practices.
