Job Description Summary

The Embedded Software Engineer will play a crucial role in contributing to the development, testing, verification and continuous improvement of GE Aircraft Engine software. Execute with guidance assigned tasks and contribute to LEAN and Software tools development activities.

Job Description

Roles and Responsibilities

  • Learn engine software build process and execute builds as required
  • Learn engine software testing/verification procedures and execute as required
  • Participate in software tools development
  • Participate in LEAN activities to improve software development process.
  • Provide timely communication to leadership & team members on assigned tasks
  • Interact with Global team on daily basis to meet tasks objectives
  • Learn GE's Engine Software development process
  • Execute assigned tasks consistent with engineering policies & procedures

Required Qualifications

  • Bachelor's degree from an accredited university or college (or a high school diploma / GED with at least 6 years of experience in Job Family Group(s)/Function(s)).

Desired Characteristics

  • Familiarity with developing real time software for embedded systems
  • Working knowledge of C and knowledge with other computing languages such as: assembly, Perl, or Python
  • Familiarity with test systems and methods for embedded software design
  • Experience with requirements documentation and analysis
  • Familiarity with the following tools: DOORs, SCADE, RTC or Synergy / Change, SIMICS,
  • Simulation using NPSS
  • Familiarity with military or commercial software qualification process (MIL-HDBK 516C, DO-178C and related commercial software certification standards for embedded software)
  • Familiarity with digital communication protocols such as: Mil-Std-1553B, ARINC 429, IEEE-1394, ARINC 664, EADIN, and/or CAN bus
  • Knowledge of data structures, algorithms, and software engineering principles
  • Strong oral and written communication skills. Ability to document and execute project tasks.
  • Deliver with focus.

This role is restricted to U.S. persons (i.e., U.S. citizens, permanent residents, and other protected individuals under the Immigration and Naturalization Act, 8 U.S.C. 1324b(a)(3)) due to access to export-controlled technology. GE will require proof of status prior to employment.
