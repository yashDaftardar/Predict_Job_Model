Full Job Description

Job Summary :

You will build/maintain web sites and web applications providing development and technical support in an effort to help establish business requirements and solve business issues.

Duties and Responsibilities:

  • Code specific web and windows applications C#, .NET CORE, MVC, HTML/HTML5, CSS, JavaScript, XML, jQuery, IIS and SQL and additional languages to meet current business needs.
  • Maintain and enhance current web and windows applications and make necessary modifications.
  • Coordinate with team members to build/enhance website and web applications
  • Perform IIS changes, code deploy, and administrative tasks as needed.
  • Work with current vendors and service providers to manage company’s home and community information on remote web sites
  • Work with consultants and temporary staff augmentation configurations on an “as-needed” basis
  • Perform other duties as needed

Required Skills


Additional Skills and Abilities:

  • Self-motivated
  • Detail-oriented
  • Able to complete tasks with little supervision


Required Experience


Minimum Qualifications (Required and/or Preferred) :

  • Skilled in Web Development
  • At least 3 years experience in: .NET, C#, MVC, HTML/HTML5, CSS, JavaScript, XML, jQuery, IIS, SQL, Restful and SOAP web service technologies.
  • Knowledge of SiteCore a definite plus.
