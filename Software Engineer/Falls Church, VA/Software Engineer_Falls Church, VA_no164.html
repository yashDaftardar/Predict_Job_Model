Overview

We are looking to hire a talented .NET Developer to support customized SharePoint solutions as well as business applications for our Federal client. To ensure success as a .NET Developer, you should have extensive knowledge of Microsoft development suites, MS Office software web-based software development using .NET technologies, proficiency with front-end programming languages, and the ability to solve complex software issues. Ultimately, a top-class .NET Developer can create clean, scalable web applications that meet all design and functionality requirements.

Duties and Responsibilities

  • Design, develop, configure, deploy custom .NET Web Solutions.
  • Balance priorities and workload across multiple projects and multiple developers as needed.
  • Design, develop, test and debug software solutions while following software development process, coding, and configuration management standards.
  • Participate in solution design and concept visualization, create user interface mockups, prototypes and demo solutions to stakeholders.
  • Maintain existing applications including troubleshooting and debugging issues.

Qualifications

  • Bachelor's degree (B. A.) in Computer Science or Information Technology
  • 3+ years of experience as a .NET developer/SharePoint developer.
  • Good development skills and experience with web application development using ASP.NET MVC and C#
  • A good understanding of Windows SharePoint Services and other web development experiences will be required in order to implement high quality, scalable and effective SharePoint solutions to support various organizational requirements.
  • Proficient knowledge of SharePoint 2013 or newer, including workflow and content management.
  • Experience building/maintaining Data Access Layer solutions using Entity Framework.
  • Experience with front-end development framework - AngularJS or ReactJS.
  • Experience with MS SQL, Git, TFS, Microsoft SharePoint Designer and other Microsoft Office SharePoint technologies.
  • Knowledge of LDAP/Active Directory authentication and group security as it applies to SharePoint.
  • Experience with REST API development.
  • Familiarity with cloud-based application development tools – Azure is preferred.
  • Preferred to have certifications with .NET/SharePoint Development

Required qualifications for the .NET Web App Developer include:

  • Minimum 3+ years of specialized experience with a BA/BS
  • Alternatively, 1 to 2 years specialized experience with a Master’s degree

Preferred qualifications for the .NET Web App Developer include:

  • ITIL certification; and
  • Microsoft Certified Professional (MCP)

Hours:

  • Available during government closures and on weekends for special projects on as needed basis.
  • Available to take part in out of hours on call rotation.
