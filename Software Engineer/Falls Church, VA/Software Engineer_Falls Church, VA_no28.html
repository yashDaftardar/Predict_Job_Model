Full Job Description

The candidate will work in an Agile software development environment to modernize legacy applications to Angular JS and shall possess the ability to work as an integral part of an Agile team to deliver high quality products and services. Participate in the development, modernization, and enhancement of the systems. This will require a good understanding of the application’s user stories, writing high quality and reusable codes to implement user stories, and thorough unit testing of the user stories.

Primary job duties and responsibilities may include, but are not limited to the following:

  • 6+ years overall software development experience.
  • 6+ years of experience analyzing requirements from users and producing system specifications developing analysis models using Unified Modeling Language (UML) or other data modeling standards.
  • Experience with user story development or analysis in an agile development environment
  • 3+ years of web user interface development experience using following tools – AngularJS / Angular 2+, Bootstrap 3 / 4, HTML5, CSS / CSS3, Jquery, Javascript, Typescript, ReactJS, NodeJS
  • 3+ years of software programming experience – Angular JS or Java, Spring Framework, Spring MVC, RESTful web services, Spring Data, Hibernate / MyBatis, JPA, Eclipse IDE / WebStorm IDE.
  • 1+ years of experience on databases and related tools – SQL, PL-SQL, Oracle, SQL Developer
  • Hands on experience on using and configuration of any of the following web application servers – Jboss, Weblogic.
  • Knowledge of 508 compliant software application is desirable.
  • Knowledge of software testing and debugging is desirable.
  • Ability to work on agile based software development environment.
