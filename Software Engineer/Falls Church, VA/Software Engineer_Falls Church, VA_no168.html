Job Purpose

We are seeking for a creative Full Stack Developer with strong frontend as well as API development experience looking for a challenging job. Developer will be responsible in creating enterprise web applications and associated backend RESTful web services that work on a variety of interfaces - Ability to work in a small, fast paced, collaborative environment. You will also be responsible for integrating the front-end elements built by your co-workers into the application and a full ownership of CI/CD with comprehensive test coverage. Therefore, a very strong understanding of front-end, API and Testing driven Development is a must. Cloud and DevOps related technologies/frameworks familiarity is recommended.


Responsibilities:

  • Develops solutions that encompasses frontend, API Layer, and backend databases.
  • Collaborate with other developers to build APIs and enhance the user experience.
  • Develop responsive products which render seamlessly across multiple devices.
  • Integration of user-facing elements developed by front-end developers with server-side logic
  • Writing reusable, testable, and efficient code
  • Design and implementation of low-latency, high-availability, and performant applications
  • Implementation of security and data protection

Required Skills and Qualifications:

  • 2+ years of experience developing consumer-facing web applications, services, and APIs.
  • 2+ years programming experience in Node.js and frameworks available for it ex Express, StrongLoop, etc.
  • 2+ years of knowledge in a major open-source languages, such as React, Angular, Vue, Typescript, or Java.
  • 1+ years’ experience with developing on AWS or Azure or GCP
  • Understanding of the fundamental technologies of the web: HTTP, SSL, HTML, CSS, JavaScript, and JSON.
  • Focus on component reusability, data sharing and security while complying with Enterprise standards and processes
  • Preferred experience with React and/or Redux
  • Understanding the nature of asynchronous programming and its quirks and workarounds
  • Understanding accessibility and security compliance
  • User authentication and authorization between multiple systems, servers, and environments
  • Integration of multiple data sources and databases into one system
  • Understanding fundamental design principles behind a scalable application
  • Understanding differences between multiple delivery platforms, such as mobile vs. desktop, and optimizing output to match the specific platform
  • 1+ Year of experience on automation testing. Example: Selenium, Cucumber, BDD Framework. Ability to identify and write test cases.
  • Proficient understanding of code versioning tools, such as Git
  • Experience with JavaScript and Node for micro-service development and securing them from internal and external consumption.
  • Understanding of design for scalability, performance and reliability
  • Familiarity with data serialization formats like xml, json
  • Experience with Agile development methodologies and practices
  • Experience/exposure with various database engine technologies such as Oracle, MS SQL Server, Postgres, MySQL and others.
  • BS in computer sciences or related field
