Software Development Engineer
Boston, MA or Exton, PA

PTC, a global provider of CAD, PLM, IoT and AR solutions, is looking for a software developer to join our ThingWorx development team. You will be part of a fast-moving, responsive team using agile development processes to create world-class software products which bring value to our customers. If you are curious, collaborative and motivated, and would like to be part of an exceptional team of experts, then your place is with us!

Day-To-Day:
  • Work closely with developers, QA engineers, product managers and other stakeholders as part of an international team, spread across multiple continents.
  • Implement features in Java and web browser technologies (HTML5/CSS3/JavaScript) using the latest development tools and practices
  • Work in an Agile/Scrum development environment to deliver high-quality software releases
  • Be a part of the DevOps culture

Preferred Skills and Experience:
  • 2+ years of full-stack Java programming experience
  • Good understanding of concepts, and practical experience in web development (JavaScript, CSS, HTML5)
  • Software Architecture/Cloud Deployment & Scalability experience highly preferred.
  • Familiarity with build tools and deployment tools like Gradle and Docker a plus
  • Experience with Azure/AWS services a plus

Basic Qualifications:
  • Bachelor’s degree in Computer Science, Computer Engineering, or equivalent technical experience.
  • U.S. Citizenship status is required as this position needs an active U.S. Security Clearance within one year of employment.
