Title:

Senior Front End Software Engineer, Transport

Description:

Indigo has a number of open roles for software engineers to build the transportation platform we use for moving grain after purchase on the marketplace. Software engineering tasks range from modeling a trucking workflow, designing interfaces against design mockups, to writing form validation supporting a network of tens of thousands of carriers.

We look for a track record of the following:

  • Experience designing, implementing, and shipping live UI code to customers (not necessarily related to logistics or shipping)
  • Having built the wrong system enough times to avoid the common pitfalls, whether building something personally or advising others
  • Willingness to debug problems across the stack, such as networking issues, caching performance, design standards enforcement, or memory leaks

You might be a good fit if you:

  • Know your way around a Unix shell
  • Are self-directed and enjoy designing great user interfaces and abstractions, as well as figuring out the most important problem to work on
  • Own problems end-to-end, and are willing to pick up whatever knowledge you're missing to get the job done
  • Are detail-oriented, and like to get the system working but know when to cut corners versus getting it exactly right
  • Know how to communicate in a remote-first environment
  • Have worked with client-side languages like HTML, CSS, and JavaScript

In this role, you will work on a team with a mix of API and mobile developers towards a common goal.
