We love going to work and think you should too. We are customer-obsessed, work as one team, and strive to be better every day . These are our core values. So it's no surprise that we work hard and genuinely have fun working with each other to achieve great things together.

What You'll Do:

As a Backend Developer on the Engineering team, you'll be responsible for the implementation, design, architecture and quality control of business integrations and general development within our customer-facing product application, as well as internal tools used by development and technical operations. This includes troubleshooting bugs and implementing new features in existing backend applications; and the innovative opportunity build new components with a highly collaborative team in a learning environment.

A successful Backend Developer will produce all required functionality by communicating and working with colleagues to solve problems and improve the systems they work with. Furthermore, you will gain a deep understanding and knowledge level of the systems you are working with. This is an exciting opportunity to innovate in a modern technical stack with a team that values continuous growth.

What You'll Need:

  • 2+ years of full-time experience as a Java developer on Linux platform with familiarity on latest Java development framework and open-source tools
  • 3+ years of software development experience in commercial or enterprise application
  • Strong understanding of web application architectures with strong SQL / Database knowledge
  • Familiar with AWS services, other cloud platforms
  • Ability to work with a development team and develop strong, reliable, and effective relationships with team members
