Nova Biomedical is looking for a Software Engineer (entry-level) to join our R&D Software Development team in Waltham, MA. This is a terrific opportunity for a recent college graduate to join an established medical device company – to learn, contribute and grow. This role will grow to develop software embedded in our innovative, world-class medical and biotechnology instrumentation.

Responsibilities:

  • Code complex software modules under the supervision of senior level colleagues
  • Test and debug software modules
  • Participate in design and code reviews of the various modules and functions.
  • implement verification scripts and validation test protocols.

Qualifications:

  • Bachelor’s Degree in Computer Science, Software Engineering or Electrical Engineering
  • 0- 2 years’ experience
  • Object oriented programing background
  • Desire and aptitude to become fluent in C, C++, and C# in Microsoft environment
  • Ability to work in an entry-level role, within a multi-discipline group, and grow with our company.
