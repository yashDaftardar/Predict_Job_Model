GENERAL SUMMARY/ OVERVIEW STATEMENT:


The Lab for Computational Neuroimaging (LCN) develops software for the analysis of structural, functional, and diffusion MRI and PET of the human brain. LCN is best known for FreeSurfer (surfer.nmr.mgh.harvard.edu), a brain anatomical analysis package that is used by thousands of researchers around the world. We are seeking a highly motivated software engineer to be responsible for maintaining and testing existing software as well as to help implement new algorithms. LCN is a dynamic and diverse lab composed of members from a wide range of backgrounds (engineering, neuroscience, computer science, math, physics, psychology & biology).; the software engineer will work closely with researchers and other software engineers to build, improve, test, support, and document the software. The software engineer will report to the Director of LCN, the Principal Investigator of the FreeSurfer Maintenance grant, and the Research Lab Manager.

Development responsibilities:

Fully competent in C/C++ and Python programming. Applicants will be required to present C++ and Python code samples demonstrating their competency.

Implement deep learning python utilities using TensorFlow/Keras


Test responsibilities:
Develop and execute software test plans (eg, unit and regression tests) on multiple platforms
Monitor automated tests, analyze test results, and recommend corrective action


Maintenance and support responsibilities:

Refactor, optimize, and clean existing algorithms in Python and C++

Port C++ and matlab to Python

Maintain GitHub repository

Fix bugs and enhance features

Resolve multi-platform compatibility, installation, and run-time issues
Monitor the FreeSurfer mailing list and respond to end-user issues

Develop and integrate a FreeSurfer python framework with C++ code

Package new and existing tools for public distribution


Teaching responsibilities:

FreeSurfer classes locally and (optionally) nationally and internationally

One-on-one instruction with faculty and staff


Documentation responsibilities:

Maintain and upgrade the Freesurfer mailing list and website
Contribute to the FreeSurfer documentation website.

Document software (yours and others).

Data analysis responsibilities:

The software engineer will perform some data analysis to test tools and to make sure that he/she is aware of how the tools are used and to experience them from a user point of view.

The Martinos Center for Biomedical Imaging at the Massachusetts General Hospital (MGH), affiliated with both Harvard Medical School and MIT, is one of the premier brain imaging centers in the world. The software engineer will work in a general office environment alongside MGH/Harvard/MIT faculty, postdoctoral fellows, research technicians, and students. LCN has about 20 faculty from a wide range of backgrounds (engineering, neuroscience, computer science, math, physics, psychology & biology). LCN consists of about 20 faculty and post docs almost all of whom write software. There are about 8 research assistants. The software team consists of several faculty members and about 4 dedicated software engineers. The individual who fills this position will be given standard maintenance, testing, and support tasks and will work with faculty members to implement new algorithms as well as collaborators who are using the software for research.
