Summary

The Software Engineer creates desired functionality to assigned domain, products or services in application development for singular features or applications. Demonstrates knowledge of solutions within assigned domain, product or service. Responsible for creating desired functionality to assigned content, products or services in ecommerce. Develops, tests and implements software that provides robust technical infrastructure and/or software applications used by business units.

Responsibilities

  • Develops applications and the underlying framework.
  • Collaborates to evaluate application designs.
  • Implements, tests and delivers new features for multiple platforms.
  • Coordinates with QA team to initiate testing and ensure testing is completed.
  • May design framework and software standards and make suggestions for systems / software improvements.
  • Assess project requirements and provides design input.
  • Executes upgrades, migrations, optimizations, new implementations and identifies and resolves efficiency issues.
  • Writes scripts, templates, interfaces and utility applications to continuously improve functionality.
  • Executes design and deploy activities for operating environments.
  • Develops, tests, troubleshoots and documents new and enhanced systems and processes.
  • Reviews and coordinates test plans for new and existing operating environments.
  • Provides level 3 support for area of expertise.

All other Duties as assigned.
