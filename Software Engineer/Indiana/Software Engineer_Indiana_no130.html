Sweetwater seeking to add a new DevOps Engineer to our growing Platform Engineering team. As a member of this team, you will help enable Sweetwater’s engineering groups to quickly deliver value to their customers in a secure, available, and supported environment. A good candidate will have experience developing CI/CD pipelines, the ability to manage resources in a hybrid cloud environment, strong monitoring skills, and a passion for automation and innovation.

Required Skills
  • A background in devops, platform engineering, site reliability engineering, systems administration, or software development
  • Proficiency with git, including branching and merging strategies
  • Comfortable working with Infrastructure as Code tools like Terraform, CloudFormation, or Google Cloud Deployment Manager
  • Configuration management skills such as Ansible, Chef, or Puppet
  • Experience developing CI/CD pipelines using tools like Gitlab, Spinnaker, Cloud Build, or Harness
  • Self-motivated with a strong sense of ownership, urgency, and drive
  • Empathetic listening skills and a desire to continuously improve and grow
Preferred Skills
  • Experience managing resources in a hybrid environment spanning traditional on-premises data centers and public clouds
  • Working knowledge of containers and container orchestration platforms like Kubernetes
  • Ability to configure and maintain monitoring tools like NewRelic, Prometheus, or Stackdriver
  • Experience with web infrastructure and distributed systems
  • Scripting proficiency with Python or bash
Desired Skills
  • Solutions architecture experience with a focus on cloud native solutions
  • Coding and service development skills with experience in a major language such as Node.js, Golang, or Python
  • Google Cloud Platform Certifications
