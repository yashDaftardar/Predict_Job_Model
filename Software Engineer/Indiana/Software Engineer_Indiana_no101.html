Full Job Description

The Software Engineer will be working with other engineers and architects to develop highly performant, API and web-based software on our enterprise SaaS philanthropic solutions software platform. This person must be self-motivated and results-oriented with strong programming skills across modern enterprise software architectures and be skilled in working in agile software development with a focus on highly scalable, automated solutioning.

Duties and Responsibilities:

  • Write production-quality, highly performant, API services using C#/.NET Core for service-based enterprise SaaS solutions in a collaborative, agile environment
  • Write production-quality, highly performant, frontend code using ASP.NET and Aurelia in a collaborative, agile environment
  • Modification of database objects using SQL and PL/SQL (stored procedures, views, tables etc.)
  • Write Automated Unit, Integration, and UI-level Tests to increase code quality and lower defect rate.
  • Refactor existing code to improve maintainability and quality
  • Work closely with Product Management and other areas of the business to ensure market needs are met.
  • Propose and implement technical decisions in accordance with overall architecture goals and best practices
  • Adhere to defined security standards in both functional and non-functional duties of job
  • Maintain devops pipeline using Docker, Kubernetes, TeamCity

  • Bachelor's Degree in Computer Science or related field required
  • 1-3 years of full stack programming experience
  • Experience programming with .NET C# and Entity Framework
  • Experience in Database systems such as SQL Server and Oracle
  • Experience building scalable REST APIs and Microservices
  • Experience with JavaScript frameworks such as Aurelia, AngularJS, Vue is preferred
  • Experience using container services (Docker and/or Kubernetes)
  • Experience using Azure and/or AWS infrastructure services
  • Agile methodologies
  • Experience using version controls systems such as Git
  • Strength in algorithms and design patterns
  • Reporting, analytics, data warehouse experience is preferred
  • Experience with automated testing frameworks and test-driven development is a plus
  • Investment accounting industry experience is a plus
