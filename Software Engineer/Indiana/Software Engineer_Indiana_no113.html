Job Summary:

Manages and oversees the design, development and testing of software applications. Provides leadership to a team of software engineers in a fast-paced environment toward quality production rollouts.

What you will be doing:

  • Coordinate the software development process using best practices
  • Ensure the team meets overall development goals in accordance with long-term product development plan and short-term project deliverables
  • Build healthy team relationships
  • Take initiative to improve product development processes
  • Leads team with hands-on participation like code-reviews

What You Need For This Position:

  • Bachelor’s Degree or better in related field
  • 4+ years of team lead experience
  • Good analytical problem solving skills
  • Good written and verbal communication
  • 8+ years of professional software experience with strong skills
  • Dependability, adaptability, flexibility and creativity
  • Ability to understand complex issues
  • Perl, C# or C++ or Java (required)
  • Unix/Linux command line environment experience
  • SQL/relational database skills
  • Git command line plus Gitlab or GitHub (preferred)
  • Experience developing, running, and supporting microservice-based architecture in a production environment
  • Experience developing web services (SOAP/REST)
  • Experience with API Management platform (ex: Mulesoft), API architecture (desired)
  • Experience mentoring other developers
  • Experience with Agile methodologies
