Full Job Description

The IT Azure Cloud Engineer will assist with the management, support, and oversight of the virtual server environments and business continuity processes across HFHI Cloud Computing environments to include Microsoft Azure and other Cloud Hosting environments. This position will assist with managing, maintaining, monitoring and securing Azure servers and networks and services including installations, upgrades, patches and documentation.
The IT Azure Cloud Engineer will support applications teams in securing and monitoring their environments, through development support of alerting, reporting and dashboards. The IT Azure Cloud Engineer will be accountable for meeting deliverable commitments including schedule and quality compliance. They will plan and schedule their own work activities and coordinate activities with other cross-functional team members to meet project goals.

This position will be remote, preferably in Atlanta, GA.

Essential Duties and Responsibilities:
  • Monitor the Azure/Cloud environments for performance, security and improvement opportunities (10%)
  • Assist in the management and implementation of Azure windows patching per HFHI guidelines and requirements (5%)
  • Assisting with the rapid delivery of Azure services as requested by internal customers, internal projects and automated service tickets (20%)
  • Engage on advanced Azure projects including system architectural conversations (10%)
  • Work directly with the application support teams, internal customers, 3rd party vendors and Microsoft for timely resolution of support requests (20%)

  • Support of other Microsoft cloud services e.g. Active Directory, Config Manager, SQL, Sentinel, Microsoft 365 (20%)
  • Other related duties as assigned by supervisor
