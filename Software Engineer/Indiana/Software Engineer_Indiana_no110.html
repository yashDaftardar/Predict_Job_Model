We are looking for an outstanding Web Developer to be responsible for the coding, innovative design and layout of our website. This job reports to the Marketing function of the organization. You will help build our website - from concept all the way to completion, fashioning everything from the home page, to site layout, and function. You will also design, build and program email templates, landing pages, ads and more to support various marketing campaigns and initiatives.

You should be able to write clean code and ensure your programs run properly. We also expect you to be passionate about building software and perform well working in a team, along with marketers, developers, and web designers.

Responsibilities

  • Be a self-starter - take the initiative to make suggestions, give ideas, and contribute to the team before you're asked.
  • Produce well-designed, testable, efficient code by using best software development practices.
  • Build and maintain a website layout/user interface by using standard HTML/CSS practices.
  • Utilize data from various back-end services and databases.
  • Collect and review specifications and requirements based on technical needs.
  • Continual renewal of design and overall content presentation.
  • Maintain, expand, and scale our site as assigned.
  • Keep up-to-date with emerging technologies and industry trends and apply them into operations and activities.
  • Work together with web designers to match visual design intent.
  • Create graphics and illustrations for articles, editorials, blogs, message boards, etc.
  • All other duties assigned.

Qualifications

  • 4+ years' experience of working in a Web Developer or Web Programmer position.
  • Excellent knowledge of HubSpot.
  • Javascript experience.
  • Experience with Salesforce is a plus.
  • Good experience with graphic design and web application development.
  • Excellent knowledge of how web applications work including security, session management, and best development practices.
  • Excellent knowledge of modern HTML/CSS.
  • Experience with network diagnostics and network analytics tools.
  • Knowledge of Search Engine Optimization process.
  • Strong problem-solving skills with a creative approach and openness to wearing multiple hats.
  • Strong organizational skills with the ability to prioritize tasks.
  • Excited to work and thrive in a fast-paced environment, learn rapidly and master diverse web and design technologies and techniques.
