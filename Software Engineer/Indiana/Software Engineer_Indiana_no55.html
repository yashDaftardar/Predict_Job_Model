Software Build Engineer:

This individual will be an integral part of a growing team of 15-20 software engineers developing 4D-based solutions for internal use at Sweetwater Sound. Will oversee the architecture and maintenance of a robust, reliable, automated and efficient build process. Individual will propose and finalize work and data flow enhancements in the production pipeline that impact version processing.

Core Responsibilities:
  • Develop and maintain a continuous delivery pipeline for Sweetwater's internal 4D projects.
  • Ensure the robustness, reliability, efficiency, reusability, versatility and parameters of the build system
  • Carry out technological monitoring as it pertains to quality of deployments
  • Implement metrics and interpret these metrics to enhance the quality of the systems
  • Support the DevOps team with day-to-day operations, including but not limited to the management and maintenance of the various development environments.
  • Automating tasks using scripting languages - Python and/or PowerShell preferable.
  • Be flexible and prepared to work closely with development teams, clearly communicating the status of tasks.
  • Make recommendations to the programming team on build system architecture
  • Be a technical coach for the build and release specialists on the team (code review and guidelines)
  • In collaboration with other working units, recommend and increase the efficiency of version creation systems and related pipelines

Minimum Qualifications:
  • Bachelors Degree in computer science.
  • Experience with CI / CD, Build systems, and Source Code Control tools such as Git/SVN, build tools like Ant, Maven.
  • Strong scripting knowledge using PowerShell or Python.
  • Familiarity with Windows-based software development environment.
  • Must have excellent oral and written communication skills.
  • Strong interpersonal skills and proven record of performance and ability to work independently as well as in a team.
Good To Have:
  • Experience with 4D.
  • Familiarity with administration of Build Orchestration tools like Jenkins.
