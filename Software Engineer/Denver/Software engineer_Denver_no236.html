The Opportunity

We're seeking an entry-level software engineer to help us build, maintain, grow, and secure our healthcare platform, furthering our reputation as a technical pioneer, during this time of exponential growth.

You'll fit right in if:

  • You want to change people's lives and help others
  • You are self-directed but play well with others
  • You're honest and ethical
  • You continually strive to learn new things

Responsibilities

  • Creating beautiful new user experiences in React/Vue
  • Analyzing and fixing production issues
  • Consuming and collaborating on easy-to-use internal and partner-facing RESTful and GRPC endpoints
  • Working with QA to enhance product quality through automation
  • Working on both greenfield development as well as new feature enhancements
  • Participating in code quality practices such as peer reviews and documentation

Candidate Profile

  • Exceptional problem-solving skills that will be required to debug and provide solutions from HTML and CSS all the way down the stack to infrastructure level (ex: creating mechanisms for clearing and repopulating certain production server caches) if needed.
  • Front-end development experience including HTML5, CSS, JavaScript and modern JavaScript frameworks such as React/Vue
  • An understanding of REST and/or other protocol-based communication such as gRPC
  • Exposure to data structures, design patterns and architecture principles
  • Demonstrated knowledge of MySQL or another RDBMS including querying and more
  • Hands-on experience with browser debugging tools
  • Enthusiastic about code optimization, automated testing, and writing unit tests
