Title: Software Engineer Intern, E-Tools

Location: Westminster, CO

Department: Trimble Cloud, E-Tools

About Trimble E-Tools

Trimble E-Tools is the team responsible for deploying and operating cloud-based dev-ops toolset, that is used by Trimble employees. 8,000+ Trimble coworkers—designers, engineers, content creators, and product managers who depend on our platform to power their work from 40+ countries around the world. E-Tools, a suite of engineering and DevOps applications for collaboration, planning, issue tracking, source code management, continuous integration, security scanning, and product delivery; built around best-of-breed, self-hosted, third-party applications, run securely in the AWS public cloud: Jira, Bitbucket, Confluence, Bamboo, Artifactory, SonarQube, and all the glue between them.

Job Summary

We seek a software intern to work with the E-Tools team with a focus on Data Analysis. A variety of information is collected by the various tools and subsystems of E-Tools. The task is to then use this data to create interfaces and analysis tools that can help with the automation and optimization of the E-Tools resources and processes.

Measures of Success

  • Building visualization tools that help engineers, testers, and customers understand how well complex systems are working, and to pinpoint where things may be going wrong.

  • Prototyping and deploying software to control all aspects of the system and data collection

  • Work independently to deliver high-quality code.

  • Collaborate with the engineering team for brainstorming, prototyping, documentation, code reviews, and final integration testing.

Minimum Qualifications

  • Currently pursuing a Bachelor’s, Master’s, or Ph.D. in Engineering, Computer Science, or related field

  • Programming experience with Python.

  • A passion and curiosity for building things, inspecting data.

  • Usage of Data Visualization tools and techniques.

  • Understanding of APIs, REST is a plus.

  • Knowledge of Amazon Web Services will be helpful.
