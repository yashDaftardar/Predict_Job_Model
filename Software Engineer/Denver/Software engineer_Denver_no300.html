Join Alteryx, the leader in data blending and advanced analytics, and be part of a dynamic and fast-growing organization in one of the hottest software markets. Alteryx provides data analysts in departments like sales, marketing, finance, and real estate with an intuitive visual workflow designer that allows users to blend data from multiple sources, perform data analytics, generate reports, and create dynamic reusable applications that can be deployed to a cloud-based server solution.

The Senior Software Engineer, .NET will join a team of highly passionate software developers responsible for the user interface of our desktop product, which is built using C#/.NET, JavaScript/HTML and C++.

Position Details:

  • Participate in the architecture and design of features associated with a professional-grade desktop application.

  • Develop new product functionality primarily using C#/.NET.

  • Work with product management, UX, quality assurance, development team members and other engineering development teams to coordinate product development activities.

  • Support, maintain and enhance legacy code, including migration to new technologies.

  • Participate in the recruiting, interviewing, and hiring of technical talent.

Required Skills:

  • 4+ years of GUI development using Windows Forms, WPF, HTML5 or equivalent GUI development technology.

  • 4+ years of experience developing software using C#/.NET.

  • Demonstrated professional expertise in understanding and implementing OO design patterns.

  • Demonstrated ability to translate business requirements into technical solutions.

  • Demonstrated experience writing unit, component and functional tests.

  • Excellent written and verbal communication and listening skills.

  • Ability to learn quickly and work independently with minimal direction and guidance.

Valued Skills:

  • Experience developing software using JavaScript/HTML and/or C++.

  • Experience with third party component libraries, such as Telerik, MindFusion, SyncFusion and Infragistics.

  • Experience using Git, Subversion, Perforce, or equivalent SCM.

  • Demonstrated excellence in a fast-paced Agile environment.

Compensation:

  • Alteryx is committed to fair and equitable compensation practices. The salary range for this role in Broomfield, CO is $101,700 - $172,900 . This position is also remote-friendly and, as such, compensation will ultimately be in line with the location in which the position is filled. Final compensation for this role will be determined by various factors such as a candidate’s relevant work experience, skills, certifications, and geographic location. This role is eligible for variable compensation including bonus and stock grants.
