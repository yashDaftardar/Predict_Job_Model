Job Duties:


  • Conducts software integration and testing in an Agile environment using various test methodologies on sprint releases for hardware and software projects on the Human Space Flight/Deep Space Exploration project
  • Troubleshoots and resolves anomalies using JIRA for defect tracking and logging in a closed loop system
  • Writes and converts test scripts from CCL and Python to a working automated testing environment to include regression testing
  • Tests fault management, on-board command and data handling, embedded file systems, uplink/downlink encryption, C3I standards, on-board networking, and system performance analysis

  • 5 years of HW/SW experience with demonstrated experience in Space Craft systems
  • Must have working knowledge and experience in closed loop systems, and experience in the implementation of formal test processes
  • Must have experience in large-scale hardware/software vehicle-level integration, knowledge of different test methodologies, and regression testing experience
  • Experience with fault management systems, on-board command and data handling, embedded file systems, uplink/downlink encryption and decryption, C3I standards, on-board networking, and system performance analysis
  • Familiarity with the mission flight objectives and experience in human spaceflight
  • Experience with Avionics HW & SW Architecture
  • Must have working knowledge and experience in closed loop systems, and experience in the implementation of formal test processes
