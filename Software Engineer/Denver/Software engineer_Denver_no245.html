We are looking for a Full Stack Web Developer responsible for managing the interchange of data between the Web user interfaces and servers. Your primary focus will be the development of UI logic, Web server logic, server-side logic, tool logic, and automated test cases ensuring high performance and responsiveness to full cycle Web requests.

A Day in the Life

The Senior Full Stack Web Developer is responsible for providing computer science skills for use in the development of cutting-edge products used in the medical field. Specific tasks include:


  • Designing, developing, testing and documenting software components, unit tests, automated tests, and applications.
  • Writing reusable, testable, and efficient code.
  • Interfacing with mechanical, electrical, systems, and test engineers.
  • Troubleshooting, debugging, maintaining, and supporting existing software products.
  • Following Medtronic Navigation product development standards and practices.
  • All activities must be performed in compliance with the Quality System.
Must Have: Minimum Requirements
  • Bachelors degree in Computer Sceince or similar discipline required
  • Minimum of 4 years of relevant experience, or advanced degree with a minimum of 2 years relevant experience

Nice to Have
  • Extensive understanding of front-end technologies, such as React, Redux, JavaScript, HTML5, CSS3, RESTful, HTTPS, WebSockets.
  • Familiarity with promise driven programming in JavaScript.
  • Experience with JavaScript unit test frameworks, such as Jest, Mocha, Enzyme
  • Experience with modern integration test frameworks, such as Cypress.io, Selenium, Spectron.
  • Experience in Python 3
  • Proficient understanding of the Git versioning tools
  • Understanding of fundamental design principles behind scalable applications
  • Knowledge of user authentication and authorization between multiple systems, servers, and environments
  • Strong unit test and debugging skills
  • Familiarity with OOP, ORM, patterns, structures, and other programming techniques and frameworks.
