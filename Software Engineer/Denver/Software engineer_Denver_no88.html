Full Job Description

As the market leader in self-service solutions, KIOSK provides proven expertise in design engineering, application development, integration, manufacturing, field support, and managed services for even the most sophisticated self-service platforms. A deep portfolio of standard and custom KIOSK designs is deployed among Top 100 Retailers and Fortune 500 clients in a wide array of self-service vertical markets. To learn more, visit kiosk.com.

The Software Engineer is responsible for analyzing, planning, developing, implementing, testing and maintaining various hardware and cloud based IoT projects.

Responsibilities:

  • Perform full stack software development
  • Participate in pull requests and code reviews
  • Ability to work with off shore development team
  • Design and update software as needed
  • Test and maintain software products to ensure strong functionality and optimization
  • Work on production and commercial deployment of systems for enterprise customers
  • Maintain and improve the performance of software throughout the entire complete software development life cycle
  • Recommend improvements to existing software programs as necessary
  • Provide timely solutions to production issues
  • Ability to accurately estimate effort
  • Develop windows hardware abstraction layer to interface with vendor driver/firmware for components such as card reader, bar code scanner, printers etc.

Required Skills:

  • 1-2 years of experience developing C# .NET applications
  • .NET core, .NET Standard, .NET framework
  • Typescript, modern JavaScript framework such as Angular, React.js etc.
  • Test Driven Development
  • Git, Gitflow, Azure DevOps
  • AWS cloud services (preferred)
  • Experience consuming, designing and developing microservices APIs (highly preferred)

Desired Skills:

  • Adaptability – Being open to and actively solicit new ideas and opinions.
  • Experience consuming, designing and developing microservices APIs (highly preferred)

Travel:

  • Less than 5% travel

Physical:

  • Must be able to sit and stand 8 hours
  • Must be able to lift 10 pounds

Starting Salary - $74,500.00- Final compensation for this role will be determined by various factors such as a candidate’s relevant work experience, skills, and certifications.
