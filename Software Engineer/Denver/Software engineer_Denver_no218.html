Position Description:

RAMTeCH Software Solutions is a global GIS consulting, data management, and software solutions company specializing in
the utility, energy and telecommunications industries. We are seeking a Software Engineer to work in our Minneapolis/St.
Paul Minnesota area office or remotely within the United States. This position involves application and software design,
development, and implementation of GIS based solutions.

  • Work in a team environment with project managers, consultants, architects, developers, and clients to provide

analysis, design, programming/configuration support innovative technology solutions

  • Knowledge of system architecture including the component interactions and interfaces, and the technologies

and resources to be used in the design

  • Experience with systems analysis, design, testing, development, and implementation of application software
  • Experience with application development estimating planning, testing, and progress tracking
  • Knowledge of Relational Database Management software, Oracle or SQL Server RDBMS
  • Experience with Web services, application servers, Web servers, and/or SQL performance tuning SQL

performance tuning

  • Experience with C#, knowledge of Python and JavaScript
  • Knowledge of concepts, practices and procedures (Control Objectives for Information and Related Technology, and IT Infrastructure Library®), including change management, software/systems development life cycle for application development, IT Service Management (ITSM), and IT governance
  • Knowledge of mobile application development using Xamarin Forms
  • Experience using MVVM architecture, familiarity with MVVM platform such as Prism
  • Experience with Esri ArcGIS Desktop, Mobile, Web Products and SDK’s, Esri Utility Network
  • Experience with AWS Cloud implementations
  • Knowledge of electric, gas, telecommunications or water utility business and operations
  • Knowledge of software engineering concepts, software architecture, user interface design, change management,

software/systems development life cycle for application development, IT Service Management (ITSM), and IT
governance

  • Strong written and verbal communication skills and excellent listening skills for individual interactions and in group settings

Experience and Education Required:

  • Minimum of 5 years’ experience design and developing applications and integrations for large client user bases.
  • Education: Bachelor’s or master’s in computer science, GIS, cartography/geography, engineering, or related field.

Travel: Moderate travel within the U.S.
