Software Developer

Grey Matters Defense Solutions
Denver, CO

Grey Matters Defense Solutions is looking for six software developers to help us pull together an integrated software solution for the Department of Defense. We are looking for a candidate who is committed to working as part of a highly collaborative team to produce a seamless decision-making tool designed to integrate several select programs of record. The candidate must have the ability to maintain a TOP SECRET, SCI clearance and will be working within a secure facility near Buckley Air Force Base, in Aurora, CO.

Qualified candidates must have:

  • Bachelor’s degree in computer science, IT, or equivalent technical discipline, and/or ten to fifteen year’s work experience in relevant focus areas.
  • Demonstrated experience building and managing the code and processes to establish common code repositories, standards for re-use and efficiencies across team(s) and projects.
  • Subject matter expertise in software engineering using current web, application and database technologies. We are interested in a variety of programming languages, but at a minimum, the candidate should have expertise in:
    • XML, C++ and/or Java, in Unix and Linux development environments
  • Demonstrated experience engineering scalable multi-threaded web-based data applications to meet high-availability requirements.
  • US Citizenship and an active DoD TS/SCI security clearance.
  • Candidates who do not have an active clearance will not be considered for this position.

Preferred Qualifications:

  • Demonstrated experience delivering software solutions using Cloud technologies
  • Prior experience delivering software solutions that adhere to IC clients data handling, compliance and security requirements
  • Experience integrating Cloud technologies in the IC environment.
  • Experience with data analysis; target needs assessment; systems engineering and integration; data acquisition/distribution/ management/enhancement; data prioritization; SIGINT and/or GEOINT Tasking, Collection, Processing, Exploitation, and Dissemination
  • Demonstrated experience delivering technical solutions using COTS and GOTS relational and non-relational database technologies and big-data solutions.
