Job Summary
We are looking for an outstanding self-motivated Front-End Developer to be responsible for building, testing, deploying and maintaining guest-facing websites and administrative tools for the Digital team at Vail Resorts. The ideal candidate will have excellent communication skills with the ability to work in a fast-paced, team-oriented environment. Reporting to the Front-End Development Manager, this individual will leverage modern coding techniques using JavaScript, CSS, and HTML to deliver efficient and elegant experiences to our guests.


Job Responsibilities

  • Maintain and enhance existing web applications
  • Proactively contribute to the team with an Agile mindset
  • Collaborate with business analysts, graphic designers and development teams to build fully-functional, pixel-perfect components and comprehensive layouts
  • Work closely with developers, business analysts, managers and business partners to understand and deliver requirements across all modern browsers for both desktop and mobile systems
  • Leverage internal Content Management Systems as needed to deliver dynamic content
  • Troubleshoot issues and prototype technical solutions for all supported browsers
  • Perform code reviews
  • Maintain and create documentation around code and process as needed
  • Proactively find opportunities to improve development standards,
  • performance and functionality within Vail Resorts technology ecosystem


Job Requirements

  • 2+ Years developing and maintaining customer-facing websites in a
  • collaborative team environment
  • Ability to accurately replicate design comps to code
  • Superb understanding of responsive design techniques
  • Strong JavaScript, CSS (Sass/LESS), HTML skillset
  • JavaScript Frameworks and Libraries (React, Underscore, jQuery, etc.)
  • Planning and delivering solutions across multiple browsers and platforms
  • Accessibility Standards and Compliance
  • Performance Improvements and Code Optimization
  • Ability to work with Web APIs
  • Git Source Control
  • Excellent communication skills


Preferred

  • Solid understanding of the web application lifecycle
  • Experience working with Design Systems
  • Component Driven Development, Storybook
  • Working knowledge of Microsoft IIS and server technologies
  • OOP using Visual Studio, ASP.NET, C#, and MVC.NET
  • Enterprise CMS (Sitecore)
