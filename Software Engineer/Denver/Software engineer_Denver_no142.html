Software Engineer - 3D Warehouse Frontend

This Opportunity

If you’ve ever used SketchUp, you know it’s special software: 3D modeling that’s easy to learn, intuitive to use, and accessible to everyone. Try it out for free at app.sketchup.com and browse millions of free models at 3dwarehouse.sketchup.com .

As a member of the engineering team focused on SketchUp 3D Warehouse, you’ll be working to both refine and re-imagine how we deliver millions of 3D models to our customers all over the world. We’re looking for someone who is highly skilled, highly motivated and eager to learn more - and pretty low-key about it all. You should already have some experience writing scalable web applications and understand what it takes to scale a web application up to tens of millions of users and work with large datasets, and you’re excited to roll up your sleeves for the next big challenge.

About SketchUp, a Colorado startup all grown up

Part of Trimble, SketchUp is one of the most popular 3D modeling and design tools in the world. Today, millions of architects, engineers, contractors, woodworkers, artists, and other creatives use our software to design transit centers, treehouses, and everything in between. We support a healthy work/life balance and high-quality standards for software and coffee.

About Trimble, our parent company

Trimble is transforming the way the world works by delivering products and services that connect the physical and digital worlds. Our core technologies in spatial positioning, modeling, connectivity and data analytics enable customers to improve productivity, quality, safety, and sustainability in design, architecture, and construction.

Responsibilities:

  • Develop client-side Web application code for Trimble’s SketchUp 3D Warehouse.

  • Analyze, design and implement new features and improve performance of the web applications.

  • Collaborate with UX/UI designers and product managers to find optimal solutions.

Required Skills/Experience:

  • BS/BA degree in Computer Science, Software Engineering or equivalent experience.

  • 3+ years’ experience developing Web applications.

  • Experience using various web technologies including TypeScript, Vue.js, Sass, WebGL, Webpack, and Node.js

  • Knowledge of Web standards and protocols including JavaScript, HTML, CSS, HTTP.

  • Experience working with scripting languages like Python or Ruby.

  • Familiarity with common commercial software development best practices.

  • Strong problem-solving skills and eager to learn new technologies.

  • A test-driven development mentality.

Bonus Skills/Experience:

  • Experience with Amazon Web Services, Google Cloud Platform or Microsoft Azure.

  • Experience using Java, C# or C++ and building server-side applications.

  • Experience using modern Web application frameworks like Angular or React.
