About OptalX

OptalX cuts the guesswork from media mix planning by helping advertisers set an objective behind every dollar with ease. The platform compares an advertiser’s objectives to the benchmarks from their brand and industry to form a comprehensive plan and avoid waste.

Job Overview:

The Back-end Engineer will be responsible for building endpoints that serve data for our user-facing web application. The focus will be on integrating our various databases to ingest external data via API connectors.

Responsibilities:

  • Writing, maintaining, and documenting REST endpoints using AWS Lambdas
  • Designing scalable data models in PostgreSQL
  • Integrating our 3rd-party vendor to ingest external data via API connectors

Skills You Must Have:

  • Python
  • PostgreSQL
  • AWS Lambdas
  • Strong written and verbal communication skills
  • Ability to appropriately tradeoff elegant design, simplicity, and time to market
