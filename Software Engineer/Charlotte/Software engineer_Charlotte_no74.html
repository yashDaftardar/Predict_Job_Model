Job Description:

This is a EXCITING opportunity for a Full Stack Developer to join the Enterprise Finance Technology group. The person selected for this role will be experienced and skilled in building and supporting highly dynamic, responsive web applications (Java, HTML, JavaScript, CSS, AJAX, Angular, Breeze.js, C#, Python, Entity Framework, jQuery) as well as be well versed in large-scale enterprise SDLC processes. The role will be a key contributor in the design and build for a web application supporting the new metadata management solution for Corporate Treasury. The scope of the web design and development will include, but is not limited to creation of the web user interface, data input/update into heterogeneous data targets, database loading, access control and support to meet the needs of the business. Individual should have experience with various databases platforms, especially Oracle Exadata.


ESSENTIAL FUNCTIONS:

  • Demonstrates expertise in a variety of the field's concepts, practices, and procedures related to web application design and development

  • Builds software applications following coding standards and practices, builds appropriate unit tests, integration tests and deployment scripts

  • Assists in defining software architectures by collaborating with the team to explore existing systems, determines areas of complexity and potential risks to successful implementation

  • Must have intellectual curiosity to come in and obtain complete understanding of the existing solution

  • Maintain adequate communication with project team, explaining progress and/or potential roadblocks on the development effort

  • Owns success and takes responsibility for successful delivery of the solution

  • Contributes to continual improvement by suggesting improvements to user interface, software architecture or new technologies

  • Able to build appropriate interface to back-end databases, including SQL Server

  • Implement web solutions using Java, HTML, JavaScript, CSS, AJAX, Angular, Breeze.js, C#, Entity Framework, jQuery in a Windows Server 2012 environment

  • Work with business teams, technical analysts, DBAs and system administrators to design web interfaces/application

  • Partner with infrastructure partners to build/maintain the web application infrastructure

  • Perform training and consultation to business users and tech teams on usage of the target solution

  • Provide complete documentation for the web application including high-level designs, low-level designs and user training documentation


MINIMUM QUALIFICATIONS:

  • Experience writing and maintaining RESTful services written in Java

  • 5-7 years’ experience in C#, Java

  • 5 years’ experience in Python

  • Experience writing and debugging Windows Workflow

  • 5-7 years experience taking web applications from concept through implementation and support

  • 5-7 years experience setting up web application environments

  • 3-5 years experience with Microsoft Visual Studio

  • 3-4 years’ experience in Windows Server (2012)

  • 3-4 years’ experience in databases – SQL Server preferred

  • 3-4 years experience using version control software such as Subversion SVN

  • Highly skilled at front-end engineering using Object-Oriented JavaScript, various JavaScript libraries and micro frameworks (JQuery, Angular, Breeze, Dojo, Backbone, YUI, HTML5, and CSS)

  • Well-versed in software engineering principles, frameworks and technologies

  • Self-directed team player who thrives in a continually changing environment

  • Strong customer service/client service skills


PREFERRED QUALIFICATIONS:

  • Prior experience in to a Enterprise Finance Technology group
