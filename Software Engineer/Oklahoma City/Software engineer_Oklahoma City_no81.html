Norman OR Toronto OR Washington State

Team responsibilities

As a Full stack / Back-end application developer, you will be responsible for maintaining and adding functionality to a microservice web architecture built with Java Spring for backend services and a dynamic React.js frontend. You must be familiar with Java Spring, React.js, and Typescript, with a strong focus on unit and functional testing. You’ll have an opportunity to help design and develop our mobile edition as a Progressive Web Application (PWA) with modern technologies and run in the cloud.

We are building a platform to allow our research and product management teams to create and maintain views into MSCI’s financial analytical capabilities. With this tool, business users can construct custom UIs to provide the capability for sales to quickly show what MSCI has to offer and validate our analytics. The same infrastructure is used to support our clients in building their own integrations into MSCI’s financial analytical capabilities through our API Gateway.

Your key responsibilities

  • Work with researchers and product manager to collect new requirements supporting client use cases.
  • Participate in backend system architecture design for new features of a dynamically rendered React application.
  • Maintain and update JSON schemas and documentation used to construct and validate UI descriptions.
  • Continue work on a pluggable architecture to support more complex analytical inputs and result parsing.
  • Write and maintain unit test coverage during development.
  • Participate in design discussions and code reviews during development.
  • Write functional test cases to cover newly implemented functionalities and integrate them into our fully automated continuous release pipeline.

Your skills and experience that will help you excel

  • 5+ years software back-end development experience
  • 3+ years Java OR C++ development experience
  • 2+ years JavaScript development experience with a strong understanding of the DOM
  • 1+ years React.js development experience
  • Experience with React unit testing
  • Proficiency with browser-based debugging and JavaScript performance testing
  • Excellent communication skills, both written and oral
  • Experience with Docker/Kubernetes is a plus
  • Experience with Azure DevOps is a plus
  • Experience with VS Code extension development is a plus
