Software Engineer – Manufacturing Automation Applications

The Software Engineer Manufacturing Automation will enable digital factory transformation and advanced manufacturing automation through modular, scalable, high availability Microsoft .NET applications, MES, PLC, HMI. Develop and continuously improve real-time automation solutions from manufacturing floor to enterprise level using; C#, .NET, VB.NET, ASP.NET, MVC, VB.NET, VB6, HTML5, SQL, SQL Server, SSRS, Tableau, Database software, Wonderware MES, PLC programming, RTU, HMI & SCADA, Servo Motion Control, Networking, operator graphics and control software. Work directly with manufacturing engineers to create next generation manufacturing automation solutions. Transformation of manufacturing operations through engineering advanced .NET, web, data, and cloud solutions integrated with manufacturing execution Systems, advanced mechatronics, machine and process controls. Enable rich data capture for component traceability, system level unit history and multi-plant visibility of production data metrics and analytics for aggregate, single plant, or single production line within a plant. Engineer comprehensive dashboards and analytics to help see, manage, and optimize manufacturing operations.

Responsibilities:

  • Write custom visual C# and VB.NET code, SQL stored procedures, PLC ladder logic, various HMI applications, and server maintenance & configuration support.
  • Develop .NET applications to interface, drive, monitor and report on manufacturing automation systems.
  • Develop C# .NET applications based on .NET framework 4.5 or higher.
  • Develop full stack .NET applications in ASP.NET, MVC, HTML5, OOJS, Angular, React, Vue, Node.js.
  • Develop and implement API, REST Web Services with WCF.
  • MS SQL Server Database design, create tables, stored procedures, functions, triggers.
  • Design and develop BI Dashboards and reporting solutions using SSRS, SSAS, and Tableau pulling data from multiple different data sources via SSIS on SQL Server Databases.
  • Engineer Tableau BI Dashboards, visualization, and SSRS reporting to replace legacy Access applications.
  • Advance next generation applications leveraging containerization and VM architectures – docker, Kubernetes, VMWare designing for performant, scalable, and load balanced solutions.
  • Modernize .NET / SQL applications, webservices, API’s leveraging Lean Manufacturing, Lean Metrics, Continuous improvement, Value Stream Mapping for manufacturing business process improvement.
  • Full Application Lifecycle from ideation, design, development, testing, implementation, and support.
  • Do equipment integration with OPC-UA, Serial Port and TCP/IP interfaces.
  • Develop MES/SCADA Industrial Manufacturing Automation software, systems, and devices (HMI’s, PLC’s, RTU’s), Wonderware ArchstrA, Inductive Automation / Ignition Core (Design, implementation, operation)
  • Design Ignition HMI Applications & NextGen PLC Ladder Logic
  • Design and write interaction interfaces between MES and other systems.
  • Install, configurate and deployment MES related applications/services in shop floor.
  • Provide support for MES / Reports / Applications, and Systems Integration related issues on shop floor.
  • Create and maintain program road maps for the definition and deployment of Manufacturing Systems.
  • Write technical document to convert business requirement / functional spec. into technical spec.
  • Develop applications and system enhancement based on business’s requirements.
  • Clearly comment your code base for simple ongoing application refactoring in the future.
  • Conduct Unit and integration testing of programs, as well as user training
  • Monitor the operational status of programs and perform continuous improvements
  • Clearly document training, trouble shooting and support material for programs
  • Completing documentation, including project engineering notes, proposal writing, case studies, and more
  • Commissioning new solutions and finding opportunities for improvements

Requirements:

  • Bachelor’s degree in Engineering or Computer Science preferred, but not required
  • Minimum 3-5 years Software Engineering (.NET Full Stack) continuous improvement to manufacturing automation software applications, SCADA systems, Graphical HMI applications, PLC business processes MES/SCADA Manufacturing Automation (PLC/HMI) enabling digital factory transformation.

.NET full Stack Development

  • 3+ years .NET Full Stack Software Application Development using C#, ASP.NET, MVC, HTML5, OOJS, SQL
  • Backend Development Experience with C#, VB.NET, API’s, REST Web Services on.NET 4.5 framework+
  • MS SQL Server Database programming, tables, stored procedures, functions, triggers.
  • BI Dashboards and reporting development using Tableau, SSRS, SSAS, SSIS, SQL Server, Access.
  • Experience with Multiple Operating Systems – Windows 2010-2000, Server, Linux, Unix – shell scripting.
  • Be acquainted with source code control tools (e.g. TFS/SVN/Git).

MES/SCADA – PLC/HMI/RTU Development

  • Expert knowledge in design, implementation, management and operation of MES/SCADA Industrial Manufacturing Automation software, systems, and devices (HMI, PLC, RTU), Wonderware ArchstrA, Inductive Automation / Ignition Core design and communication protocols commonly used in Industrial Automation.
  • PLC Programming; Rockwell Automation (Allen Bradley) or Siemen PLC Programing software.
  • PLC Programming: Allen Bradley PLC5, SLC 500, Siemens S5 & S7 & TI, Omron, Modicon, GE, etc.
  • PLC networking / equipment integration: OPC-UA, Serial Port and TCP/IP interfaces, Industrial Ethernet, Modbus+ TCP/IP, EtherCat, Ethernet/IP, or PROFINET.
  • HMI programming: PanelMate, PanelView, CTC Interact, XYCOM, Total Control QuickDesigner.
  • SCADA (Supervisory Control & Data Acquisition) system experience: Wonderware,
  • Servo Motion Control programming: AB GML, Delta, Delta Tau, Emerson, etc..
  • RSVIEW, FactoryLink, Intellution FIX, Simatic WinCC, GE Cimplicity, etc.
  • Understands ANSI / ISA 99 standards (Security for Industrial Automation and Control Systems)
  • MES/SCADA Certs are helpful – Wonderware ArchestrA (System Platform)-, Ignition Core, etc.

Communication

  • Experience of general program development approach, like Agile/Scrum approach.
  • Experience with Lean Manufacturing, Metrics, and continuous improvement, conducting Lean and Value Stream Mapping driven business process improvement and re-engineering.
  • High Level interpersonal and leadership skills; must be able to influence and coach across multiple levels within the company and locations; including establishing global partnerships.
  • Excellent skills in communications, facilitating change management across organizations, and maintaining strong working relationships with business and project leaders.
  • Experience in partnering with executives and business leaders on developing and driving global initiatives.
