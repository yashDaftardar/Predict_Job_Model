Job Description - Overview

We are in search of self-motivated and versatile developer with 5+ years of experience to work on our Software Architecture team. The Software Architecture team handles many of our core development systems including centralized configuration, logging, application and API security, as well as assisting other development teams in designing and building large scale applications. You must be a team player, have a passion for building great software, and the ability to work in an energetic and collaborative environment. This position is located in Oklahoma City, Oklahoma. Remote work not available.

We are currently seeking a .Net Developer.

Primary Functions/Job Description
Build and maintain multi-tiered systems utilizing C# on both .NET Framework and .NET Core
Build and maintain web applications built with ASP.NET MVC, ASP.NET Web API, and ASP.NET Core
Build and deploy reusable .NET libraries as NuGet packages for other development teams to use
Explore and develop skills in new or unfamiliar technologies and frameworks
Communicate and coordinate with management and other departments on project tasks
Ability to contribute in all phases of a project
Prepare all appropriate user and technical documentation for assigned development projects
Provide end-user production support for in-house developed and purchased software including after hours support scheduled on a rotating basis with other developers

Job Description - Requirements

Knowledge of C# or other similar programming languages
Experience in building applications with ASP.NET, ASP.NET Core, or similar frameworks
Experience building reusable libraries with package managers such as NuGet, npm or similar
Experience in building secured RESTful web APIs and n-tier distributed systems
Experience in developing and supporting large production applications
Solid understanding of HTTP and Web protocols
Knowledge of object-oriented and/or functional programming paradigms
Experience in a source control system, preferably Git
Familiar with relational database design
Ability to write and debug SQL queries and stored procedures
Preferred Skills
Experience in building cross platform applications for macOS, Linux, and Windows
Experience working with containerized platforms such as Docker
Experience with the Model-View-Controller architectural pattern (ASP.NET MVC, Spring or similar)
Knowledge of JavaScript or TypeScript and how to utilize them in browser based applications
Experience in building SPAs with Angular or a similar framework
Familiarity in securing applications with OAuth 2.0 and Open ID Connect
