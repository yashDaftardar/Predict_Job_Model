Full Job Description

Overview:

This position is to develop web-based applications to be used by both external and internal users. Web-based applications currently utilize the React, DevExtreme, Material-UI, Redux libraries. Must possess the ability to design and create efficient and visually appealing web applications utilizing the aforementioned libraries. The ability to interact with ASP.NET Web API is also required.


Candidate must have great (not good) React and DevExtreme development skills and the ability to review and contribute to the ASP.NET Web API code base. A clear understanding and focus on security are essential.

Responsibilities:

Reasonable accommodations may be made to enable individuals with disabilities to perform the essential functions.

  • Develop and document a React/DevExpress/Material-UI/Redux based interface to intact with our Web API.
  • Develop custom React components and JavaScript code for specialized components.
  • Participate and contribute in product design discussions and decisions.
  • Contribute to the ASP.NET Web API to create necessary API calls.
  • Work in an agile based environment.
  • Collaborate with development, technical, and business teams.
  • Conduct unit tests of proposed solutions to determine if the solution meets business and project goals.
Qualifications:
  • Highly Productive with React/Redux
  • Solid Experience with DevExtreme Components
  • Solid Experience with the Material-UI React Components
  • Fluent in Both JavaScript and CSS
  • Familiarity with ASP.NET Web API (.NET Core 3)
  • Results Driven
  • Problem Solving/Analysis
  • Professionalism
  • Organizational Effectiveness

Supervisory Responsibility

This position does is not expected to have any supervisory responsibility.


Work Environment

This job operates in a professional office environment. This role routinely uses standard office equipment such as computers, phones and copy machines. There is an opportunity of working remotely.

Physical Demands

Travel

No travel is expected for this position. The location of this position will be headquartered in Oklahoma City. The team would prefer the candidate be based in or near Oklahoma City so they can attend in person meetings with the team.


Position Type and Expected Hours of Work

This is a full-time position. Hours of work and days are flexible, and may vary depend on assigned projects. Occasional evening and weekend work may be required as job duties demand.


Required Education and Experience

  • Proven experience in developing a React web-based application
  • Experience with JavaScript, DevExtreme, and Material-UI controls
  • Experience in developing modern, efficient and scalable web-based applications
  • Ability to review .NET Web API code to understand business logic and request parameters
  • Ability to use MSSQL to review database data and structures
