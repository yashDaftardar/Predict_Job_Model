.Net Software Engineer Opportunity!

Locations: Open to remote work or at any CSI Location!

Launch your career with a dynamic and innovative company! Computer Services, Inc. (CSI) is a leader in the banking industry, offering end-to-end technology solutions to financial institutions nationwide. And we are growing! With excellent benefits and a superb work environment.

CSI is a leading provider of core and item processing solutions for financial institutions across the United States. This position would work with our software development teams to create user-friendly and innovative solutions. You will work on challenging Microsoft development projects using the .Net framework and SQL tools.

Preferred applicants will possess the following skills, education, and experience:

  • B.S. in Computer Science Software Engineering, or equivalent
  • Web Service development, utilizing a common framework (e.g. Angular, React, Vue, etc.)
  • .NET experience (C#)
  • Experience building database applications
  • Familiar with SQL Server
  • Application design, development, deployment, and support experience
  • Strong understanding of object-oriented software design, microservice-based architectures, and containerization
  • Ability to multi-task
  • Strong problem-solving skills
  • Self-motivated to learn and stay current on new and emerging relevant technologies
  • Comfortable working in an Agile/Scrum environment
  • Good verbal and written communication skills
