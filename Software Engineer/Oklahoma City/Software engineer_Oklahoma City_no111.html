SOFTWARE ENGINEER

Jasco Products Company Llc
Oklahoma City, OK 73114

The Software Engineer is responsible for the design and development of user interfaces to key software products including new functionality, as well as maintenance and enhancements of the core products comprising of our Enterprise platforms.

  • Update and develop database-driven website content using C#, ASP/ASP.NET, HTML5, CSS JavaScript, XML, AJAX, and SQL.
  • Develop code & processes in a variety of areas with our in-house custom software, create Tableau data-sources, and work with a variety of API development projects.
  • Will serve as technical resource for our BI/Workfront/API development efforts.
  • Create, advance, and implement various company data automation efforts via scripting/scheduling.
  • Will work with Product Management, Development and WMS teams to develop creative solutions satisfying technical and business requirements.
  • Take strategic ideas and help design ways to implement them.
  • Performs other related duties as required.

Competencies:

Action Oriented, Communicates Effectively, Ensures Accountability, Manages Complexity, Optimizes Work Processes, Self-Development, Tech Savvy

  • Bachelor’s or Associates degree from a college, university or trade school in computer science or a closely related field preferred, or equivalent experience.
  • Minimum 5 years of experience in one or more of the following: C#, ASP.NET, SQL, HTML5, MSSQL, Data Analytics, PM/BI Tools.
  • Skills in database driven programming such C#, ASP.NET, SQL, HTML5, JAVA, Data Analytics, SSRS.
  • High level of knowledge around system interfaces using and developing API’s, including Webhooks, Operating System API’s (Windows/IBM i), and Remote API.
  • Exposure to MVC (Model View Controller), Python, and Powershell.
  • Exposure to Workfront/Fusion platforms is highly preferred.
  • High level of programmatic automation via scripting and scheduling.
  • High understanding of JavaScript Object Notation (JSON) data formats.
  • Microsoft Windows Server 2012/2016 and IIS management exposure is necessary.
  • Experience with communication, workflow, and management systems – such as Workfront/Fusion, Share Point, Tableau/Power-BI etc.
  • Moderate to high understanding of SQL Server Reporting Services.
  • Working knowledge of Windows Server 2012/2016, SQL Server, SQL Server Reporting Services (SSRS), IIS, and publishing to IIS.
  • Ability to Code, Test and document changes for new functionality and enhancements using the appropriate tool(s) such as C#, ASP.NET, HTML, HTML5, CSS, JSON, and/or JavaScript.
