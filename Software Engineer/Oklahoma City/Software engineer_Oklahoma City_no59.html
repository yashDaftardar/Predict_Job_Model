Full Job Description

KPA exists to help organizations keep their employees safe and their businesses running smoothly. KPA recently announced the acquisition of iScout - now KPA EHS, a provider of flexible, intuitive EHS software designed for small and mid-sized organizations. With KPA EHS, our clients can easily collect, evaluate, and share safety information across their organization to cultivate a culture of safety and enable regulatory compliance.
With the combined team, we are excited for both the enhanced offering to our clients and collaboration within our growing team.

The Senior Software Developer for the iScout mobile solution is a full stack web-application generalist who also works in the native iOS and Android applications. The candidate writes and maintains high-quality code that empowers customers with the resources they need to keep their team safe. The innovative technology and user customizable platform enables clients to collect, evaluate, understand and implement safety and training information in real-time.

The Senior Software Developer

    • Writes and maintains high-quality code that gets the job done
    • Generally brings 5 or more years of experience in full stack web application development to the team
    • Collaborates with peers to meet customer needs
    • Is always learning and continuously improving
    • Is experienced in building web applications
    • Is experienced with JavaScript and interested in using it on the server (NodeJS)
    • Has worked with modern web frameworks such as Vue and WebPack
    • Is familiar with secure coding practices (OWASP or SDLC) and has an eye for security concerns like injection attacks

Technology and Frameworks we use:

    • NodeJS / JavaScript
    • Vue / WebPack
    • Git / GitHub
    • MongoDB
    • PostgreSQL
    • Objective-C for iOS
    • Java for Android
