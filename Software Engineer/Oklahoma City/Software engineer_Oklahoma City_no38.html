Job Description

Alion Science and Technology is looking for a qualified Computer Science / Software Developer to support The Strike Systems Program Office located at Tinker AFB, Oklahoma. Candidate must have experience on most of the following:

Job Responsibilities

  • Knowledgeable in DoD and Air Force Technology regulations, services, and instructions to ensure communications and computer services are in compliance with applicable regulations and instructions.
  • Must standardize procedures for account establishment, system maintenance, and system recovery.
  • Must be able to develop and maintain and implement maintenance and upgrade procedures to minimize disruption of normal business functions of organization within the installation.
  • Experience in developing network Standard Operating Procedures, and implementing audits, threat and vulnerability assessments, and applying security-related software tools to manage and control classified information systems.
  • Managing the configuration of classified information systems.
  • Multi-Level secure networks and automated cross-domain security solutions.
  • Experience designing, installing, configuring and maintaining Microsoft SharePoint
  • Virtualized server environments to include Storage Area Networks (SANs) with equipment from multiple vendors.
  • Advanced router and firewall configurations used to isolate network traffic in order to maintain security.
  • Must be able to install and configure Linux operating systems in a mixed Linux/Windows environment.
  • Standardize procedures for account establishment, system maintenance, and system recovery.
  • Ability to upload and download files across classified networks operating at different classification levels, and controlling and recording all incoming and outgoing digital media
  • Conduct self-assessments using the Management Internal Control Toolset (MICT).
  • Resolving information systems security incidents.
  • Update and maintain software system of the Logistics Management Data System (LMDS).
  • Managing relational databases to include Access, SQL, and Oracle.

Requirements

  • DoD ACAT Program experience
  • Must have at least five (5) years of information technology background in the following:
    • Java Script
    • jQuery
    • Cascading Style Sheets (CSS)
    • HTML5
    • SharePoint
    • Database Administration experience using Oracle and SQL Development.
  • Must possess one or more of the following certifications:
    • Security +
    • Information Assurance Management Level II Certificate or equivalent
    • IAT Level II certifications consist of CompTIA Security +

Must have an active Secret Clearance
