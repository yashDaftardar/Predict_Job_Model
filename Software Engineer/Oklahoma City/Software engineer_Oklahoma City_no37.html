Job Description - Overview

We are in search of self-motivated and versatile developer with 5+ years of experience. You must best a team player, have a passion for building great software and the ability to work in an energetic and collaborative environment.

This position is located in Oklahoma City, OK. Remote work not available.

Primary Functions/Job Description
Build and maintain new multi-tiered systems utilizing languages such as C#, Java, and JavaScript
Build and maintain front-end applications using Angular, React, NextJS, etc.
Maintain existing applications built with Java, ASP.NET MVC, ASP.NET Web API, Angular, etc.
Explore and develop skills in a new or unfamiliar technologies and frameworks
Communicate and coordinate with management and other departments on projects tasks
Ability to manage all phases of a project
Prepare all appropriate user and technical documentation for assigned development projects
Provide end-user production support for in-house developed and purchased software including after-hours support scheduled on a rotating basis with other developers

Job Description - Requirements

Knowledge of C#, Java or other similar programming languages
Experience in building front-end applications (JavaScript, Typescript, Angular, React, etc)
Experience in developing cloud-based applications and services (AWS, GCP, Azure, etc)
Solid understanding of HTTP and Web protocols
Knowledge of object-oriented and/or functional paradigms
Familiar with relational database design, preferably MS SQL Server or MySQL
Experience in building RESTful web services and n-tiered distributed systems
Experience in a source control system, preferably Git
Ability to write and debug SQL queries and stored procedures
Preferred Skills
Experience in building and maintaining enterprise level message-based applications
Experience in building microservices
Experience in working with containerized platforms such as Docker, Kubernetes, Openshift, etc.
Experience with Model-View-Controller architectural pattern (ASP.NET MVC, Spring or similar)
Experience in building SPAs with Angular or React and have a good understanding of SEO
Experience in building applications using NoSQL databases such as MongoDB or DocumentDB
Familiarity with securing applications with OAuth2
Degree in MIS or Computer Science
Good understanding of Agile methodology
