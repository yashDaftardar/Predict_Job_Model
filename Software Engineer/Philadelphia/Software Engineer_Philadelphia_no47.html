ThingWorx Software Engineer

ThingWorx Analytics, a PTC business, based in Wayne, PA, is seeking a talented Software Engineer to be part of an exciting and innovative team. Ideal candidates will enjoy solving challenging problems with clear models and efficient solutions, designing and implementing scalable systems that handle a large amount of data.

Day-To-Day:
You will be collaborating with team members to design, develop, integrate, test, and deploy ThingWorx Analytics software. You will be working closely with the rest of the ThingWorx Analytics R&D team while following best practices for the full software development life cycle including coding standards, code reviews, source control management, build processes, testing, and operations.

Preferred Skills and Knowledge:
  • Excellent communication skills.
  • Ability to work independently as well as collaboratively within a team.
  • Ability and willingness to innovate and learn new technologies quickly.
  • Ability to deliver rapid iterations of software features.
  • Require experience with Java, Spring, Eclipse, Unit Testing, Git.
  • Prefer experience with ORM (JPA/Hibernate), REST, Message Queues, Unix/Linux, Maven, Jenkins, Agile/Scrum, JIRA/Atlassian Tools, Distributed/Parallel Computing
  • Value experience with Microservices, Docker, Kubernetes, Apache Spark, AI/Machine Learning, ZooKeeper, Big Data, Internet of Things


Basic Qualifications:
  • Bachelor’s Degree or higher in Computer Science or related disciplines.
  • 2+ years of experience in building software applications using Java and related technologies.
  • Computer Science fundamentals in algorithm design, data structures and problem-solving.
