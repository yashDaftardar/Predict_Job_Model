Description

20 Digits is looking for highly talented full stack software engineers to help build our next generation service delivery platform. This platform embraces the latest in cloud optimized technologies in order to deliver massive scale, extreme availability and full automation as our organization rapidly expands its offerings to clients and third-party service providers leveraging our solutions.

We are looking for modern software engineers who are passionate about:

  • Building scalable, highly available application solutions that leverage public cloud services
  • Designing and developing customer and vendor facing applications to be cloud optimized
  • Working alongside business, development and infrastructure teams.
  • Providing deep software development knowledge with respect to full stack development, design patterns and object-oriented programming
  • Understanding and implementing industry best practices
  • Working as both an infrastructure and application development specialist
  • Agile software development and DevOps
  • Implementing DevOps practices such as infrastructure as code, continuous integration and automated deployment

Basic Qualifications

  • Full-Stack design and development experience (must have front-end, service layer, and backend data storage experience)
  • BS level technical degree or equivalent experience; Computer Science or Engineering background preferred.
  • 3+ years of large-scale software development or application engineering with recent coding experience in two or more of the following languages: Java, JavaScript, C/C++, C#, Objective-C, Node.js, .NET, Go, Python, PHP or Ruby
  • Good experience with Continuous Integration and Continuous
  • Experience in Agile development methodologies a must

Preferred Qualifications

  • Extensive experience designing and programming applications in an internet based or Cloud environment
  • Experience with a front-end framework such as Angular or React.
  • Experience with Mobile Development for iOS and Android.
  • Experience training newer employees
  • Experience with designing and building application using public cloud services
  • Designing and building application using Container and serverless technologies
  • Experience building highly available systems that utilize load balancing, horizontal scalability and high availability
  • Experience with continuous integration tools (e.g. Jenkins)
  • Experience with configuration management platforms (e.g. Chef, Puppet, Salt, or Ansible)
  • Strong scripting skills (e.g. Powershell, Python, Bash, Ruby, Perl, etc.)
  • Strong practical application development experience on Linux and Windows-based systems
  • Extra curricula software development passion (e.g. active open source contributor)

All employees must successfully complete a background check and drug screening prior to the start of their employment.

Starting salary is based on experience with the opportunity for an annual bonus!

20 Digits is proud to be an Equal Opportunity Employer!
