With more than one million downloads and millions of end users, Cesium is defining the standard for 3D across industries from the construction site to the battlefield, from outer space to underground.

Our leading technical team has written the books, taught the courses, and created the open standards; we are now building the platform for the 3D geospatial ecosystem together with the open-source community. It’s like a game engine to create digital twins of the real world.

We are looking for a creative JavaScript developer to contribute to the open-source CesiumJS library and applications built upon it.

What you’ll do:

  • Use JavaScript and modern software development practices to create reusable libraries, unit tests, and documentation
  • Collaborate with our development team and the open-source community to define the new standard for 3D geospatial
  • Participate in collaborative development using GitHub, code reviews, and CI
  • Work in a fast-paced continuous learning environment

What we’re looking for:

  • Excellent JavaScript development skills with a focus on modular architecture, API design, and performance
  • Fantastic communication in writing and in person, internally and externally
  • Exposure to TypeScript, ES6, npm, or git is a plus
  • Exposure to JavaScript mapping libraries, parallel programming, graphics algorithms, glTF, or WebGL is a plus
  • BS or MS in Computer Science, Math, or a related field, or equivalent experience
