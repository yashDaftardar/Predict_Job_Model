Software Engineer/Programmer

The Software Engineer, reporting to the Chief Technology Officer, will design and develop key features on the backend and frontend of the web site, using Java, JavaScript, SQL, HTML and CSS, and several open-source technologies on the Linux platform.

This role involves developing cutting edge software on a high-energy team, tasked with writing efficient, accurate, clear code to specification. You will be utilizing programming standards and approaches to make certain efficient program logic and data manipulation.

The successful candidate will be a part of all phases of the software development lifecycle including mapping new features to meet requirements, and use many new technologies throughout the development process.

Key Responsibilities:

  • To develop, code, test, deploy, document, maintain and support new web based features and enhancements for our domain name portfolio and GTW related products.
  • Working directly with our Architect and CTO to contribute to the overall architecture and implementation or the core software platform and high performance engine.
  • Designing initiatives ranging from Internet development and multiplatform systems integration to network security and technical support.
  • Conduct software performance tuning, reliability and maintainability improvement tasks.
  • Support the design and development of new product offerings and the design and development of enhancements and extensions to existing systems.
  • Participate in design reviews, code reviews, cross functional teams, risk assessments, and unit test inspections.
  • Identify and evaluate new technologies for implementation or integration.
  • Collaborate with cross-functional teams facilitating decisions on new features and problem resolution.
  • The successful candidate will be committed to delivering quality work on an aggressive schedule, and must be able to demonstrate team work and ability to interact with other developers and testers.

Requirements:

  • Strong software development skills, including design, development, testing, deployment and support.
  • 3+ years previous experience as a Software Developer, Software Engineer, Java Developer, J2EE Developer, or Mobile Web Developer with some or all of the following technologies: Java, J2EE JSP, JSF, Servlets, JDBC, HTML/HTML5, JavaScript, AJAX, XML, Web Service Technologies including SOAP and WSDL, Spring (2.5), Hibernate and SQL.
  • Experience with application servers (Apache, Tomcat, JBoss), Web Services, WebSphere or WebLogic.
  • Ability to write clear documentation for all code and processes.
  • Experience with source code control software such as CVS, Perforce, etc.
  • Previous experience creating core platforms, core products, high transaction software, highly scalable software, or social network software.
  • Software design and development experience with mobile device applications for the new generation Smart Phones and devices is a plus (Blackberry, iPhone/iPad, Andriod)

Skills/Qualifications:

  • Bachelor’s or Master’s Degree in Computer Science or a related technical field (Computer Engineering, Computer Information Systems, Software Engineering, Systems Engineering) from a top school.
  • Minimum 3 years web based software development, including design, implementation and maintenance responsibilities.
  • Proven experience with both server and client side web technologies.
  • Ability for Testing/Quality Control.
  • Problem solver with proven debugging and diagnostic skills.
  • Strong work ethic and a high level of professionalism and integrity.
  • Ability to interface with clients and functional analysts to understand systems design and requirements.
