Required Skills

Analyze, configure, develop, and implement PeopleSoft Financials 9.x. Develop conversion programs using Component interface and File Layout for Accounts payable, project costing, and supplier maintenance. Integrate Purchase Orders and Invoice interfaces between multiple systems. Conduct Bank reconsolidation process with client financial systems using knowledge of supplier payment files creation for multiple bank formats and secure transfer of payment files. Develop interface programs using application engine, web services, and DB links for creating external systems like MAXIMO, Hyperion, ARIBA, I2 and COBOL manufacturing tools. Load configuration data into PeopleSoft and XML publisher utilizing knowledge of DMS scripts and EXCEL. Develop custom bolt on Records, Pages, and Components involving workflow and external links. Other similar professional duties as assigned. This position will use technology including PeopleSoft Accounts Payables, Employee Travel and Expense, Project Costing, Purchasing and Receivables, Commitment Control, Advance Work Flow Engine, People Tools, People Code, Application Engine, File Layout, Workflow, Web services, Data mover scripts, PS Query, SQR, XMLP, Component Interface, web services, Oracle and MS SQL.


Position requires up to 30% domestic travel.
