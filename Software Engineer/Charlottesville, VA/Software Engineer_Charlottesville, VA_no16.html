Role Summary

As a Software Engineer your mission is to take on the engineering projects that ensure the quality and security of the entire CoConstruct technology platform. You’ll team up with Software Engineers, Product Managers, Architects, and others. You will tackle important software development projects that aim to improve product quality, security, user experience, and system performance. You will perform security related tasks and implement solutions.

Key Accountabilities

  • Develop new functionality.
  • Analyze and resolve defects.
  • Work on performance enhancements and maintenance of the software delivery lifecycle, including all aspects of secure coding practices and integrations.
  • Analyze and resolve security findings based on industry best practices and guidelines.
  • Stay up-to-date on emerging technologies and security best practices across the organization.
  • Respond and implement solutions to security related incidents.


Requirements/ Experience

  • 3 or more years of experience with full-stack development.
  • A bachelor’s degree in computer science or equivalent experience.

Skills

  • Programming skills with object-oriented languages (C# or Java), Web UI frameworks, and SQL.
  • Knowledge of security assessments and common web application security vulnerabilities.
  • Understanding of engineering/development best practices and (backend/frontend/mobile) systems.
  • In-depth knowledge of web technologies, browser behavior, and optimization techniques.
  • Able to troubleshoot and solve problems quickly and independently.
  • Understanding of the web development process, including design, development, deployment, and some back-end knowledge.
