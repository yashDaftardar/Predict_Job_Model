Description

As a member of the Cell Data Engineering team, your software will help contribute to accelerating the pace of battery innovation. You will oversee the user interfaces that Cell Engineering relies on to make decisions on the next battery technology that will go into Tesla vehicles and energy installations. You will oversee designing and creating the websites for building test plans, monitoring test status, and viewing results as well as the immediate backend API that routes to other microservices.


Responsibilities

  • Designing and creating our team’s web applications.
  • Work closely with cell engineers to understand UX challenges, suggest improvements, and accelerate our team’s ability to make decisions on cell programs.
  • Responsible for creating unit testing for the web applications designing and creating the web application.
  • Creating mock pages to be vetted and iterated over.
  • Keeping a close connection to our users for prototyping and creating mock pages


Requirements

  • BS/MS in Computer Science or a similar background with strong software engineering experience
  • Experience building modern web applications using React/Redux, Angular, or similar JavaScript component-based libraries.
  • Strong UI and graphic design sensibilities
  • Experience with Python or similar language for creating immediate backend APIs.
