Summary

Varex Imaging is looking for a motivated summer intern to help our FPGA team extend a large set of automated simulation and hardware script verification tests into a flexible regression test suite. Work will involve setting up PCs for testing, interfacing with detectors, writing Python test scripts, debugging Python test scripts, analyzing and debugging Verilog simulations and working with another intern to connect everything to our Jenkins build server. A senior engineer will be available to train and mentor this individual.

Essential Duties and Responsibilities:

  • Learn X-Ray safety procedures
  • Evolve a plan of action with the second intern
  • Set up Linux and Windows PCs
  • Interface with detectors over 10Ge or CoaXPress
  • Run+Debug existing Python scripted tests / develop new scripted tests
  • Develop software interfaces to new hardware as needed
  • Run/Debug/Write SystemVerilog automated testbenches
  • Automate testing and reporting with the Jenkins build server
  • Document the test system developed
  • Report weekly to the FPGA team on progress

Required Knowledge, Skills, And Abilities:

  • Basic-level programming skills
  • Scripting – prefer Python/C/TCL/Bash
  • Some Verilog or System Verilog experience helpful
  • FPGA experience helpful
  • Experience with PC hardware
  • Experience running experiments, scientific method, data analysis
  • Some statistics experience useful
  • Familiarity with lab equipment
  • 2nd year undergraduate or above in any science related field (Physics, Engineering, Computer Science, etc…)
  • No prior work experience required
  • Ability to write reports / Word / Excel

Travel will not be required. Work will be partially remote and partially onsite. See the job posting for more information.
