Zycada Networks, Inc. seeks Software Engineer in Campbell, CA under the direction of the Vice President of Engineering to design, develop and deploy specialized application software, distributed system modules for products using C/C++, python, bash in Linux based environment.


Duties:

  • Design and develop applications using transmission control protocol (TCP) / internet protocol (IP), hypertext transfer protocol (HTTP).
  • Analyze, pinpoint and resolve production issues in border gateway protocol (BGP), domain name systems (DNS),) using tools like tcpdump, strace, gdb as well as improve efficiency of software by profiling and analyzing data with tools like gprof, pprof.
  • Design and develop policies and tools to secure Zycada’s cloud infrastructure.
  • Position based at company office in Campbell, CA area and may telecommute from home in greater Campbell, CA area with occasional commute to office for meetings, trainings, etc.

Requirements:

  • 2 years of work experience in programming experience with C/C++, python, bash in a Linux based environment using STL containers, boost libraries and design patterns (singleton, proxy, chain of responsibility, publisher/subscriber); bash (awk, cut, grep, find) to write tools and Unix terminologies (inode, binaries vs libraries, static vs dynamic linking, out of memory, top, system activity report etc). 2 years work experience in system and network monitoring/troubleshooting using tools gprof, tcpdump, gdb and attaching a debugger to a process and debugging issues and using alternate debugging tools strace, tcpdump, sar and hypervisors vs baremetals, virtual machines,VM attacks, security techniques.
  • 2 years of work experience with designing and developing distributed applications representing an application in object-oriented notations (sequence diagrams) and resource contentions, mutex, deadlocks, causal ordering, consistency.
  • 2 years work experience in networking protocols (TCP/IP, BGP, OSPF, HTTP, TLS, DNS) with TCP Finite state machine, congestion control algorithms.
  • 1 year work experience utilizing of celery python framework and familiarity with asyncio, coroutines.
  • 6 months or 3 credit hours of graduate coursework knowledge in lock free data structures and algorithms, unicast/anycast/multicast in the context of border gateway protocol (BGP), Djikstra’s algorithm in the context of OSPF, HTTP methods GET/POST/PUT/PATCH/DELETE, response codes – 1xx/2xx/3xx/4xx/5xx and use, and DNS record types (CNAME, A, ALIAS, MX, NX, NS) and its role in internet architecture, computing, storage, networking & security in cloud infrastructures.
