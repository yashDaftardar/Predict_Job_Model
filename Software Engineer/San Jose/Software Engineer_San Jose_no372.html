As a Programmer, you would be working with experienced engineers creating solutions which will enable future deep space exploration, provide the next generation of spacecrafts or provide support to the U.S. Intelligence Community. Lockheed Martin Space is building the spacecraft that enable these missions and more.

The following are examples of duties that may be included:
  • Providing qualified embedded flight software to meet program needs
  • Supporting the full life-cycle development of spacecraft flight software solutions following the program Software Development Plan (SDP) to include requirements analysis, object oriented analysis & design, code and unit test, integration, and support to formal test / delivery.
  • Performing object oriented analysis and design
  • Generating engineering documentation, reports, drawings, and flow charts
  • Completing the coding and unit level testing
  • Supporting the final system integration and testing, generating the required documents for assembly delivery

Basic Qualifications:
  • Associate Degree or equivalent experience in Computer Engineering, Information Technology, Computer Science, Information Assurance/Cyber Security or related discipline.
  • Basic understanding of STEM (science, technology, engineering, math) principles as well as the ability to interpret, generate, and present technical material
  • Demonstrated organizational skills with attention to detail to create high-quality work products
  • Ability to work in an Agile environment and handle multiple priorities and internal customers
  • Desire to continually learn, ambition to proactively address challenges, and aspiration to work in a dynamic team environment
  • Ability to obtain security clearance
Desired Skills:
Knowledge and/or experience in one or more of the following areas:
  • Familiar with real-time development environments and languages (C, C++).
  • Experience with Python, Matlab, Simulink, R, or equivalent data science toolsets
  • Experience or classroom knowledge with embedded software / hardware
  • Network experience – LAN/WAN/TCP/IP Protocol
  • Information Security Domain knowledge
  • Database experience – SQL/NoSQL
  • Proficiency in Microsoft Office Applications
