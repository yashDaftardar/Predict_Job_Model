The Role

Tesla Energy is looking for a passionate, driven Embedded Software Engineer to bring the next generation of energy storage and energy generation products to the market. The candidate should display a hunger for owning the product and delivering top quality firmware. The responsibilities range from architecting and implementing firmware to supporting production and fleet improvements.

Responsibilities

  • Develop and improve firmware for new and existing products, in C
  • Drive firmware architecture for performance, scalability, robustness, and tight integration with the hardware
  • Hands-on hardware bring up, system debugging and code optimization
  • Attention to detail shown through continuous improvements and enforcing best practices within the team, and also the larger organization
  • Deliver optimized solutions co-designed between firmware and hardware teams to meet aggressive schedules

Requirements

  • BS or higher in Engineering, Computer Science, Physics or proof of exceptional skills in related fields, with practical engineering experience.
  • Capable of delivering top quality C code in a real time embedded environment
  • Strong understanding of object oriented programming and familiarity with C++
  • Expert software fundamentals including data structures, code modularity, task and resource management in a RTOS, and coding practices
  • Eager and determined to deliver complete, working code built through test driven development using unit tests, software in the loop, hardware in the loop methodologies.
  • Positive outlook to tackling challenging problems and desire to excel
  • Own assignments and take full accountability for overall team success
  • Experience with Wi-Fi, ethernet, or cellular on embedded systems is a big plus
