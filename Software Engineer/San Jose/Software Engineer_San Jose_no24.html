Company Description

Microchip Technology Inc. is a leading provider of embedded control applications. Our product portfolio comprises general purpose and specialized 8-bit, 16-bit, and 32-bit microcontrollers, 32-bit microprocessors, field-programmable gate array (FPGA) products, a broad spectrum of high-performance linear, mixed-signal, power management, thermal management, radio frequency (RF), timing, safety, security, wired connectivity and wireless connectivity devices, as well as serial Electrically Erasable Programmable Read Only Memory (EEPROM), Serial Flash memories, Parallel Flash memories, and serial Static Random Access Memory (SRAM). We also license Flash-IP solutions that are incorporated in a broad range of products.

Job Description

The Software Integration Team has responsibility for Configuration Management, Development Tools and Inrastructure, Continuous Integration, Testing Automation, and Software Release.

Responsibilities include :

  • Manage release pipeline.
  • Identify & develop new scripts, tools or applications to automate/streamline the build, integration and release process.
  • Investigate issues related to builds, continuous integrations, release and deployment .
  • Administer and support current development environment tools: ClearCase/eClearCase, SVN/BitBucket, Jenkins continuous integration, Bugzilla, JIRA.
  • Support cross-functional and global teams using Windows and Linux development environments.
  • Work flexible hours to support a global team.

Job Requirements

Qualifications:

  • Bachelor’s or Master’s degree in Computer Science or equivalent.
  • New college graduate accepted.
  • Strong programming foundation (C++/Java/Python/NodeJS).
  • Web programming knowledge is highly desirable.
  • Linux, Python, Bash scripting experience.
  • Familiarity with repository management such as Git, SVN.
  • Familiarity with container build and management tools such as Docker, Kubernetes.
  • Familiarity with infrastructure management/orchestration (Ansible/Puppet/Chef).
  • Knowledge of continuous delivery and software lifecycle management tools (JIRA, Git, Jenkins).

Additional skills:

  • Ability to work with minimal supervision and/or remote team is strongly required.
  • Good interpersonal and communication skills
  • Self-starter, motivated, innovative
  • Must be efficient, organized and extremely detailed oriented.
