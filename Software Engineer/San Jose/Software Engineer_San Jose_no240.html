HPE Ezmeral is an enterprise grade solution that accelerates customers’ journey from hybrid to cloud through its unified container platform. This solution is designed for cloud-native and non-cloud native applications which allows the customer to run on any infrastructure: either on premises, in multiple public clouds, in a hybrid model, or at the edge. Combining 100% open source Kubernetes with unique software innovations and integrated persistent storage technology, customers can now modernize existing enterprise applications without re-architecting.

Seeking talented System Software Engineer to design and enhance HPE Ezmeral. Candidate will design and build Ezmeral Data Fabric Kubernetes that addresses various operational and security challenges. The candidate will be exposed to a broad variety of emerging AI/Machine Learning and Big Data technologies.


About You:

  • Bachelors or masters degree in computer science, computer engineering, or a technology-related degree program

  • Experience with container-based platforms (Docker, Kubernetes)

  • Experience with AWS, GCP, Azure, or other Public Cloud Services

  • Experience in designing and supporting enterprise infrastructure system software

  • In-depth knowledge of Linux.

  • In-depth knowledge of storage systems

How You'll Make Your Mark:

  • Work with the following technologies - network configuration, fault tolerant, monitoring, and data security.

  • Gain experience with C and Python. Knowledge in GoLang and Erlang.

  • Deliver multi-tenant, enterprise software, delivered as a service (SaaS) in both on Premise and Cloud Infrastructure

  • Debug scalable cluster operational issues is a plus.

  • Work within Machine Learning operation.

  • Work with Agile development practices and familiar with tools like Jira
