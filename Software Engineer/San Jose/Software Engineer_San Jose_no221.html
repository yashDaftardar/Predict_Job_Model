About the team

Are you interested in making a real impact on a product enjoyed by hundreds of millions of users worldwide? We are eBay's Search web frontend team, and we're looking for a strong senior web engineer to join us. As a small team of dedicated engineers, we strive to provide our users with world class experiences. We work closely with product managers, designers, and our fellow engineers to deliver a robust and responsive web application. If you're interested in seeing the software we are building, please go to www.ebay.com and perform a search.

Responsibilities

  • Work closely with PMs to clarify requirements and provide any technical expertise
  • Work with designers to pave development approaches
  • Work hand in hand with our services team to define interfaces, scope integration efforts, and prototype proof of concepts
  • Conduct and participate in design, PR, test plan, and release reviews
  • Write and maintain unit, functional, and automation tests
  • Quickly resolve issues related to speed, security, and accessibility
  • Rotational on-call support monitoring production metrics and providing support if issues arise

Requirements

  • BS/MS in Computer Science/Engineering or related technical discipline
  • 3-5 years of experience in designing, building, debugging, refactoring and releasing large web apps with support for both mobile and desktop browsers
  • Expertise in NodeJS, JavaScript, HTML, CSS, LESS, JSON and browser dev tools
  • Expertise in REST API interface design
  • Proficiency with web template engines, Marko preferred, testing frameworks, environment configurations, GIT, JIRA and IDE
  • Familiar with Java and JSP
