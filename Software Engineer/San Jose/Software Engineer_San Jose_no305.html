Full Job Description

The Role

As a member of the off-board firmware development team, you will be responsible for delivering high quality code for the superchargers, charging adapters, and various other types of charging equipment. Come join a team of knowledgeable engineers to build the next generation of robust EV charging infrastructure using cutting edge software development tools and practices.

Responsibilities

  • Deliver high-quality C code in a real-time embedded environment.

  • Develop component level tests on our software-in-the-loop (SIL) platform.

  • Integrate and debug features at the system level and release the changes to production.

  • Specify, design, and implement functionality and behaviors of embedded subsystems.

  • Hands-on hardware bring-up, system debugging and code optimization.

  • Make performance and optimization trade-offs to meet product requirements.

Requirements

  • BS in EE/CE/CS, or the equivalent in experience with evidence of exceptional ability.

  • 2+ years of experience in embedded development in robotics, real-time controls, or system software.

  • Strong proficiency in C, familiarity with C++ a plus.

  • Fluent in software fundamentals including clean software design and maintainability.

  • Experience with the embedded microprocessor tools / JTAG debugging interfaces.

  • Test driven development and familiarity with developer driven automated testing.

  • Ability to collaborate and communicate complex technical concepts concisely.

  • Must be comfortable with hands-on lab work (including the use of scopes/logic analyzers, CAN tools, soldering irons, etc as needed to support integration).

  • Understanding of physics and basic principles is highly preferred.
