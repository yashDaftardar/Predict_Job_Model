Full Job Description

Among the active and planned work that this position may participate in are:
- Building, configuring and managing Esri ArcGIS Enterprise configuration (10.8 and later) as target platform for enterprise application and services as well as ArcGIS Pro (2.5 and later)
- Migrating Esri 10.2.1 legacy systems and Geometric Networks to Esri’s Utility Network and 10.8 Enterprise platform
- Migrating Smallworld GIS systems to Esri’s Utility Network and 10.8 Enterprise platform
- Building and maintaining components (e.g. Python and .Net scripts and webservices) to integrate with other enterprise systems
- Building and maintaining mobile and web based applications to support a variety of field and in-office needs

In-depth experience desired. Knowledge, skills and abilities, as well as experience desired in operating systems include:
- Experience in application configuration of Windows Server (2012, 2016, 2019)
- Web server (IIS)
- Load balancing and scaling for performance optimization and high availability
- Related enterprise services, including networking, security, virtualization
- Experience with Oracle and/or SQL Server
Preference will be given to candidates with experience in the following:
- Esri ArcGIS Enterprise Platform Deployment, Administration, and Optimization
- Schneider Electric’s ArcFM product suite
- Configuring and optimizing ArcGIS PRO, APR, Utility Network
- Esri Utility Network and associated data models and tools
- Configuring and managing Map/Feature/Geoprocessing Services
- Configuring and optimizing ArcSDE (10.x and beyond), versioning and replicas, ArcFM Geodatabase Manager and ArcFM Process Framework
- ETL platforms such as FME and development of scripts/workbenches

Candidate should have experience with one or more of the following:
- Experience with the full software development lifecycle
- Python 2.7 or 3, with exposure to python libraries such as Flask, ArcPy, and SqlAlchemy
- C#.Net, with exposure to ArcObjects API
- Javascript, with exposure to JQuery, DoJo, and the Esri Javascript API, and Boostrap.css
- Relational data modelling and SQL

Education Bachelors:
preferred Discipline Business, Computer Engineering, Computer Science, Information Systems, Mathematics
(willing to possibly accept a candidate with an Associates and 5 years of experience)

Top 3 Required Skills
1) Experience with the Esri platform, either system management (installation, configuration, monitoring, troubleshooting)
2) Experience with Python (2.7 or 3.0), particularly with the Arcpy library.
3) Experience with Esri’s Utility Network and associated data models and APIs.

Nice to Have Skills:
IIS management and configuration.
SQL (SQL Server or Oracle) database management
Understanding of Enterprise security protocols
Esri’s ArcGIS Portal administration

High Level Project Overview:
This position will be responsible for Production Support of a variety of GIS related components – Esri platform software, geodatabases, applications and services across the enterprise. This may involve managing user access, upgrading Esri software, and development, maintenance, deployment, and support of custom components and applications.
