Full Job Description

We are hiring an ORACLE ADF SOA BPEL Developer with knowing of HIPAA X12 transactions. These Senior Developers will support the Manager and development leads with Development, implementation and integration of a robust EDI Solution. They will ensure that component planning/activities are aligned with the Integration Strategy; Collaborate with Project Managers, Business Analysts and Subject Matter Experts on the Encounter and integration component requirements. Collaborate with client's staff and vendors on the implementation requirements of the encounter / Operational Data source integration for the system.

Location: Columbia, SC

Duration: 1 2 months

Candidate Type: W2 Only; Unfortunately NO Corp-to-Corp or 1099


REQUIRED SKILLS

Seven or more years in a development role using the following skills :
  • Design and develops SOA BPEL composites using Oracle JDeveloper 11g &12c
  • Demonstrable experience with JMS Publish/Subscribe model
  • Develops web services using SOA Suite JCA Adapters, XML (XSLT, XPath, XQuery etc.) and JAVA/JAVA Script, CSS3; HTML5
  • Applied experience using Oracle Fusion B2B, two or more years preferred
  • Minimum 5 years mapping experience with HIPAA transaction sets (270/271, 274, 276/277, 278, 820, 834, 835, 837I, 837P,837D,997/999/TA1)
  • Experience mapping NCPDP D.0 and Post Adjudication
  • Experience using Oracle Healthcare Adapter for ASC X12
  • Experience with Oracle ADF SOA / Web Service Strategy
  • Ability to create detailed logical and executable maps for transactions
  • Ability to understand and create logical and executable maps from copybooks, data models, WebMethods, etc.
  • Ability to create and execute test plans for transactions
  • Ability to create process flows for transactions using Visio and PowerPoint
  • Ability to create companion guides for transactions and other methods of data exchange
  • Design, develop and implement complex enterprise Oracle SOA /EDI and B2B integration solutions to meet clientâ€™s business needs.
  • Experience in developing applications using Oracle Fusion Middleware technologies like Oracle SOA suite 11g, webservices, Oracle BPEL process Manager, Mediator, business Rules, Work flow.
  • Experience on Oracle Enterprise Service Bus, Oracle B2B and Oracle AIA methodologies.

PREFERRED SKILLS:

  • Experience with DEVOPs tools like: Jenkins, GitLab, etc
  • Prior experience working in areas of emerging technology and driving new points of view and capabilities
  • Experience with defining and creating highly functional design and coding in a fast paced evolving environment

  • Solid testing and validation skills and understanding of simulated data
