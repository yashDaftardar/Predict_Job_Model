DevOps Associate Engineer

As a member of the DevOps program, the DevOps Associate Engineer integrates people, processes, and tools to create reliable, scalable, and efficient solutions for our engineering teams by designing, developing, implementing, and managing the low to moderate complexity software development build, deployment, testing and release automated pipelines/processes.

Responsibilities :

  • Designs, develops, implements, and manages the software development build, deployment, testing and release automated pipelines/processes to achieve scalable Continuous Integration/Continuous Deployment (CI/CD) objectives with close direction from a more senior engineer.
  • Builds and tests automation tools for infrastructure provisioning with close direction from a more senior engineer.
  • Introduces CI/CD processes, technologies and tools to all the engineering teams with close direction from a more senior engineer.
  • Builds dashboards to monitor the status of applications and tools.
  • Works closely with engineers of other specialties (including Cloud Engineers, Software Engineers, DBAs, Network Administrators) to ensure that business requirements are met, using the right tools for the job
  • Develops and maintains low to moderate complexity technical documentation and processes
  • Supports existing systems and automating root cause solutions
  • Troubleshoots and supports low to moderate complexity build, test, and deployment incidents if they happen with close direction from a more senior engineer
  • Debugs and resolves low to moderate complexity software bugs or issues with close direction from more senior engineers and finds the real root cause and provide a fix without collateral damage.
  • Writes automated unit and integration-level tests with direction from other team members.
  • Follows development standards and effectively demonstrate technical solutions to other engineers in code reviews.
  • Perform basic source code management tasks at the direction of more senior engineers.
  • Perform other related duties and activities as required.

Requirements :

Education and Work Experience:

  • Bachelor’s degree, or higher education level, or its foreign equivalent, in Computer Science, Computer Information Sciences, and/or related field; related work experience can be substituted for educational requirement.
  • Total Work Experience: 1 year minimum

Specialized Knowledge, Skills, and/or Abilities:

  • Familiar with automated build, deploy and test processes as well as package/dependency management
  • Familiar with Object-oriented design, Java or .NET development, Relational OLTP queries and Relational database design
  • Familiar with XML/XSLT document design, Powershell, vbscript, JavaScript development
  • Familiar with operating scripting language like Pearl, Ruby, etc.
  • Familiar with developing and maintaining CI/CD process for OnPrem and SaaS applications by accessing tools like TeamCity, Azure DevOps, etc
  • Familiar with operating Windows environment with good command over any scripting language such as Powershell, etc.
  • Familiar with using configuration management tools like json, Puppet
  • Familiar with installer/package creation using tools like Nuget, Wix etc.
  • Familiar with building and handling VM’s, Containers utilizing tools like Azure Portal and DevTest labs, Kubernetes, Docker, etc.
  • Familiar with building, designing and maintaining cloud-based applications with AWS, Azure, etc
  • Ability to manage to deadlines, communicate in a team, and operate independently with guidance
  • Understands how to estimate, the Software Product Development Lifecycle with Agile methodology

Other Requirements:

  • Travel: 0-10%
  • Work Authorization: Legally authorized to work in the country of the job location.
  • Physical: Exerting up to 10 pounds of force occasionally and/or negligible amount of force frequently or constantly to lift, carry, push, pull or otherwise move objects, including the human body. Sedentary work involves sitting most of the time. Jobs are sedentary if walking and standing are required only occasionally and all other sedentary criteria are met.
