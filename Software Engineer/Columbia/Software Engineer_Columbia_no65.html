We are hiring a Senior .Net Programmer Analyst who will serve as technical specialist on complex projects involving Internet and server technologies; analyzes project requirements and recommends techniques, tools and methodologies to achieve project goals.

Must have in depth knowledge of computer software, and development life cycle processes are necessary. Expert knowledge in analysis, design, testing, debugging, and maintenance techniques are essential. Knowledge and experience utilizing Microsoft products, especially those such as COM+, Classic ASP, VB 6.0, XML, HTML, client-scripting languages, .NET, WCF, as well as the latest MS RDMS tools and techniques, including the latest generation of Window Server OS are vital.

Added requirements would include a familiarity of ACH, PCI, FTI PUB1075, and NIST standards. Working experience in these above areas would be an added benefit to the hiring manager and functional team.
