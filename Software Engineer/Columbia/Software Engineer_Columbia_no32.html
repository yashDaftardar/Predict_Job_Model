We are hiring a System Software Programmer

Those with GIS experience are strongly encouraged to apply!

Location: Columbia, SC

Duration: 12+ months


DAILY DUTIES / RESPONSIBILITIES:

Enhance and maintain QA/QC processes and integration routines (custom commands) using Intergraph’s GeoMedia programming
Incorporate requested enhancements to improve the LADC process
Resolve issues/problems identified in LADC process testing
Perform additional data testing to mitigate future LADC processing problems.
Resolve issues/problems with RIMS data processing to ensure RIMS compliance
Provide documentation and training support for final LADC products.
Track and maintain daily/weekly work effort.

REQUIRED SKILLS (RANK IN ORDER OF IMPORTANCE):
Minimum 8 years of experience in GIS type software development
  • Oracle 11G
  • VB.NET
  • Microsoft Visual Studio .NET (2010 or greater)
  • MS Access
  • ESRI shape file

PREFERRED SKILLS (RANK IN ORDER OF IMPORTANCE):

  • Intergraph GeoMedia Transportation Manager
  • Intergraph GeoMedia custom commands – modal and modeless
  • Recursive programming
  • Transportation linear referencing systems
  • Understanding of transportation-sector related GIS and Roadway inventory data


REQUIRED EDUCATION/CERTIFICATIONS:

BACHELOR DEGREE IN ENGINEERING, COMPUTER SCIENCE OR RELATED FIELD
