Full Job Description

Summary

Plan, design, and develop complex and business critical software solutions, utilizing appropriate software development methodologies either individually or as part of a project team. Responsible for ensuring that browser-based systems and applications meet client requirements and operate properly, using software development methodologies.


Description

  • 25% Develop functional design and technical specifications, concept diagrams, and work flow diagrams, in coordination with analysts, development, and/or applicable administrative areas, including user interfaces for web applications, maintenance, monitoring, and software builds and releases.

  • 25% Develop coding to implement project designs for new features or new browser-based systems applications, and debug existing code.

  • 20% Ensure team members understand the business requirements that drive the analysis and design of quality technical solutions, organizational standards are maintained, and applications are delivered according to project timeliness. Provide expert system knowledge, perform research, attend meetings, and review documents.

  • 20% Provide post implementation support, including creation of systems enhancements, and instruction to production support team. Research customer issues and determine appropriate resolutions.

  • 10% Provide estimates as to the duration and cost of future projects.

Must be able to obtain a security clearance, which requires U.S. citizenship.
