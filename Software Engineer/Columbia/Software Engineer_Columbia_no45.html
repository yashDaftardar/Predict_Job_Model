Sr. AWS Developer (Jr. Architect)

Position Description
CGI has an immediate need for a Sr. AWS Developer (Jr. Architect) to join our financial services team in one of our several CGI Offsite and Onshore Delivery Center locations. This is an exciting opportunity to work in a fast-paced team environment supporting one of the largest leaders in the secondary mortgage industry. We take an innovative approach to supporting our client, working side-by-side in an agile environment using emerging technologies.
  • We partner with 15 of the top 20 banks globally, and our top 10 banking clients have worked with us for an average of 26 years!
  • We have over 73,000+ CGI Members in 40 countries and over 5k+ loyal Clients who are leveraging our end-to-end services across the globe

Your future duties and responsibilities
  • Hands on development of applications in AWS and JAVA platform
  • Providing solutions based on Enterprise Integration patterns, Event Driven Architecture, API, Web services, SOA and Micro Services
  • Developing solution architecture across multiple architecture domains such as business, data, integration, resiliency and application
  • Application solution design in AWS using AWS native services
  • Data Analytics solutioning in AWS cloud using AWS native services (EMR, Redshift, Athena etc.), Apache Spark framework and using Python

Required qualifications to be successful in this role
  • 10+ yrs of hands on experience in Java and AWS technologies
  • 2+ yrs of experience in defining deployment and resiliency architectures using serverless architecture like ECS, Fargate, Lambda etc.
  • 2+ yrs of experience working with Application servers and containers such as JBoss, Tomcat, WebLogic, Oracle Application Server
  • 2+ yrs of experience w/performance optimizations such as improving scalability, availability, throughput, failover, etc.
  • Very good understanding of AWS Cloud Security.
  • Demonstrated experience of various Cloud Technologies, industry trends and solutioning experience in Public, Hybrid Cloud and PaaS will be essential.
  • Ability to work with both senior management level executives and development teams.
  • Excellent communication, presentation, and documentation skills.
  • Professional demeanor

Tools
  • Skilled in AWS offerings, development and networking platforms.
  • Experience using APIs for developing or programming software
  • Experience using JIRA, Confluence, Visio and other Confluence tools
  • Skilled in XML, Java, JavaScript, Kubernetes, Docker, VMWare. Terraform

EDUCATION REQUIREMENTB
Bachelor degree in Computer Science, Information Systems or related field
AWS Solution Architect Associate or Professional certification.
