Job Summary

DevOps Engineer will design and configure cloud infrastructure in AWS that is secure, scalable, and highly available. You will work closely with engineering team to define infrastructure and deployment requirements. Strong experience with DevOps in AWS is required for this role.

Responsibilities and Duties:

  • Design and configure secure, scalable and highly available cloud infrastructure in AWS
  • Work collaboratively with software engineering team to define infrastructure and deployment requirements
  • Ensure configuration and compliance with configuration management tools
  • Support teams in the automated build management process using CI/CD tools
  • Provision, configure and maintain AWS cloud infrastructure defined as code
  • Troubleshoot problems across a wide array of services and functional areas
  • Build and maintain operational tools for deployment, monitoring, and analysis of AWS infrastructure and systems
  • Perform infrastructure cost analysis and optimization


Qualifications and Skills:

  • 5+ years experience building and maintaining AWS infrastructure
  • Experience deploying and managing infrastructure with Terraform
  • Hands-on experience deploying in AWS
  • Strong experience with CI/CD pipeline
  • Experience with Docker and Kubernetes
  • Scripting experience using Python
  • Good knowledge of AWS Code Pipeline

Desired Certifications

  • AWS Certified DevOps Engineer

Type and Location:

  • Full-time, W2 with competitive salary and benefits
  • Position is located in Columbia, SC
