Embedded Software Test Engineer, R&D - Lexington, SC

As an Embedded Software Test Engineer for Apex Tool Group, you will be part of a global, diverse team that is truly dedicated to anticipating and reshaping how tools are made tomorrow – with higher quality at better cost. You will build strong, personal relationships as you work together to push the boundaries of innovation. The environment will be fast-paced, exciting and challenging, but you will be inspired to adapt and evolve.

In this Lexington, SC, based position, your core responsibility will be to develop and improve test process and routines in support of new software releases for a variety of tool and controller products.


This position does require the ability to interface with engineering, quality, and production personnel.

Key Responsibilities
  • For individually assigned specific test projects, you will be responsible for the following primary job functions:
  • Subject matter expert on software development; this position will improve, refine, and innovate on key product performance Critical To Quality (CTQs)
  • Work with software testers, system testers, product technical specialists and customers to develop software test methodologies for complex systems; troubleshoot, debug, and optimize software based on results
  • Read, understand, and evaluate C/C++ code files for unit test; Create environments and write unit tests
  • Develop and optimize current practices, processes, and standards.
  • Create software and documentation per the requirements and specifications and complete unit testing to verify functionality
  • Test data analysis, post-processing, trend analysis. Writing test reports and technical papers
  • Assisting in developing software for new and existing test equipment.
  • Providing engineering support to others while working in a team, have collaborative and knowledge-sharing attitude

Education & Experience
  • Bachelor or master’s degree in Software Engineering, Electrical / Electronic Engineering, or similar field
  • Minimum of 3 years’ experience with software test and test automation of embedded systems.
  • Experience in software development (C/C++; Linux; Phyton, svn, git).
  • Ability to write software to implement automated product testing
  • Knowledge of microcontrollers with Linux (ARM) and electronic hardware (I²C, CAN, SPI).
  • Knowledge of motor control
  • Responsible, independent, and solution-oriented way of working.
  • Team-player, willing to share knowledge and support team members in a global team.
  • Fluency in speaking and writing German is desired, but not required.

Core Competencies
  • Solid analytical and problem-solving skills
  • Good interpersonal skills
  • Strong communication and organizational skills
  • Be able to work with minimum supervision and willingness to learn are crucial
  • Ability to interact with all levels of company personnel to review, discuss, and resolve product testing issues
