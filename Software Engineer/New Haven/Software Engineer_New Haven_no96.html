Thinklogical (a Belden Brand) , is looking for a Full Stack Software Engineer for our location in Milford, CT . Our Full Stack Software Engineer will be the architect and developer for web server/client applications.

Responsibilities of our Full Stack Software Engineer:

  • Participate in brainstorming sessions and contribute ideas to Thinklogical solutions.
  • Test and improve design of user interface and web server.
  • Document and maintain software functionality.
  • Develop software verification plans.
  • Manage the software development lifecycle.
  • Troubleshoot, debug and upgrade existing systems.
  • Work closely with cross-functional departments and customers.
  • Front-End Web design optimized for OA&M of Thinklogical solutions.
  • Develop features to enhance user experience.
  • Back-End Architect and design server-side modules.
  • Analyze / support current codebase for enhanced performance.



Required Skills and Experience of our Full Stack Software Engineer:

  • Bachelor’s degree in computer science / engineering or equivalent experience.
  • Programming and scripting Languages: JavaScript, CoffeeScript, jQuery, nodejs.
  • Working knowledge of HTML / DOM / CSS.
  • EmbeddedLinux systems (Debian, CentOS, RedHat).
  • Understanding of network / OS security protocols.
  • Understanding of TCP/IP and WebSockets.
  • Understanding of SNMP, Active Directory, LDAP, FIPS, OpenSSL is a plus.
  • Written / verbal communication and interpersonal skills.
