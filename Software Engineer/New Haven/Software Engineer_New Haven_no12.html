About Acuity Brands:

Acuity Brands is the North American market leader and one of the world's leading providers of innovative lighting systems. Our LED lights are computing, sensing, network connected platforms, and everywhere there are people, there are lights. We are perfectly positioned to enable the world of IoT, and we have plenty of awesome projects to work on! We are driving the industry's transformation to smart lighting; redefining how people live, work and play.

About the role:

We're seeking talented and enthusiastic Full-Stack nLight Developers whose primary responsibility is software architecture and development for nLight systems. Additional responsibilities include participation in project time/task estimation, code and architecture reviews, providing support for junior developers, documentation of system architecture and a supporting role for all phases of the development life-cycle (project definition, process mapping, architecture, coding, acceptance testing, installation, turnover to support, etc.).

What you will do:

  • Understand user requirements and translate the requirements to nLight web applications
  • Design, develop, test, and maintain the components of our platform
  • Interact with project managers, UI designers, software engineers, and other applications developers
  • Actively participate in systems architecture to create new innovative solutions
  • Utilize Agile/Scrum methodologies for software development

Required skills:

  • Bachelor’s Degree in Computer Science, MIS or equivalent.
  • 3+ years of proven experience with Full-Stack Software Development
  • Experience using modern frameworks and tools like .NET and Node.JS
  • Proficient C# skills with extensive knowledge of debugging practices
  • Proficient front-end web development with JavaScript, HTML5, and CSS3
  • General knowledge of software engineering practices
  • Basic understanding of databases (relational or non-relational)
  • Basic understanding of user authentication and authorization
  • Understanding of Agile software development
  • Excellent verbal and written communication skills

Preferred SKills

  • Experience with cloud or mobile services (specifically Microsoft Azure)
  • Expertise in Azure RDBMS and other modern data stores (e.g. SQL Server, Table Storage, Blob Storage, Redis Cache)
  • Proficiency in using Microsoft Visual Studio Team Services/Azure DevOps
  • Typescript
