The Opportunity:
As a member of the Quality Assurance team, the QA Automation Engineer is responsible for writing code to automatically test various areas of Axos' core banking functionality, including Web UI, SQL databases, REST APIs, and mobile devices.
The Role:
  • Review, interpret, and apply business rules and logic to ensure a high level of software quality, consistency, and repeatability
  • Collaborate closely with project, product, development, and production support teams to meet agreed-upon deliverables
  • Work as part of a dedicated automation team developing new automation code during each Agile sprint
  • Develop and implement automated tests within our current framework and infrastructure
  • Maintain existing automation suites and troubleshoot failures as needed
  • Review new and existing user stories to ensure good understanding of Axos' core banking functionality
  • Document and report on issues found to the quality assurance, development, and product teams
  • Assist other Axos QA teams with overflow automation items as needed
Are You A Fit? :
  • 2+ years’ in a quality assurance role with automation responsibilities
  • 1+ year's creating SQL queries in Oracle and/or MS SQL
  • 2+ years’ automating use or test cases
  • 2+ years’ implementing automation using the Selenium Web Driver
  • 2+ years’ implementing automation testing REST APIs
  • 1+ year's creating automation tests for mobile devices (Android and iOS)
  • Strong verbal and written communication skills
  • Good understanding of the Software Development Life Cycle
  • Strong analytical and problem-solving skills
  • Proven ability to automate testing of Web UI, SQL databases, REST APIs, and Mobile devices
  • Able to use independent judgment to plan, prioritize and execute on a diversified workload
  • Work effectively in a fast-paced environment with changing priorities
  • Bachelor's degree in STEM or related field
Preferred:
  • Financial services
