Summary

The UI Web Developer works with the E-commerce team to develop code modules to be used in website optimization, email templates, blogs, and landing pages. The Developer also designs, develops and produces web components using HTML5 / CSS3 / JavaScript / JQuery / AJAX / React / GraphQL / REST / JSON. This role manages and configures A/B testing with Oracle Maxymiser, Google Optimize or similar. The UI Web Developer supports analysis of production environments using Google Analytics and supports SEO in addition to being assigned other special projects related to web development.
