Full Job Description

We are an EHR and practice management software company looking for an QA Automation Engineer to join our growing business. This position requires exceptional drive and forward-thinking, a high level of collaboration and multitasking capabilities, and the flexibility to grow with additional responsibilities and respond quickly to changing priorities.

Job Duties

Write and maintain automation scripts using an automation framework built on Selenium WebDriver for execution of scripts on desktop browsers and mobile devices using Appium mobile framework.

Develop and maintain an automation framework based on WebdriverIO and Node.js to support the development of automation scripts.

Develop and maintain factories that generate test data used with the execution of automated scripts.

Work effectively in both cloud-based or on-premise software application environments.

Work with product development, product owners, and UX designers to implement user stories for new cloud offering using modern web technology.

Review software requirements and designs and develop test scenarios to be executed for user stories during sprints.

Uncover potential issues that a user might encounter when running the software, assist in developing solutions for the issues before the software is released.

Record and document test results and expected results within JIRA.

Participate in an Agile development lifecycle, including daily scrums, sprint planning, sprint review, etc.

Ensure solutions are aligned with business and IT strategies and comply with the organization's standards.

Able to move between priorities in a fast-paced environment and deliver high quality results.
