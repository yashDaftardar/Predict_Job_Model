Job Description


Performs basic design, development, testing, documentation, and analysis of various software applications. Performs well-defined application development tasks of a low to moderate degree of complexity, focusing on basic product enhancements and testing solutions for identified issues.


Essential Functions

  • Participate in the development lifecycle with a focus on developing new features and functionalities for assigned software applications
  • In conjunction with the UX Design team, estimate, develop and implement user stories
  • Assess issues and follow established protocol for problem resolution
  • Self-motivated
  • Ability to work in a collaborative environment
  • Provides other ad hoc support and duties as assigned


Preferred Skills

  • Knowledge of at least one of the following: .Net / Java / Javascript
