The candidate will install new software on device under test (DUT). Run automated test cases and manual test cases and report test status in a written document. Create new test cases as needed. Create experiments to ensure tests work as designed correctly. Regress bugs after fixes have been released. Evaluate Android device software with focus on enterprise device management by executing (and modifying as needed) automated and interactive test cases. Apply sound technical judgement in an environment of evolving design specifications by assessing product behavior beyond what is formally documented. Communicate design issues and evaluation results via e-mail, bug reports, and status reports.
Troubleshoot and isolate faults in the assigned technical area by designing, documenting, and executing ad hoc experiments, as well as running and/or creating diagnostic utilities and scripts.


Responsibilities

  • Deploy pre-release images on devices and execute test cases
  • Work with multiple enterprise management test environments used in tandem with device image testing
  • Document results in bug reports, status reports, e-mail
  • Read, understand, and (possibly) modify scripts used for testing and diagnostics
  • Detect design issues and conduct experiments to further isolate and characterize faults
  • Improve validation process by suggestions for test cases, automation, and process methods

Qualifications

  • Three years of Android development experience
  • Experience shipping a phone running Android
  • Experience Managing and supporting mobile devices (Windows, Android, iOS) in an enterprise environment.
  • Experienced with Android architecture, diagnostics and debugging of devices and software
  • Familiar with multiple enterprise management systems a plus (eg. Active directory, Intune, SCCM, any Linux enterprise management, or MDM / EMM)
  • Working knowledge of a scripting or compiled languages (C#, PowerShell, Linux bash etc)
  • Experience in SW/HW product release processes
  • Effective written communication
  • Methodical and organized
