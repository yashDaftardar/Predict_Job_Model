Job Details

What you’ll be doing…

We are building the next generation infrastructure to support cross-platform development. Our mission is to reduce the effort needed for our development teams to release their features to customers. We will be on the leading edge of technologies and will be partnering with many teams across the development organization.


Some of the things you'll be doing include…

· Working in a highly collaborative environment.

· Work across teams to make sure we are building great experiences for customers while using new infrastructure and technologies.

· Work across the code stack of the Tableau Desktop code base in C++ with some exposure to modules built with TypeScript and end-to-end automation code using Java

· Refactor the code in order to accelerate feature development

· Perform performance analysis using code profilers in order to contribute to a fast user experience

· Work with other teams to improve, refine our browser-based experience to make it truly delightful

· Work with developers across the organization to improve the developer experience

Who you are…

· Experienced. You have multiple years of solid development skills with large scale codebases using object-oriented languages such as C++

· Highly Technical. You are a seasoned developer and fluent in commonly used design patterns. You can constructively review the work of others to ensure high quality and adherence to best practices. You have a solid understanding of architectures relevant to multi-tier applications.

· Passionate. You love to write maintainable and understandable quality code, and have a passion for solving complex problems and enjoy building infrastructure that others will leverage. You write tests for your code as a way to maintain high quality. You understand what it takes to write software that is used by thousands, even millions, of people. You love products that "just work" - that are robust, scalable and perform well.

· Curious. You love to get to the root of the problem. You debug across code bases and languages to wherever the problem takes you.

· A True Team Player. You enjoy collaborating, learning from and teaching others so we can all become better developers. You assume the best in others, and actively do your part to create a positive work environment.

· Educated. You have a degree in Computer Science, Mathematics, Statistics, Engineering, or a related field.

· You are a Recruiter! Tableau hires company builders and in this role you will be asked to be on the constant lookout for the best talent to bring onboard to help us continue to build one of the best companies in the world!
