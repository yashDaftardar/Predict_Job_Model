Position Overview:

The Midlevel Database Programmer uses a strong background in design and administration of Microsoft SQL Server, and extensive experience with writing T-SQL stored procedures, triggers, and DTS packages to perform database administration duties.

Responsibilities:

  • Administer, maintain, develop and implement policies and procedures for ensuring the security and integrity of the company’s SQL Server database.
  • Work with daily data feeds, cleansing and cleanup and data loads.
  • Create and run queries of stored procedures to handle database operations and tasks.
  • Schedule database jobs.
  • Install and run backups and restores on existing databases.
  • Responsible for documenting new and existing processes.
  • Ensure that database improvements are successfully implemented and monitored to increase efficiency.
  • Requires frequent interface with team members and other programming teams
  • Perform other duties and/or special projects as assigned.


Skill & Experience Requirements:

  • Bachelor’s degree in Computer Science, Computer Engineering, Information Technology or related field strongly preferred. Proven industry experience will be considered in lieu of the degree requirement.
  • Minimum of 3-4 years of demonstrated work experience in database management and operation strongly preferred or at manager’s discretion.
  • Networking knowledge and experience, SQL 2005/08 and SQL CE.
  • Windows NT; Windows 2000 Servers, VBScript, and XML.
  • Experience in designing and maintaining OLTP databases.
  • Troubleshoot database errors and slow database performance.
  • Must have knowledge with replication and topologies, specifically merge replication.
  • Thorough understanding and ability to resolve issues relating to database performance, capacity and monitoring.
  • Understanding of relational database theory and management systems required.
  • Detailed experience creating database Stored Procedures and Triggers.
  • Exceptional customer service and problem solving skills required.
  • Excellent quantitative and analytical skills.
  • Capacity to quickly learn, understand, and apply complex data structures to your work.
  • Ability to resolve issues in a timely, accurate and thorough manner.
  • Team player with the ability to work in a fast-paced environment.
  • Ability to meet deadlines.
  • Creative thinker with the ability to “think outside of the box”.
