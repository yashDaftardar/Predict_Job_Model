Position Summary:


AutoNation is looking for a seasoned Hyperion Engineer to assist with development and support of AutoNation’s suite of Hyperion applications; which include Financial Consolidation & Close Cloud Services (FCCS), Enterprise Planning & Budgeting Cloud Services (EPBCS) and Oracle Analytics Cloud (OAC).

The Engineer will work onsite in our beautiful building in Fort Lauderdale and will work closely with the business to ensure our applications continue to meet their needs, and lead the effort when changes are necessary.

Job Responsibilities:

  • Take part in the support of the FCCS, EPBCS and OAC applications in our new cloud environment
  • Analyze problems as assigned and provide solutions via technical specifications to address business needs
  • Take ownership of data interfaces, automation jobs and other Hyperion processes
  • Produce consistent and accurate technical and end-user documentation of assigned applications
  • Understand how assigned applications/systems link with other programs/processes along with the overall system architecture and provide recommendations on how to improve existing/proposed processes
  • Adhere to company change management practices
  • Provide end user support on Hyperion Essbase and SmartView
  • Supports the monthly application maintenance process
  • Participates in the feasibility analysis for new systems/applications
  • May serve as a technical lead on projects, leading design efforts
  • Monitors the operating efficiency of existing application systems and provides (or arranges for) proactive maintenance
  • Reviews, analyzes, and modifies applications, including coding, testing, configuring, and debugging for new Hyperion applications. Maintains records to document the system and revisions
  • Provides guidance and/or arranges resources for technical support to address production issues
  • Participates in training, as appropriate, on new or revised systems/ applications
  • Tests own work and contributes to the development of test plans and participates in post-implementation reviews
  • Provides second or third level operational support for production systems when necessary. Writes documentation that describes installation and operating procedures
  • Provides guidance to junior team members
  • Communicates applications changes to Technology team members
  • Estimates resource requirements and scope for incoming tasks as needed

Technical Expertise:

  • Bachelor’s Degree in Computer Science, Engineering, similar discipline or extensive Hyperion application admin experience
  • Minimum 5 – 7 years of experience with Hyperion applications
  • Demonstrates an advanced understanding of the following Oracle EPM products:
    • Oracle Data Integrator (ODI) development
    • Data Relationship Management (DRM)
    • Financial Management (HFM)/Financial Consolidation and Close Cloud Services (FCCS)
    • Hyperion Planning/Enterprise Planning and Budgeting Cloud Services (EPBCS)
    • Essbase/Oracle Analytics Cloud (OAC)
    • Oracle Cloud Rest API
    • Oracle Data Visualization
  • Demonstrates an understanding of the following:
    • SmartView VBA
    • Visual Studio VSTO - Excel C#
    • SQL
    • PowerShell
    • CLI
    • MaxL
  • Acts as a resource to others on the existing applications supporting a business area. Serves as a key resource to ensure the system is performing up to requirements
  • Works independently and provides guidance within technical area, applying in-depth knowledge of multiple technologies, as appropriate
  • Understands architectural issues, and factors them into decisions and recommendations
  • Provides leadership in their area(s) of technical specialization

AutoNation is an equal opportunity employer and a drug-free workplace.
