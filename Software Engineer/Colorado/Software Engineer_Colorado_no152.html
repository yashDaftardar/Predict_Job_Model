Job Overview:

Qualcomm is a company of inventors that unlocked 5G ushering in an age of rapid acceleration in connectivity and new possibilities that will transform industries, create jobs, and enrich lives. But this is just the beginning. It takes inventive minds with diverse skills, backgrounds, and cultures to transform 5Gs potential into world-changing technologies and products. This is the Invention Age - and this is where you come in.

The TBS program is responsible for building prototype base stations for the latest radio technologies (including 5G-NR/4G-LTE/C-V2X). Product teams are comprised of Software, Hardware, Systems and Test Engineers, providing custom designed, cutting edge, end-to-end digital/RF solutions.

Software positions in this program involve a wide range of efforts including design, development, and support of end-to-end radio protocol stacks (e.g. MAC, RRC) in 5G network infrastructure (e.g. gNB and 5GC). High performance software is written in C++ and is targeted for multi-core, real-time servers running multi-threaded applications. Fast paced development is done in a Linux environment and supported by support by Continuous Integration processes and systems.

Teams in this program are distributed across Boulder, San Diego, New Jersey and India offices.

Minimum Qualifications

  • Bachelor's degree in Engineering, Information Systems, Computer Science, or related field.
  • 1+ years demonstrated experience with C++ Programming Language

Preferred Qualifications

  • Bachelor's degree in Computer Science/Engineering, Electrical Engineering or related fields.
  • 1+ years demonstrated experience with C++ programming language
  • Experience in Object Oriented software design/development.
  • Familiarity with developing/working in Linux environment.
  • Knowledge of multi-threading programming and associated OS concepts.
  • Familiarity with network protocols (TCP/UDP/IP/HTTP).
  • Familiarity with industry standard software development and debugging tools (IDEs, build systems, git, gdb).
  • Strong communication skills.
