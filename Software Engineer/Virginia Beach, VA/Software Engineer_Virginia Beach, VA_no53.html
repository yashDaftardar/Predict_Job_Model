ADP has created a new IT Asset Management Team (ITAM)

and we need a Lead Full Stack Developer to help!


ITAM is responsible for unifying IT asset life cycle management systems for hardware, software, EUC and cloud assets that embodies increased transparency. Tactical and strategic decisions now become data driven so business units have a clear understanding of the costs that go into designing and running their products. This will provide reduced lead time for acquisition and disposition of assets. In turn, this will result in cost savings and provide accurate compliance data.

ITAM team is looking for a Full Stack Developer to join their newly created and dynamic team to produce highly scalable software solutions. You'll be part of a cross-functional team that's responsible for the full software development life cycle, from conception to deployment.

As a Full Stack Developer, you should be comfortable around both front-end and back-end coding languages (ReactJS, JavaScript, Java, etc.), development of frameworks and third-party libraries. You should also be a team player with a knack for visual design and utility.

If you're also familiar with Agile methodologies, we'd like to meet you!

Responsibilities:

  • Deliver a complete front-end application - from the creation of UX prototypes from ideas, mockups from wire frames until the functional code implementation
  • Ensure high application quality and performance
  • API development using Java and/or NodeJS
  • Create Unit Tests and participate in code reviews
  • Create software documentation and update existing documents
  • Participate actively/follow Agile / Scrum development methodology

Skills and Qualifications:

  • Deep knowledge with ReactJS
  • Knowledge of multiple front-end languages and libraries (e.g. HTML5 / CSS3 / JavaScript ES6)
  • Knowledge of multiple back-end languages (e.g. Java, Python) and JavaScript frameworks (e.g. Angular, React, Node.js)
  • Experienced with responsive layouts
  • Interfacing with Restful APIs
  • Experience with automation scripting
  • Experience with working environment with GIT, Linux
  • Degree desired, or equivalent work experience
