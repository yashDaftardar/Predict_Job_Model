Job Description

CACI is seeking a strong DevOps / Cloud Engineer that is able to work in a DevOps environment but help a team transition to DevOps practices.

What You’ll Get to Do:

  • Focus on automating more aspects of the system instantiation, patching, and maintenance activities.
  • Help lead innovative system design improvements to enable increasing cloud-native capabilities in software solutions.
  • Communicate the benefits of following these DevSecOps practices in support of information system sustainment efforts.
  • Lead development efforts in CloudFormation, PowerShell, Python, PERL, and related technologies in support of Infrastructure as Code (IaC).
  • Communicate best practices and design patterns for automating infrastructure as code (CloudFormation, PowerShell, etc.), automated testing, and improving CI/CD pipeline efforts so that performance is maximized, and benefits realized.
  • Work closely with other team members and team leads to ensure that all aspects of solutions are designed, developed, and integrated to align with an overall target architecture.
  • Work with Java Stack: Java, Gradle, JSP, MyBATIS, Angular, ext.js, Spring, along with SAP ASE, SAP ASA, & SQL Server databases.
  • Also support Delphi Stack as needed: Delphi RAD Studio, Smart Reports, ODBC, with additional libraries.
  • Improve cloud-based network security, performance, and reliability via standard DoD design practices.

You’ll Bring These Qualifications:

  • Should be a self-starter and familiar with infrastructure as Code, agile methodologies (e.g., scrum), and able to help the team transition a system in operation from a Navy Data Center to an AWS Cloud environment.
  • Demonstrate to the team by example how DevOps (DevSecOps) methodology works in practice.
  • Bachelor’s degree in Computer Science (or engineering, or close technical equivalent)
  • 1+ years’ Demonstrated Cloud Experience (AWS or Azure preferred)
  • 3-5 years’ Software Development and/or IT Operations Experience involving scripting, preferably in a DevOps environment
  • Strong communication skills
  • SECRET clearance / Ability to get clearance
  • CompTIA Security+ certification
  • 2-5 years’ Agile Software Development Experience
