Full-Time Front End/Backend Web Developer

Studio Center, one of the busiest content creation studios in the country, is looking for an experienced Front End/Backend Website Developer who is quick on his/her feet and ready to work in a fast paced, creative environment. Our Digital Division is responsible for web design and development, internet marketing, mobile websites, e-commerce, email marketing and collateral design.

We offer a competitive salary PLUS commission on every completed website.

Interested candidates should is able to code CSS, HTML5, JavaScript, Ajax, PHP, MySQL (and variants) and Linux, responsive design and any other current coding languages inside and out; server management via Plesk web panel and shell; implement name server and DNS changes; troubleshoot email transport problems; implement SEO best practices; write clean markup and avoid unnecessary tags; provide as close as possible output between browsers from MSIE 10 and up, Chrome, Firefox, and Safari. Candidates should have knowledge of Content Management Systems like Concrete5, Drupal, Joomla, WordPress and Magento is essential.

Location: Virginia Beach or Richmond, VA

Interested candidates should send their resumes to chriswilson@studiocenter.com
