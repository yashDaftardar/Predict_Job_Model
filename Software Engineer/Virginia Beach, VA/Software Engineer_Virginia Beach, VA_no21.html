Accelerate the mission of Old Dominion University through High Performance Computing and Devops. Work with a dynamic, strategy oriented team of individuals focused on making a difference. Develop systems and applications that support researcher’s needs through high speed parallel computing, cloud environments and container deployments. Combine technical skills with big picture thinking to make an impact in High Performance Computing and AI solutions.

Type of Recruitment Knowledge, skills and abilities

Extensive knowledge of scripting and programming languages such as bash, python and c/c++. Thorough knowledge of the application development lifecycle including build, release and deployment. Some knowledge of public and private cloud environments and associated API’s. Some knowledge of Windows and Linux hardware and software support and appropriate system practices. Considerable knowledge of principles, practices and procedures for system and application administration. Strong problem solving and analytical skills. Ability to work in a team. Ability to plan work and set priorities.

Special licenses, registration or certification

None

Education or training

None

Level and type of experience

Considerable experience in application development using scripting and/or programming languages. Considerable experience in the support of users and applications using the Linux operating systems. Considerable experience in the automation and orchestration of enterprise environments. Some experience in the management of containerized and virtual environments. Some experience working in high performance computing
Some experience supporting environments for high performance, high throughput and massively parallel computation. Some experience building and deploying containerized, GPU-enabled applications in Docker, Singularity and Kubernetes. Some experience with AI and Machine Learning toolsets and Data Analytics frameworks.

Preferred knowledge, skills, abilities, education, experience, licensure, certification

Preferred experience using public/private clouds such as AWS, Azure and Openstack. Some experience administering and enterprise server environment including Windows server, Redhat Linux and/or CentOS servers. Some experience using virtualization technologies such as VMware or Openstack. Some experience with HPC job scheduling systems and cluster management tools. Some knowledge of HPC software and networking technologies. Bachelor’s degree in Computer Sciences or related discipline or equivalent combination of education and job related experience.
