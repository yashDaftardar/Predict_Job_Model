Full Job Description

WR is seeking a lead Software Engineer to support software development and employment for Department of the Navy operational initiatives. The selected candidate will consult with clients to gather information about program needs, objectives, functions, features, and input and output requirements. This position will also formulate and design software-based systems using scientific analysis and mathematical models to predict and measure outcome.


Position responsibilities will include, but are not limited to the following:


  • Writes and maintains technical specifications.
  • Consults with hardware engineers and engineering staff to evaluate interface between hardware and software.
  • Designs program interfaces and ensure interoperability with other systems.
  • Converts designs and specifications into computer code, as well as analyze code for Software development best practices make appropriate revisions to the program.

Candidates must possess the following knowledge, skills, and experience:


  • B.S. in Computer/Software Engineering OR Computer Science.
  • Six(6)+ years of experience of demonstrated software development using microcontrollers or FPGAs or embedded computers.
  • Proficient with a scripting language ad regular expressions (e.g. Python, PHP, Bash, Perl).
  • Experience with hardware preferred, but not required.
  • Proficient in Embedded C.
  • Three (3)+ years of experience of demonstrated software development using one or more of these languages under the Unix, Windows, Linux, and/or MS-DOS.
  • Object-oriented program language experience a plus (C#, C++, Java, Perl, Ruby).
  • Relational DB programming experience a plus.

Active Secret Security Clearance required.
