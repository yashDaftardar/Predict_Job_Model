DESCRIPTION:

Seeking a mid-level Java Developer who will join a team focused on development of moving a Navy application from client/server to the web. The application has been rewritten in Java, Angular 5 and uses Weblogic for an application server with Oracle database. The new team member must have a good understanding of Object-oriented languages, some experience with C# or Java. Experience with developing RESTful web services, consuming and returning JSON, in MVC environment a plus.

Requirements:

  • Bachelor’s Degree in Computer Science, Engineering, Business Information Systems, or equivalent, plus 1+ years software development experience, or Associates’ Degree in the same plus 3+ years software development experience.
  • 3+ years ASP.NET Development with Java, SQL, Javascript
  • 3+ years’ experience SQL development
  • 3+ years’ experience UI development
  • Working experience in interfacing with SOAP / REST APIs / Web services.
  • Experience with version control tools – such as Git as well as some experience with both Windows and Linux systems
  • Experience with Angular and Security+ certification is highly desired
  • US Citizenship and be willing/able to obtain a DoD Secret Clearance, as a condition of employment.

Preferred Skills:

  • 5+ years’ software development experience
  • 5+ years’ full-stack development
  • 5+ years ASP.NET Development with Java, SQL, Javascript, jQuery, and web services, with experience up through at least .NET 4.0
  • Experience with Oracle 11g development is desired but not required
  • Ability to work well with others and collaborate on complex technical matters
  • Active DoD Secret Security Clearance a plus but not required
  • Any experience with build environments is a plus
  • Any support of backend work a plus.
  • Experience or exposure to Agile development is a plus
