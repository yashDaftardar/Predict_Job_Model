FUNCTION OVERVIEW:

The Front-end Software Tools Engineer is responsible for developing front-end software code for internal software tools for appliance electronics.


JOB RESPONSIBILITIES:

  • Understand customer requirements and translate into software architecture and development requirements
  • Design, development and deployment of appliance customization software
  • Participate in software design reviews, conduct peer code reviews, and provide input and feedback to other members of the development team
  • Document software design and implementation
  • Release software for validation and production
  • Contribute across the product development from technical design / architecture, through implementation, to production support and maintenance
  • Working with cross functional teams applying Agile and attending daily stand-up meetings
  • Interface with external teams and internal engineers to define and implement new features
  • Design of debug, test, data acquisition, parameters search and monitoring software


MINIMUM REQUIREMENTS:

  • Bachelor's Degree in Computer Science
  • 2+ years Web Development fundamentals; HTML5, Cascading Style Sheets (CSS) and JavaScript (JS)


PREFERRED REQUIREMENTS:

  • Collaborative attitude and strong desire to succeed as part of the team
  • Self-motivated and a strong passion for learning
  • React Library
  • Typescript/JavaScript
  • REST API and/or GraphQL
  • Single Page Web Applications design
