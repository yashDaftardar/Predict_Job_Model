Full Job Description

Eisen is inviting applications from experienced software engineers to join our new product development team at Lansing, Michigan. You will be involved in the design and manufacture of portable medical devices. We are inviting individuals with expertise in some of these technologies:

REQUIRED:
  • Event driven and thread-based programming
  • Systems integration and device communication using RS232, IP, USB, or UART
  • GUI programming for deployment on embedded devices using real time operating systems
  • Development environment using Visual Studio or QT for deployment on embedded OS such as QNX, Linux, IoT, WinCE
  • Developing application architecture, Linux integration, and programming with C++

PREFERRED BUT NOT REQUIRED:
  • Solid understanding of DLL and CMAKE configurations
  • Programming device controllers using ARM based compilers

EDUCATION AND WORK REQUIREMENTS:
  • Higher Computer Software Engineering degree or equivalent experience with focus in product architecture, systems integration, or event based GUI product development for lean operating systems.
  • Work on-site in Lansing, Michigan

Include a resume and cover letter with your application. Applicants must also submit the Employment and Internship Questionnaire from Eisen careers page at https://eisenusa.com/es/careers/
