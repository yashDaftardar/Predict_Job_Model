Steer-by-Wire technology is an exciting development in the automotive industry, gathering considerable interest from many areas. It provides significant benefits for both traditional and autonomously-driven vehicles, being a key enabler for the Nexteer Quiet Wheel technology and seamless transitions between autonomous and human drivers.

The Nexteer Automotive Steer-by-Wire engineering team continues to grow and evolve as we are working with multiple customers in support of their development of demonstration vehicles and design requirements. At the same time the team is developing and refining our own production-intent designs for offering on anticipated upcoming requests for new business quotes.

The Embedded System Software Engineer is responsible for the overall Steer-by-Wire software development including understanding technical requirements from the customer or internal team, system software implementation, verification of software function and work with cross functional teams. The successful candidate will have related experience software implementing Electric Power Steering highly safety critical embedded system.

This position provides the opportunity to develop next generation highly safety critical embedded technology which could result in patents. The candidate will work with a highly motivated Steer-by-Wire team for design and implement lowest cost safety critical software solution.

In this role, the Embedded System Software Engineer will work with Autosar Software Component Implementation, function design, Matlab Autocode Implementation & Software Integration, Serial Communication Implementation, Software Integration, trouble-shoot software issue, Tool development, Embedded hardware and Microsoft Office.
