Description

SAIC is seeking a Remote Intermediate .Net Developer to join our team in support of our customer at the Human Resource Command in Fort Knox, KY, while working from home. This intermediate developer will implement, enhance, and maintain client server and Web applications; analyze/resolve software-related issues/problems; maintain development software for the development team. Work with network team to maintain server hosted development tools. Work with Database Administrators on design/development and Network Administrators on network requirements. The developer will write code encompassing all tiers of the software development process with primary focus on front end and user interface design and development. This is a Remote / Virtual / Work from Home position.

Duties & Responsibilities:

  • Write, modify, and debug web applications.
  • Assist other developers in coding and debugging issues.
  • Work with business representatives to define requirements and develop technical specifications.
  • Monitor quality assurance and required up- times of supported systems and their data sources.
  • Maintain versioning and implementation of third party products used in the development process.
  • Participate in activities accordingly to ensure on time implementation deliverables are met including day-to-day planning and coordination with testers, developers, project managers and business representatives.

Qualifications

Required Education and Experience:

  • 2+ years' experience and a Bachelor’s Degree or equivalent in Computer Science, Engineering or Information Technology and zero years’ experience. (4 additional years of related experience will be considered in lieu of a Bachelor’s degree.)
  • Comfortable using Microsoft Visual Studio for Web based development
  • Demonstrate level of competence in object- oriented analysis, design and coding in a .NET server-side environment, specifically utilizing the Microsoft MVC framework.
  • Some experience using a major database (e.g. Oracle, SQL Server, MySQL, etc...)
  • Comfortable with debugging and analyzing software errors
  • Comfortable with object oriented analysis and design
  • Comfortable working with and writing HTML and XML
  • Excellent written and verbal communications skills.
  • US citizen / no dual citizenship.
  • DOD Secret Clearance Required, but an Interim Secret is acceptable to start work and SAIC will process this for the qualified candidate.
  • CompTIA Security Plus Certification is required prior to start date.
