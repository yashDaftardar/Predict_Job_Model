KPMG is currently seeking a Manager in Technology Enablement for our Consulting practice.

  • Develop and coordinate cloud architecture across diverse areas including application development, identity and access management, network and data management, and security
  • Provide high level cost and resource estimates for Infrastructure as a Service (IaaS), Platform as a Service (PaaS) and Software as Service (SaaS) offerings
  • Use standardized process for mapping business needs to available cloud technology options, including assessment of risks and possible mitigations
  • Adopt a wholistic approach that considers operational and support concerns, with a strong preference towards designing for reliability via available tools and approaches (e.g., automation, monitoring)
  • Act as a trusted advisor to clients; ensures the efficiency of cloud requests and manage customer driven, cloud projects; create Cloud documentation of technical implementation plans; represent Cloud Operations to all stakeholders

Qualifications:

  • Minimum of five years of recent and relevant experience
  • Comprehensive knowledge of Infrastructure as a Service (IaaS), Platform as a Service (PaaS) and Software as Service (SaaS) across Microsoft Azure cloud and the Microsoft technology stack in general (Azure, .NET, Power Platform), IT Security, software development, and infrastructure deployment methodologies and industry best practices
  • Proven experience with scalable architectures using Azure App Service, API management, serverless technologies, Kubernetes container orchestration
  • Experience with Big Data solutions on Azure and delivering data pipelines (e.g. Azure Data Lake Service, Power BI, Azure Data Bricks, Azure Synapse and Azure Data Factory)
  • Ability to travel as needed and applicants must be currently authorized to work in the United States without the need for visa sponsorship now or in the future
