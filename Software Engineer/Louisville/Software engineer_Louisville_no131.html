Full Job Description

New American Business Association Inc; a 501c3 nonprofit organization that is using information technology and data-driven processes to empower Louisville community small businesses. We are looking for more to help with our community projects in transforming public data into meaningful information/knowledge for small business communities.

As an intern, you'll gain a deeper understanding of technology, business process, and software development with highly IT experienced industry leaders/experts through our partnership with the Louisville Institute of Technology. You will receive training, mentorship, and professional coaching and software development with real-life software developers, computer engineers, and principal software architects who have worked and successfully developed, implemented, deployed, maintained, and managed small to series of multi-million dollars solution in fortune 100 companies such as Humana and UPS.

We'll trust you to:
Take ownership of projects under the guidance of your mentor
Be a collaborative member of your team
Have a deep understanding of data structures and algorithms
Be an excellent problem solver

Software That Will Learn and Work With:
Azure Cloud Solutions and Tools as below:
1- .Net Core 3.1
2- C#
3- Web API - Backend
4- React.JS - Frontend
5- Azure Cloud with Fully Managed App Service and Azure SQL - Database
6- End to End - Fully integrated CICD on Azure DevTest and Git/GitHub code versioning and management repository

7- Test Automation - Unit Test, API Test, and FrontEnd Test (using Selenium) with integration with CICD pipeline

You'll need to:
Have a deep desire to learn - because we can teach you on the job
Have a preparation and openness to work with others - because teamwork is the keys
Have a goal in your IT Career - because you have to know what you want in your IT Career next
Have programming experience in some programming languages C, C++, Java, Javascript, Python, or ...
Be working toward a BA, BS, MS, or Ph.D. in Computer Science
If this sounds like you, please apply below!
