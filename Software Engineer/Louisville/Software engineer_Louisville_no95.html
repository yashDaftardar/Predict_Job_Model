Description

SAIC is seeking an Intermediate Java developer to join our team support of work taking place for our customer at the Army Human Resources Command in Fort Knox, KY, while working from home. This position will be supporting the U.S. Army PaYS Program. The candidate chosen will be working to write, maintain and enhance web applications as on the Java platform. This will include database SQL work as well as server applications that are also Java based. This is a Remote / Virtual / Work from Home position.

Responsibilities:

  • Provides support in specialized technology areas that can include, but is not limited to, elements such as: Web design, requirements/specifications definition, analysis, assessments, planning, design, development, integration, testing, installation, performance tuning, deployment, or maintenance.
  • Assists with strategic or tactical planning to justify, market, or manage the technology needed for successful business operations.
  • Support user training, as required.
  • Contributes to or develops technical documentation.
  • Contribute to technical presentations.
  • Use applicable methodologies, modeling/estimating techniques, tools, applications, systems, software, or databases to perform assigned tasks.
  • Candidate must compile with the standard and organization requirements relative to specific assignments.

Qualifications

Required Education and Experience:

  • Bachelor’s degree in Computer Science or related field and 2 years’ experience. Four years of additional general experience may be substituted for a Bachelor’s Degree.
  • Ability to interface with clients and understand requirements
  • Ability to work in small, rapid-development teams
  • Experience with Object Oriented programming methods
  • Experience working with web applications programming
  • Knowledge of relational database systems to include basic knowledge of SQL
  • Good foundational knowledge of computer science and algorithms
  • DOD Secret Clearance Required to begin, an interim Secret is acceptable to start work. SAIC will assist with the processing of the security clearance for the qualified candidate.
  • CompTIA Security + Certification is required prior to start date.

Preferred Qualifications (not required):

  • BS in Computer Science or has equivalent work experience
  • Experience with Spring (Security, Core, MVC, WS, Data, Web Flow)
  • Experience with Maven and its lifecycles
  • Experience with Selenium or other testing tools
  • Experience with jQuery, Angular or other modern JS Frameworks
  • Experience with Tomcat, JBoss, WebSphere or any other Java servers
  • Understanding of SSL setups, SOAP/WSDL and RESTful services
