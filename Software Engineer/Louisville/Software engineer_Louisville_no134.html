Description:
A Louisville-based engineering firm is hiring a Software Engineer to assist for new and existing product lines. The ideal candidate will be responsible for developing real-time embedded automotive applications. They will also be responsible for designing and implementing testable and scalable code. The application will be for sound systems that are installed in motorcycles, ATVs, utility vehicles and snowmobiles.

Responsibilities:
  • Develop quality software and web applications
  • Analyze and maintain existing software applications
  • Contribute to electrical hardware design
  • Discover and fix programming bugs Additional Skills & Qualifications:
  • BS Degree in Engineering or other technical discipline (Electrical Engineering, Computer Engineering, Computer Science)
  • 1-5 years development experience with C and C++ programming languages
  • DSP (digital signal processing) and/or audio engineering experience
