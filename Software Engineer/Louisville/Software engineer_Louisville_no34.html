Responsibilities

This position will be assisting in the development of an Autism Claim Audit system to help ensure that TRICARE autism claims are being submitted and processed properly.

The Senior Software Engineer standardizes the quality assurance procedure for software. Oversees testing and debugging and develops fixes. Researches complaints and makes necessary adjustments and/or recommendations to resolve complex software related issues. Begins to influence departmentâ€™s strategy. Makes decisions on moderately complex to complex issues regarding technical approach for project components, and work is performed without direction. Exercises considerable latitude in determining objectives and approaches to assignments.

KEY ACCOUNTABILITIES

Planning, Analysis and Design

  • Understand business purpose of software solution requests
  • Contribute to major feature planning
  • Contribute to process of estimating costs for proposed solutions
  • Contribute to analysis of how to best meet purpose and features resulting in logical requirements documentation (can include build, buy or hybrid solutions)
  • Help design solutions to meet objectives (data structures, system processes, services, user interfaces, etc.)
  • Guide more junior level team members as directed
  • Work with Lead Software Engineer and architecture team to explore new tools, techniques, etc. to improve solution delivery

Development, Integration, Testing and Implementation

  • Write, update, test and maintain in-house engineered software solutions utilizing Humana Military approved tools
  • Write, update, test and maintain software to integrate purchased software solutions utilizing Humana Military approved tools
  • Configure purchased software to meet business and IT requirements
  • Guide junior team members with their software engineering tasks as directed
  • Assist in creating comprehensive test cases for functional and regression testing plans
  • Assist testing teams with automating test cases
  • Assist end users with UAT as needed
  • Assist in creating detailed deployment plans to include rollback
  • Prepare system documentation as directed
  • Provide on call support as scheduled (a few times per year)
  • Assist with Peer Code Reviews


Required Qualifications

  • Our Department of Defense contract requires U.S. Citizenship
  • Successfully receive interim approval for government security clearance (eQIP - Electronic Questionnaire for Investigation Processing)
  • Bachelor's degree (technical area desired), or equivalent technical training
  • Five plus years of Information Technology experience, working with web applications and development
  • Web Development experience with C#, JavaScript, JSON, HTML, CSS and SQL
  • Two years of current experience using Visual Studio, .NET Core, MVC
  • Database development experience with Microsoft SQL Server, SQL Stored Procedures and SSIS
  • Experience with configuring and integrating chat and secure messaging software
  • Experience in designing and developing efficient and high availability Web Services/Rest APIs
  • Performance testing, profiling and tuning
  • Ability to perform analysis and design, requirements gathering, documentation, programming, testing, system installation and/or conversion
  • Project management experience
  • Good written and verbal communication skills
  • Communicate status and issues in an understandable, capable way with both peers and non-technical business partners
  • Experience with Microsoft Office tools
  • Ability to work well in a team development setting
  • Must be passionate about contributing to an organization focused on continuously improving consumer experiences

Preferred Qualifications

  • Master's Degree
  • Familiarity with Azure DevOps
  • Healthcare experience
  • Full Stack development
  • Services architecture
  • Cloud Services experience
