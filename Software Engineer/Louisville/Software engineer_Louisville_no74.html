Full Job Description

The Senior Cloud Engineer – Cloud Networking enables cloud to provide an optimized, well architected, automated, and on-demand network environments. The Senior Cloud Engineer – Cloud Networking will excel in driving the automation of executing new designs, testing and deployment, and driving operational frameworks to steady state.

Responsibilities

Works with cloud, infrastructure, solutions, and other engineering teams to drive cloud network requirements for products from design execution to production release

Interacts with product teams, service delivery leaders, and engineering to analyze and provide technical support in resolving cloud infrastructure related problems

Builds Docs as Code (DaC) to ensure architecture standards and documentation stay current with the infrastructure that they reflect

Works with architects and leaders to build, test, and validate new cloud network solutions, which solve problems of diverse scope and complexity


Required Qualifications

  • 3+ years of technical and network focused experience with high-performing IT organization
  • 2+ years of experience with cloud networking in Amazon Web Services, Azure, or Google Cloud
  • 1+ years of experience leveraging agile methodologies and DevOps practices

Preferred Qualifications

  • Associate or Professional Level cloud or network certifications (Azure preferred)
  • Experience deploying cloud networking (VNet or VPC) solutions in a hybrid infrastructure
  • Experience with cloud based connectivity options, routing, traffic engineering, and support
  • Experience with Network Virtual Appliances (NVAs) and automated deployment
  • General understanding and application of networking protocols and features (Specifically BGP)
  • General understanding of on-premises and cloud based DNS design, behavior, and support
  • Experience leveraging Infrastructure as Code (IaC) via CI/CD pipelines in a highly automated environment
  • Understanding of managed Kubernetes platform networking (specifically Azure AKS)
  • Experience with Linux system administration
  • Experience with Ansible
  • Experience with Terraform
  • Experience with Microsoft Technology Stack
