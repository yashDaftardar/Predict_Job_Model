Position Description

The Systems Programmer IV is a server administrator for the University Libraries. The Systems Programmer will administer the back-end application servers such as database, backup, and media servers. In conjunction with the other administrators in the Office of Libraries Technology (OLT), this position will manage the Libraries' network, including Active Directory (AD) accounts and security; manage application servers hosted on the VM platform, including a backup server, servers for PC management, database servers, media servers, and Web servers that are Windows or LINUX based; and coordinate on server hardware and software acquisition, installation, maintenance, and disaster recovery. This position will assist the Head of OLT in the Libraries' technology planning and acquisitions. The Systems Programmer will also assist with client computer hardware and software installations, upgrades, and troubleshooting problems.
