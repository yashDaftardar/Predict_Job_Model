Job Description

Business Integration Analyst I (BIA-I) works with the Software Development group and Quality Assurance (QA) team to facilitate deployment of software for project implementations as well as new product enhancements. This includes scripting the set-up, installer validation, and documentation. The main goals for this position are to assist team members in delivering a fully configured and solid working system to the Quality Assurance team for testing.


General Job Duties

  • Work with the project analyst during the FSD phase to run the system in a demo mode if needed.
  • Work with the project analyst during the FSD phase to document configuration required for desired operations.
  • Create/maintain standard scripts for system configuration.
  • Document and script out planned configurations for an implementation per the FSD in preparation for project’s Dev environment configuration.
  • Work with the analyst to define screen management changes needed for the project, or saved queries that need to be added.
  • Once the Dev environment is setup, configure the needed application configuration for a project’s modules.
  • Gather warehouse configuration from the project manager and set up customer’s information on products, locations, zones, etc. within the Dev environment.
  • Perform an initial “happy path” test to validate the installer, make sure modules are operating as desired with system configuration and customer data.
  • Track issues found in testing in Azure DevOps with work items following existing processes.
  • Promote release to the QA team for in-house testing.
  • Support the QA group as needed in their efforts to develop test plans and perform testing.
  • Support the Operations team in testing and implementation as needed.
  • Close communication with the Business Systems Analyst Manager on both good news and bad news.


Superior Performance Objectives

  • Close collaboration with the team to get a project up and running.
  • Strong attention to detail, provide complete releases (eliminate missing components).
  • Rapid project development. Meeting specified deadlines.
  • Flexibility to adapt processes to our customer’s ever-changing requirements.
  • Provide helpful input on how we can get better. Continuous improvement.
  • Help build the Bastian team to be a recognized leader in software development and a great place to work.
  • Keeps current on industry trends as they relate to Bastian’s software products.
  • Timely completion of administrative tasks (time entry, time sheets, expenses, update of requirements in requirement management software, etc.).


Requirements

  • Must be eligible to work in the USA long term without sponsorship.
  • Knowledge of/experience with software testing.
  • Strong SQL experience, scripting.
  • Strong analysis skills to understand best approach on verifying complex modules.
  • Very organized and detail oriented. Stay on top of all portions of a release for hand-off.
  • Bachelor of Science, Bachelor’s degree in Business or Computer Science.
  • Excellent problem-solving capability (business and technical)
  • Excellent verbal and written communication skills
  • 1-3 years experience in working with WMS, Supply Chain Management and Logistics applications – preferably Exacta experience.
  • Enjoys working in the software development and automation technology field
  • Willing to travel in the beginning for training purposes.
  • Outstanding demonstrated performance in preceding career path
  • Strong sense of customer service
  • Team player
  • Happy, positive, energetic attitude
  • Ambition and desire to succeed
