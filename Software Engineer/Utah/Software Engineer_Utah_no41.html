Software Engineer for BYUtv & BYUradio

Job Summary

This position is responsible for designing, developing, testing and integrating platform infrastructure and client applications that deliver live television and on-demand video, audio, and other metadata to applications on websites, mobile devices, streaming media boxes, game consoles, and other connected devices to a global audience.

Essential Functions

  • Develop software code for the BYUB digital platform
  • Create and maintain relational and non-relational databases
  • Develop web services and APIs
  • Develop content management systems (CMS)
  • Build platform integrations into 3rd party platforms like Facebook, Twitter, and YouTube
  • Develop methods of collecting analytics from software
  • Develop code with high security and support for international standards
  • Develop code for all website BYUB properties
  • Develop backend services and API for websites
  • Develop front-end user interfaces using JavaScript, HTML, and CSS
  • Develop code for mobile applications for iOS, Android, and Windows
  • Develop backend services and API for mobile apps
  • Develop client applications and user interfaces
  • Develop code for media applications for other connected “over the top” devices like AppleTV, Roku, Xbox, Amazon FireTV, Chromecast and GoogleTV
  • Develop backend services and API for connected devices
  • Develop client applications for connected OTT devices
  • Manage and mentor student developer teams
  • Manage student tasks
  • Develop and deliver student training
  • Manage student schedules
  • Give input into student hiring and development
  • Provide regular management feedback to students

Qualifications

Level 54 (Software Engineer): Bachelor’s degree (preferably in computer science or related field) and a minimum of 2 years’ of relevant industry experience, OR equivalent combination of education and/or experience

Level 55 (Sr. Software Engineer): Bachelor’s degree (preferably in computer science or related field) and a minimum of 5 years’ of relevant industry experience, OR equivalent combination of education and/or experienced

Level 56 (Principal Software Engineer): Bachelor’s degree (preferably in computer science or related field) and a minimum of 5 years’ of relevant industry experience. With at least 3 years’ of experience using AWS to build highly scalable applications, OR equivalent combination of education and/or experienced with at least 3 years’ of experience using AWS to build highly scalable applications

Skills, abilities, knowledge, licenses, certifications:

  • Understanding of programming methodology, object-oriented languages, & service-oriented architecture (SOA)
  • Experience with the following programming languages or formats:
    • Proficient knowledge in the following: JavaScript, CSS, HTML
    • Basic knowledge in the following: Node.js, JSON, SQL, C#
  • Experience with the following development environments and toolsets:
    • AWS, Git, VS Code, Vue, Nuxt, Express, RESTful web services
  • Experience writing code for both relational (SQL) and non-relational (document) databases
  • Experience with digital media and content distribution networks helpful
  • Comfortable in MAC and PC operating systems
  • Ability to contribute to managed code libraries and to write clean, secure, well-commented code
  • Ability to prioritize work and handle multiple simultaneous projects & tasks
  • Comfortable learning new tools, platforms, and technologies

Ability to evaluate technologies and suggest best tool for problem
