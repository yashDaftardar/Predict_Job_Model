Job Description:

As a Software Development Intern on Merkle’s Digital Experience team, you will be responsible for developing front end and back end / AEM code. You will work with senior developers and architects to implement user stories (JIRA ticket), which contains the requirements and acceptance criteria provided by a business analyst. The JIRA story will also contain creative assets for reference. You will successfully create a functioning AEM component, or front end piece of code that consists of HTML / CSS / JS that meets the JIRA ticket acceptance criteria.


  • Produce clean, efficient, high quality source code for applications
  • Fix and improve existing software
  • Integrate software components and third-party programs
  • Troubleshoot, debug and upgrade existing web applications
  • Develop technical documentation that accurately represent application design and code
  • Evaluate existing applications to reprogram, update and add new features
Qualifications:
  • Rising college Junior or Senior pursuing a degree in computer science or related major preferred; or recent graduate or anyone with relevant experience
  • Knowledge of coding languages (e.g. .Net, Java, JavaScript)
  • Knowledge of HTML and CSS
  • Excellent communication skills
  • Resourcefulness and troubleshooting aptitude; analytical thinking and problem-solving capability; attention to details

Applicants must be currently authorized to work in the U.S. on a full-time basis. Employment-based visa sponsorship (including H-1B sponsorship) is not available for this position.
